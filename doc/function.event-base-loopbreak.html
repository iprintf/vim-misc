event_base_loopbreak (PECL libevent >= 0.0.1) event_base_loopbreak — Abort event loop 说明 bool event_base_loopbreak ( resource $event_base ) Abort the active event loop immediately. The behaviour is similar to break statement. 参数 event_base Valid event base resource. 返回值 event_base_loopbreak() returns TRUE on success or FALSE on error.
