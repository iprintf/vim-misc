XMLReader::moveToAttribute (PHP 5 >= 5.1.0, PHP 7) XMLReader::moveToAttribute — Move cursor to a named attribute 说明 public bool XMLReader::moveToAttribute ( string $name ) Positions cursor on the named attribute. 参数 name The name of the attribute. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • XMLReader::moveToElement() - Position cursor on the parent Element of current Attribute • XMLReader::moveToAttributeNo() - Move cursor to an attribute by index • XMLReader::moveToAttributeNs() - Move cursor to a named attribute • XMLReader::moveToFirstAttribute() - Position cursor on the first Attribute
