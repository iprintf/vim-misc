apcu_exists (PECL apcu >= 4.0.0) apcu_exists — Checks if entry exists 说明 mixed apcu_exists ( mixed $keys ) Checks if one or more APCu entries exist. 参数 keys A string, or an array of strings, that contain keys. 返回值 Returns TRUE if the key exists, otherwise FALSE Or if an array was passed to keys, then an array is returned that contains all existing keys, or an empty array if none exist. 范例 Example #1 apcu_exists() example 以上例程的输出类似于： Foo exists: apple Baz does not exist array(2) { ["foo"]=> bool(true) ["bar"]=> bool(true) } 参见 • apcu_cache_info() - Retrieves cached information from APCu's data store • apcu_fetch() - Fetch a stored variable from the cache
