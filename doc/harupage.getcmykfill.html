HaruPage::getCMYKFill (PECL haru >= 0.0.1) HaruPage::getCMYKFill — Get the current filling color 说明 array HaruPage::getCMYKFill ( void ) Returns the current filling color. 参数 此函数没有参数。 返回值 Returns the current filling color as an array with 4 elements ("c", "m", "y" and "k"). 错误／异常 发生错误时抛出 HaruException 异常。
