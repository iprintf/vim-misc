SplFileObject::seek (PHP 5 >= 5.1.0, PHP 7) SplFileObject::seek — Seek to specified line 说明 public void SplFileObject::seek ( int $line_pos ) Seek to specified line in the file. 参数 line_pos The zero-based line number to seek to. 返回值 没有返回值。 错误／异常 Throws a LogicException if the line_pos is negative. 范例 Example #1 SplFileObject::seek() example This example outputs the third line of the script which is found at position 2. seek(2); echo $file->current(); ?> 以上例程的输出类似于： $file->seek(2); 参见 • SplFileObject::current() - Retrieve current line of file • SplFileObject::key() - Get line number • SplFileObject::next() - Read next line • SplFileObject::rewind() - Rewind the file to the first line • SplFileObject::valid() - Not at EOF
