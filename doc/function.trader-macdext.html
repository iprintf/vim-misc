trader_macdext (PECL trader >= 0.2.0) trader_macdext — MACD with controllable MA type 说明 array trader_macdext ( array $real [, integer $fastPeriod [, integer $fastMAType [, integer $slowPeriod [, integer $slowMAType [, integer $signalPeriod [, integer $signalMAType ]]]]]] ) 参数 real 浮点数数组。 fastPeriod Number of period for the fast MA. Valid range from 2 to 100000. fastMAType Type of Moving Average for fast MA. TRADER_MA_TYPE_* series of constants should be used. slowPeriod Number of period for the slow MA. Valid range from 2 to 100000. slowMAType Type of Moving Average for slow MA. TRADER_MA_TYPE_* series of constants should be used. signalPeriod Smoothing for the signal line (nb of period). Valid range from 1 to 100000. signalMAType Type of Moving Average for signal line. TRADER_MA_TYPE_* series of constants should be used. 返回值 Returns an array with calculated data or false on failure.
