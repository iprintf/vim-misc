ImagickDraw::push (PECL imagick 2.0.0) ImagickDraw::push — Clones the current ImagickDraw and pushes it to the stack 说明 bool ImagickDraw::push ( void ) Warning 本函数还未编写文档，仅有参数列表。 Clones the current ImagickDraw to create a new ImagickDraw, which is then added to the ImagickDraw stack. The original drawing ImagickDraw(s) may be returned to by invoking pop(). The ImagickDraws are stored on a ImagickDraw stack.
