DOMText::__construct (PHP 5, PHP 7) DOMText::__construct — Creates a new DOMText object 说明 public DOMText::__construct ([ string $value ] ) Creates a new DOMText object. 参数 value The value of the text node. If not supplied an empty text node is created. 范例 Example #1 Creating a new DOMText appendChild(new DOMElement('root')); $text = $element->appendChild(new DOMText('root value')); echo $dom->saveXML(); /* root value */ ?> 参见 • DOMDocument::createTextNode() - Create new text node User Contributed Notes 1 note =============================================================================== I experinced that DOMText, given a integer as only value to the constructor failed to return valid data. I had to typecast the input to a string first: $intValue = 3252; // Post/Zip-code $some_element->appendChild( new DOMText( (string) $intValue) ); When not typecasting using "(string)" the XML output from $root->saveXML() would not validate.
