PDO::pgsqlCopyToFile (PHP 5 >= 5.3.3, PHP 7) PDO::pgsqlCopyToFile — Copy data from table into file 说明 public bool PDO::pgsqlCopyToFile ( string $table_name , string $filename [, string $delimiter = '\t' [, string $null_as = "\\\\N" [, string $fields ]]] ) Copies data from table into file specified by filename using delimiter as fields delimiter and fields list 参数 table_name String containing table name filename Filename to export data delimiter Delimiter used in file specified by filename null_as How to interpret null values fields List of fields to insert 返回值 Returns TRUE on success, 或者在失败时返回 FALSE.
