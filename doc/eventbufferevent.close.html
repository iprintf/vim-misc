EventBufferEvent::close (PECL event >= 1.10.0) EventBufferEvent::close — Closes file descriptor associated with the current buffer event 说明 public void EventBufferEvent::close ( void ) Closes file descriptor associated with the current buffer event. This method may be used in cases when the EventBufferEvent::OPT_CLOSE_ON_FREE option is not appropriate. 参数 此函数没有参数。 返回值 没有返回值。
