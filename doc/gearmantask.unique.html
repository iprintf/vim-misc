GearmanTask::unique (PECL gearman >= 0.6.0) GearmanTask::unique — Get the unique identifier for a task 说明 public string GearmanTask::unique ( void ) Returns the unique identifier for this task. This is assigned by the GearmanClient, as opposed to the job handle which is set by the Gearman job server. 参数 此函数没有参数。 返回值 The unique identifier, or FALSE if no identifier is assigned. 参见 • GearmanClient::do() - Run a single task and return a result [deprecated] • GearmanClient::addTask() - Add a task to be run in parallel
