bzerror (PHP 4 >= 4.0.4, PHP 5, PHP 7) bzerror — 返回包含 bzip2 错误号和错误字符串的一个 array 说明 array bzerror ( resource $bz ) 返回文件指针中返回的 bzip2 错误的错误号和错误字符串。 参数 bz 文件指针。它必须是有效的并且指向 bzopen() 成功打开的文件。 返回值 返回一个关联数组，包含错误码于 errno 条目，以及错误信息于 errstr 条目。 范例 Example #1 bzerror() 范例 参见 • bzerrno() - 返回一个 bzip2 错误码 • bzerrstr() - 返回一个 bzip2 的错误字符串
