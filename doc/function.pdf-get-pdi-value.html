PDF_get_pdi_value (PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0) PDF_get_pdi_value — Get PDI numerical parameter [deprecated] 说明 float PDF_get_pdi_value ( resource $p , string $key , int $doc , int $page , int $reserved ) Gets the contents of a PDI document parameter with numerical type. This function is deprecated since PDFlib version 7, use PDF_pcos_get_number() instead.
