Imagick::transparentPaintImage (No version information available, might only be in Git) Imagick::transparentPaintImage — Paints pixels transparent 说明 bool Imagick::transparentPaintImage ( mixed $target , float $alpha , float $fuzz , bool $invert ) Paints pixels matching the target color transparent. 此方法在Imagick基于 ImageMagick 6.3.8以上版本编译时可用。 参数 target The target color to paint alpha 透明级别：1.0 是完全不透明，0.0 是完全透明。 fuzz The amount of fuzz. For example, set fuzz to 10 and the color red at intensities of 100 and 102 respectively are now interpreted as the same color. invert If TRUE paints any pixel that does not match the target color. 返回值 成功时返回 TRUE。 范例 Example #1 Imagick::transparentPaintImage() setimageformat('png'); $imagick->transparentPaintImage( $color, $alpha, $fuzz * \Imagick::getQuantum(), false ); //Not required, but helps tidy up left over pixels $imagick->despeckleimage(); header("Content-Type: image/png"); echo $imagick->getImageBlob(); } ?>
