SolrClient::deleteById (PECL solr >= 0.9.2) SolrClient::deleteById — Delete by Id 说明 public SolrUpdateResponse SolrClient::deleteById ( string $id ) Deletes the document with the specified ID. Where ID is the value of the uniqueKey field declared in the schema 参数 id The value of the uniqueKey field declared in the schema 返回值 Returns a SolrUpdateResponse on success and throws an exception on failure. 错误／异常 Throws SolrClientException if the client had failed, or there was a connection issue.
