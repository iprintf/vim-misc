rpm_open (PECL rpmreader >= 0.1.0) rpm_open — Opens an RPM file 说明 resource rpm_open ( string $filename ) rpm_open() will open an RPM file and will determine if the file is a valid RPM file. 参数 filename The filename of the RPM file you wish to open. 返回值 If the open succeeds, then rpm_open() will return a file pointer resource to the newly opened file. On error, the function will return FALSE. 范例 Example #1 rpm_open() example 参见 • rpm_close() - Closes an RPM file User Contributed Notes 2 notes =============================================================================== It's worth noting that a URL to an RPM package can be passed to rpm_open().
