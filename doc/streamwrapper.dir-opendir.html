streamWrapper::dir_opendir (PHP 4 >= 4.3.2, PHP 5, PHP 7) streamWrapper::dir_opendir — Open directory handle 说明 public bool streamWrapper::dir_opendir ( string $path , int $options ) This method is called in response to opendir(). 参数 path Specifies the URL that was passed to opendir(). Note: The URL can be broken apart with parse_url(). options Whether or not to enforce safe_mode (0x04). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • opendir() - 打开目录句柄 • streamWrapper::dir_closedir() - Close directory handle • parse_url() - 解析 URL，返回其组成部分
