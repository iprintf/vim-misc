ibase_commit (PHP 5, PHP 7) ibase_commit — Commit a transaction 说明 bool ibase_commit ([ resource $link_or_trans_identifier = NULL ] ) Commits a transaction. 参数 link_or_trans_identifier If called without an argument, this function commits the default transaction of the default link. If the argument is a connection identifier, the default transaction of the corresponding connection will be committed. If the argument is a transaction identifier, the corresponding transaction will be committed. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
