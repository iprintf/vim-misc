pspell_config_ignore (PHP 4 >= 4.0.2, PHP 5, PHP 7) pspell_config_ignore — Ignore words less than N characters long 说明 bool pspell_config_ignore ( int $dictionary_link , int $n ) pspell_config_ignore() should be used on a config before calling pspell_new_config(). This function allows short words to be skipped by the spell checker. 参数 dictionary_link n Words less than n characters will be skipped. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 pspell_config_ignore()
