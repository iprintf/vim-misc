MongoGridFSFile::getBytes (PECL mongo >=0.9.0) MongoGridFSFile::getBytes — Returns this file's contents as a string of bytes 说明 public string MongoGridFSFile::getBytes ( void ) Warning: this will load the file into memory. If the file is bigger than your memory, this will cause problems! 参数 此函数没有参数。 返回值 Returns a string of the bytes in the file. 范例 Example #1 MongoGridFSFile::getBytes() example my_db->getGridFS('images'); $image = $images->findOne('jwage.png'); header('Content-type: image/png;'); echo $image->getBytes(); ?>
