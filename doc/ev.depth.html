Ev::depth (PECL ev >= 0.2.0) Ev::depth — Returns recursion depth 说明 final public static int Ev::depth ( void ) The number of times Ev::run() was entered minus the number of times Ev::run() was exited normally, in other words, the recursion depth. Outside Ev::run() , this number is 0 . In a callback, this number is 1 , unless Ev::run() was invoked recursively (or from another thread), in which case it is higher. 参数 此函数没有参数。 返回值 ev_depth() returns recursion depth of the default loop. 参见 • Ev::iteration() - Return the number of times the default event loop has polled for new events.
