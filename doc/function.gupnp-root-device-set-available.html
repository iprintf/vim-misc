gupnp_root_device_set_available (PECL gupnp >= 0.1.0) gupnp_root_device_set_available — Set whether or not root_device is available 说明 bool gupnp_root_device_set_available ( resource $root_device , bool $available ) Controls whether or not root_device is available (announcing its presence). 参数 root_device A root device identifier, returned by gupnp_root_device_new(). available Set TRUE if root_device should be available. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • gupnp_root_device_get_available() - Check whether root device is available
