ssh2_sftp_rename (PECL ssh2 >= 0.9.0) ssh2_sftp_rename — Rename a remote file 说明 bool ssh2_sftp_rename ( resource $sftp , string $from , string $to ) Renames a file on the remote filesystem. 参数 sftp An SSH2 SFTP resource opened by ssh2_sftp(). from The current file that is being renamed. to The new file name that replaces from. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 Renaming a file via sftp 参见 • rename() - 重命名一个文件或目录 User Contributed Notes 1 note =============================================================================== An already existing file will NOT be overwritten. http://www.openssh.com/txt/draft-ietf-secsh-filexfer-02.txt "It is an error if there already exists a file with the name specified by newpath."
