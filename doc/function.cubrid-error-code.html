cubrid_error_code (PECL CUBRID >= 8.3.0) cubrid_error_code — Get error code for the most recent function call 说明 int cubrid_error_code ( void ) The cubrid_error_code() function is used to get the error code of the error that occurred during the API execution. Usually, it gets the error code when API returns false as its return value. 参数 此函数没有参数。 返回值 Error code of the error that occurred, or 0 (zero) if no error occurred. 范例 Example #1 cubrid_error_code() example 以上例程会输出： Error facility: 4 Error code: -30015 Error msg: Some parameter not binded 参见 • cubrid_error_code_facility() - Get the facility code of error • cubrid_error_msg() - Get last error message for the most recent function call
