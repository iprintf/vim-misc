HaruPage::moveTextPos (PECL haru >= 0.0.1) HaruPage::moveTextPos — Move text position to the specified offset 说明 bool HaruPage::moveTextPos ( float $x , float $y [, bool $set_leading = false ] ) Moves text position to the specified offset. If the start position of the current line is (x1, y1), the start of the next line is (x1 + x, y1 + y). 参数 x The specified text position offset. y The specified text position offset. set_leading If set to TRUE, the function sets the text leading to -y. 返回值 Returns TRUE on success. 错误／异常 发生错误时抛出 HaruException 异常。 参见 • HaruPage::moveToNextLine() - Move text position to the start of the next line
