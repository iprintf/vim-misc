sybase_deadlock_retry_count (PHP 4 >= 4.3.0, PHP 5, PHP 7) sybase_deadlock_retry_count — Sets the deadlock retry count 说明 void sybase_deadlock_retry_count ( int $retry_count ) Using sybase_deadlock_retry_count(), the number of retries can be defined in cases of deadlocks. By default, every deadlock is retried an infinite number of times or until the process is killed by Sybase, the executing script is killed (for instance, by set_time_limit()) or the query succeeds. 参数 retry_count Values for retry_count -1 Retry forever (default) 0 Do not retry n Retry n times 返回值 没有返回值。 注释 Note: 此函数仅适用于对 Sybase 使用 CT 库接口，而不适用于 DB 库。
