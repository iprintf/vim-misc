MongoCursor::hasNext (PECL mongo >=0.9.0) MongoCursor::hasNext — Checks if there are any more elements in this cursor 说明 public bool MongoCursor::hasNext ( void ) 参数 此函数没有参数。 返回值 Returns if there is another element. 错误／异常 Throws MongoConnectionException if it cannot reach the database and MongoCursorTimeoutException if the timeout is exceeded.
