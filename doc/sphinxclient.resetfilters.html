SphinxClient::resetFilters (PECL sphinx >= 0.1.0) SphinxClient::resetFilters — Clear all filters 说明 public void SphinxClient::resetFilters ( void ) Clears all currently set filters. This call is normally required when using multi-queries. You might want to set different filters for different queries in the batch. To do that, you should call SphinxClient::resetFilters() and add new filters using the respective calls. 返回值 没有返回值。 参见 • SphinxClient::resetGroupBy
