IntlChar::getCombiningClass (PHP 7) IntlChar::getCombiningClass — Get the combining class of a code point 说明 public static int IntlChar::getCombiningClass ( mixed $codepoint ) Returns the combining class of the code point. 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 Returns the combining class of the character. 范例 Example #1 Testing different code points 以上例程会输出： int(0) int(1) int(232)
