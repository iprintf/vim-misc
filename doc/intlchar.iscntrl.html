IntlChar::iscntrl (PHP 7) IntlChar::iscntrl — Check if code point is a control character 说明 public static bool IntlChar::iscntrl ( mixed $codepoint ) Determines whether the specified code point is a control character. A control character is one of the following: • ISO 8-bit control character (U+0000..U+001f and U+007f..U+009f) • IntlChar::CHAR_CATEGORY_CONTROL_CHAR (Cc) • IntlChar::CHAR_CATEGORY_FORMAT_CHAR (Cf) • IntlChar::CHAR_CATEGORY_LINE_SEPARATOR (Zl) • IntlChar::CHAR_CATEGORY_PARAGRAPH_SEPARATOR (Zp) 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 Returns TRUE if codepoint is a control character, FALSE if not. 范例 Example #1 Testing different code points 以上例程会输出： bool(false) bool(false) bool(true) bool(true) 参见 • IntlChar::isprint() - Check if code point is a printable character • IntlChar::PROPERTY_DEFAULT_IGNORABLE_CODE_POINT
