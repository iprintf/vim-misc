SQLite3Stmt::reset (PHP 5 >= 5.3.0, PHP 7) SQLite3Stmt::reset — Resets the prepared statement 说明 public bool SQLite3Stmt::reset ( void ) Resets the prepared statement to its state prior to execution. All bindings remain intact after reset. 参数 此函数没有参数。 返回值 Returns TRUE if the statement is successfully reset, FALSE on failure.
