xmlrpc_is_fault (PHP 4 >= 4.3.0, PHP 5, PHP 7) xmlrpc_is_fault — Determines if an array value represents an XMLRPC fault 说明 bool xmlrpc_is_fault ( array $arg ) Warning 此函数是实验性的。此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本 中未通知就被修改。使用本函数风险自担。 参数 arg Array returned by xmlrpc_decode(). 返回值 Returns TRUE if the argument means fault, FALSE otherwise.
