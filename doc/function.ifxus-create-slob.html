ifxus_create_slob (PHP 4, PHP <=5.2.0) ifxus_create_slob — Creates an slob object and opens it 说明 int ifxus_create_slob ( int $mode ) Creates an slob object and opens it. 参数 mode A combination of IFX_LO_RDONLY, IFX_LO_WRONLY, IFX_LO_APPEND IFX_LO_RDWR, IFX_LO_BUFFER, IFX_LO_NOBUFFER. 返回值 Return the new slob object-id, or FALSE on errors. 参见 • ifxus_close_slob() - Deletes the slob object • ifxus_free_slob() - Deletes the slob object
