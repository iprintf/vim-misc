SplFileObject::fscanf (PHP 5 >= 5.1.0, PHP 7) SplFileObject::fscanf — Parses input from file according to a format 说明 public mixed SplFileObject::fscanf ( string $format [, mixed &$... ] ) Reads a line from the file and interprets it according to the specified format, which is described in the documentation for sprintf(). Any whitespace in the format string matches any whitespace in the line from the file. This means that even a tab \t in the format string can match a single space character in the input stream. 参数 format The specified format as described in the sprintf() documentation. ... The optional assigned values. 返回值 If only one parameter is passed to this method, the values parsed will be returned as an array. Otherwise, if optional parameters are passed, the function will return the number of assigned values.
