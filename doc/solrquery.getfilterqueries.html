SolrQuery::getFilterQueries (PECL solr >= 0.9.2) SolrQuery::getFilterQueries — Returns an array of filter queries 说明 public array SolrQuery::getFilterQueries ( void ) Returns an array of filter queries. These are queries that can be used to restrict the super set of documents that can be returned, without influencing score 参数 此函数没有参数。 返回值 Returns an array on success and NULL if not set.
