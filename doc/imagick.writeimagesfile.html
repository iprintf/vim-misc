Imagick::writeImagesFile (No version information available, might only be in Git) Imagick::writeImagesFile — Writes frames to a filehandle 说明 bool Imagick::writeImagesFile ( resource $filehandle ) Writes all image frames into an open filehandle. This method can be used to write animated gifs or other multiframe images into open filehandle. 此方法在 Imagick基于ImageMagick 6.3.6以上版本编译时可用。 参数 filehandle Filehandle where to write the images 返回值 成功时返回 TRUE。
