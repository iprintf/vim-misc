ps_moveto (PECL ps >= 1.1.0) ps_moveto — Sets current point 说明 bool ps_moveto ( resource $psdoc , float $x , float $y ) Sets the current point to new coordinates. If this is the first call of ps_moveto() after a previous path has been ended then it will start a new path. If this function is called in the middle of a path it will just set the current point and start a subpath. 参数 psdoc Resource identifier of the postscript file as returned by ps_new(). x x-coordinate of the point to move to. y y-coordinate of the point to move to. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ps_lineto() - Draws a line
