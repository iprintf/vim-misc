readline_info (PHP 4, PHP 5, PHP 7) readline_info — 获取/设置readline内部的各个变量 说明 mixed readline_info ([ string $varname [, string $newvalue ]] ) 获取/设置readline内部的各个变量. 参数 varname 变量名. newvalue 如果提供，将被设置成这个值. 返回值 如果调用时没有参数，这个函数将返回一个包括了所有已采用的readline设置的数组. 数 组元素将索引是以下这些值：done, end, erase_empty_line, library_version, line_buffer, mark, pending_input, point, prompt, readline_name, and terminal_name. 如果调用是提供了一个或两个参数，原来的值将被返回.
