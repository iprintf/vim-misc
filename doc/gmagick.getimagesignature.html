Gmagick::getimagesignature (PECL gmagick >= Unknown) Gmagick::getimagesignature — Generates an SHA-256 message digest 说明 public string Gmagick::getimagesignature ( void ) Generates an SHA-256 message digest for the image pixel stream. 参数 此函数没有参数。 返回值 Returns a string containing the SHA-256 hash of the file. 错误／异常 错误时抛出 GmagickException。
