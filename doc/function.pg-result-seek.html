pg_result_seek (PHP 4 >= 4.3.0, PHP 5, PHP 7) pg_result_seek — 在结果资源中设定内部行偏移量 说明 array pg_result_seek ( resource $result , int $offset ) pg_result_seek() 在结果资源中设定内部行偏移量。如果出错则返回 FALSE。 参见 pg_fetch_row()，pg_fetch_assoc()，pg_fetch_array()，pg_fetch_object() 和 pg_fetch_result()。 User Contributed Notes 1 note =============================================================================== Ah, this is a handy feature for resetting the record index, for example, if you're used pg_fetch_{row,array,assoc} to iterate over the result set, and you want to do it again later on, without reexecuting your query.
