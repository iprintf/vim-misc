px_close (PECL paradox >= 1.0.0) px_close — Closes a paradox database 说明 bool px_close ( resource $pxdoc ) Closes the paradox database. This function will not close the file. You will have to call fclose() afterwards. 参数 pxdoc Resource identifier of the paradox database as returned by px_new(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • px_open_fp() - Open paradox database • The example at px_new() - Create a new paradox object
