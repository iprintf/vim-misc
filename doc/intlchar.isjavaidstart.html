IntlChar::isJavaIDStart (PHP 7) IntlChar::isJavaIDStart — Check if code point is permissible as the first character in a Java identifier 说明 public static bool IntlChar::isJavaIDStart ( mixed $codepoint ) Determines if the specified character is permissible as the start of a Java identifier. In addition to IntlChar::isIDStart(), TRUE for characters with general categories "Sc" (currency symbols) and "Pc" (connecting punctuation). 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 Returns TRUE if codepoint may start a Java identifier, FALSE if not. 范例 Example #1 Testing different code points 以上例程会输出： bool(true) bool(true) bool(false) bool(false) 参见 • IntlChar::isIDStart() - Check if code point is permissible as the first character in an identifier • IntlChar::isJavaIDPart() - Check if code point is permissible in a Java identifier • IntlChar::isalpha() - Check if code point is a letter character
