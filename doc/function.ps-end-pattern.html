ps_end_pattern (PECL ps >= 1.2.0) ps_end_pattern — End a pattern 说明 bool ps_end_pattern ( resource $psdoc ) Ends a pattern which was started with ps_begin_pattern(). Once a pattern has been ended, it can be used like a color to fill areas. 参数 psdoc Resource identifier of the postscript file as returned by ps_new(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ps_begin_pattern() - Start a new pattern
