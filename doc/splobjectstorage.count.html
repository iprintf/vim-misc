SplObjectStorage::count (PHP 5 >= 5.1.0, PHP 7) SplObjectStorage::count — Returns the number of objects in the storage 说明 public int SplObjectStorage::count ( void ) Counts the number of objects in the storage. 参数 此函数没有参数。 返回值 The number of objects in the storage. 范例 Example #1 SplObjectStorage::count() example attach($o1); $s->attach($o2); $s->attach($o1); var_dump($s->count()); var_dump(count($s)); ?> 以上例程的输出类似于： int(2) int(2) 参见 • SplObjectStorage::attach() - Adds an object in the storage • SplObjectStorage::detach() - Removes an object from the storage
