ImagickPixel::isPixelSimilar (No version information available, might only be in Git) ImagickPixel::isPixelSimilar — Check the distance between this color and another 说明 bool ImagickPixel::isPixelSimilar ( ImagickPixel $color , float $fuzz ) Checks the distance between the color described by this ImagickPixel object and that of the provided object, by plotting their RGB values on the color cube. If the distance between the two points is less than the fuzz value given, the colors are similar. This method replaces ImagickPixel::isSimilar() and correctly normalises the fuzz value to ImageMagick QuantumRange. 参数 color The ImagickPixel object to compare this object against. fuzz The maximum distance within which to consider these colors as similar. The theoretical maximum for this value is the square root of three (1.732). 返回值 成功时返回 TRUE。
