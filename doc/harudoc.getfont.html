HaruDoc::getFont (PECL haru >= 0.0.1) HaruDoc::getFont — Get HaruFont instance 说明 object HaruDoc::getFont ( string $fontname [, string $encoding ] ) Get a HaruFont instance. 参数 fontname The name of the font. See Builtin Fonts for the list of builtin fonts. You can also use the name of a font loaded via HaruDoc::loadTTF(), HaruDoc::loadTTC() and HaruDoc::loadType1(). encoding The encoding to use. See Builtin Encodings for the list of supported encodings. 返回值 Returns a HaruFont instance with the specified fontname and encoding. 错误／异常 发生错误时抛出 HaruException 异常。 参见 • HaruPage::setFontAndSize() - Set font and fontsize for the page • HaruPage::setCurrentFont()
