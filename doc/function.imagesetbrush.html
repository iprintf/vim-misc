imagesetbrush (PHP 4 >= 4.0.6, PHP 5, PHP 7) imagesetbrush — 设定画线用的画笔图像 说明 bool imagesetbrush ( resource $image , resource $brush ) 当用特殊的颜色 IMG_COLOR_BRUSHED 或 IMG_COLOR_STYLEDBRUSHED 绘画时， imagesetbrush() 设定了所有画线的函数（例如 imageline() 和 imagepolygon()）所使 用的画笔图像。【注：使用画笔图像，所画的线是由 brush 所代表的图像构成的。请参考 并尝试运行 imagesetstyle() 中的例子以帮助理解。】 Note: 使用完画笔图像后不需要采取什么特殊动作。但如果销毁了画笔图像，在设定一个新 的画笔图像之前不能使用 IMG_COLOR_BRUSHED 或 IMG_COLOR_STYLEDBRUSHED！ Note: 本函数是 PHP 4.0.6 添加的。 User Contributed Notes 1 note =============================================================================== Use a brush to create a thick line. To create a 3x3 red brush: Then use imageline() or imagepolygon() with IMG_COLOR_BRUSHED as the color. To stop using the brush, destroy it: The brush can also be created from an existing image.
