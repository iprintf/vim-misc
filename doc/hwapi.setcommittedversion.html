hw_api::setcommittedversion (PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN) hw_api::setcommittedversion — Commits version other than last version 说明 hw_api_object hw_api::setcommittedversion ( array $parameter ) Commits a version of a document.
