ps_setdash (PECL ps >= 1.1.0) ps_setdash — Sets appearance of a dashed line 说明 bool ps_setdash ( resource $psdoc , float $on , float $off ) Sets the length of the black and white portions of a dashed line. 参数 psdoc Resource identifier of the postscript file as returned by ps_new(). on The length of the dash. off The length of the gap between dashes. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ps_setpolydash() - Sets appearance of a dashed line
