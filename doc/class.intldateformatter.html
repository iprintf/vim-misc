The IntlDateFormatter class (PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0) 简介 Date Formatter is a concrete class that enables locale-dependent formatting/ parsing of dates using pattern strings and/or canned patterns. This class represents the ICU date formatting functionality. It allows users to display dates in a localized format or to parse strings into PHP date values using pattern strings and/or canned patterns.
