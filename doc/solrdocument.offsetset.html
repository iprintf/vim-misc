SolrDocument::offsetSet (PECL solr >= 0.9.2) SolrDocument::offsetSet — Adds a field to the document 说明 public void SolrDocument::offsetSet ( string $fieldName , string $fieldValue ) Used when the object is treated as an array to add a field to the document. 参数 fieldName The name of the field. fieldValue The value for this field. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
