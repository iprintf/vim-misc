GearmanJob::sendFail (PECL gearman >= 0.6.0) GearmanJob::sendFail — Send fail status 说明 public bool GearmanJob::sendFail ( void ) Sends failure status for this job, indicating that the job failed in a known way (as opposed to failing due to a thrown exception). 参数 此函数没有参数。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • GearmanJob::sendException() - Send exception for running job (exception) • GearmanJob::setReturn() - Set a return value • GearmanJob::sendStatus() - Send status • GearmanJob::sendWarning() - Send a warning
