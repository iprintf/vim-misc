ps_new (PECL ps >= 1.1.0) ps_new — Creates a new PostScript document object 说明 resource ps_new ( void ) Creates a new document instance. It does not create the file on disk or in memory, it just sets up everything. ps_new() is usually followed by a call of ps_open_file() to actually create the postscript document. 返回值 Resource of PostScript document 或者在失败时返回 FALSE. The return value is passed to all other functions as the first argument. 参见 • ps_delete() - Deletes all resources of a PostScript document
