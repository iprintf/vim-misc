posix_ttyname (PHP 4, PHP 5, PHP 7) posix_ttyname — Determine terminal device name 说明 string posix_ttyname ( mixed $fd ) Returns a string for the absolute path to the current terminal device that is open on the file descriptor fd. 参数 fd The file descriptor, which is expected to be either a file resource or an integer. An integer will be assumed to be a file descriptor that can be passed directly to the underlying system call. In almost all cases, you will want to provide a file resource. 返回值 On success, returns a string of the absolute path of the fd. On failure, returns FALSE User Contributed Notes 1 note =============================================================================== returns: string(10) "/dev/pts/0" When using pseudo terminal 1 (ie ssh)
