SplFileInfo::getType (PHP 5 >= 5.1.2, PHP 7) SplFileInfo::getType — Gets file type 说明 public string SplFileInfo::getType ( void ) Returns the type of the file referenced. 参数 此函数没有参数。 返回值 A string representing the type of the entry. May be one of file, link, or dir 错误／异常 Throws a RuntimeException on error. 范例 Example #1 SplFileInfo::getType() example getType().PHP_EOL; $info = new SplFileInfo(dirname(__FILE__)); echo $info->getType(); ?> 以上例程的输出类似于： file dir
