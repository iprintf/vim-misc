Thread::isJoined (PECL pthreads >= 2.0.0) Thread::isJoined — 状态监测 说明 public boolean Thread::isJoined ( void ) 线程是否已经被加入（join） 参数 此函数没有参数。 返回值 布尔值，表示是否被加入 范例 Example #1 检测线程状态 synchronized(function($thread){ if (!$thread->done) $thread->wait(); }, $this); } } $my = new My(); $my->start(); var_dump($my->isJoined()); $my->synchronized(function($thread){ $thread->done = true; $thread->notify(); }, $my); ?> 以上例程会输出： bool(false)
