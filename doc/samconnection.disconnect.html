SAMConnection::disconnect (PECL sam >= 0.1.0) SAMConnection::disconnect — Disconnects from a Messaging Server 说明 bool SAMConnection::disconnect ( void ) Calling the "disconnect" method on a SAMConnection object disconnects the PHP script from a messaging server.
