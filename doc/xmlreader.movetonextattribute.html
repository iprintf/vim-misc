XMLReader::moveToNextAttribute (PHP 5 >= 5.1.0, PHP 7) XMLReader::moveToNextAttribute — Position cursor on the next Attribute 说明 public bool XMLReader::moveToNextAttribute ( void ) Moves cursor to the next Attribute if positioned on an Attribute or moves to first attribute if positioned on an Element. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • XMLReader::moveToElement() - Position cursor on the parent Element of current Attribute • XMLReader::moveToAttribute() - Move cursor to a named attribute • XMLReader::moveToAttributeNo() - Move cursor to an attribute by index • XMLReader::moveToAttributeNs() - Move cursor to a named attribute • XMLReader::moveToFirstAttribute() - Position cursor on the first Attribute
