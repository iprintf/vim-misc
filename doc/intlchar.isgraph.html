IntlChar::isgraph (PHP 7) IntlChar::isgraph — Check if code point is a graphic character 说明 public static bool IntlChar::isgraph ( mixed $codepoint ) Determines whether the specified code point is a "graphic" character (printable, excluding spaces). TRUE for all characters except those with general categories "Cc" (control codes), "Cf" (format controls), "Cs" (surrogates), "Cn" (unassigned), and "Z" (separators). 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 Returns TRUE if codepoint is a "graphic" character, FALSE if not. 范例 Example #1 Testing different code points 以上例程会输出： bool(true) bool(true) bool(true) bool(false)
