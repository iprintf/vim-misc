trader_sub (PECL trader >= 0.2.0) trader_sub — Vector Arithmetic Subtraction 说明 array trader_sub ( array $real0 , array $real1 ) Calculates the vector subtraction of real1 from real0 and returns the resulting vector. 参数 real0 浮点数数组。 real1 浮点数数组。 返回值 Returns an array with calculated data or false on failure.
