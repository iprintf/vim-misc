cubrid_field_len (PECL CUBRID >= 8.3.0) cubrid_field_len — Get the maximum length of the specified field 说明 int cubrid_field_len ( resource $result , int $field_offset ) This function returns the maximum length of the specified field on success, or it returns FALSE on failure. 参数 result result comes from a call to cubrid_execute() field_offset The numerical field offset. The field_offset starts at 0. If field_offset does not exist, an error of level E_WARNING is also issued. 返回值 Maximum length, when process is successful. FALSE on failure. 范例 Example #1 cubrid_field_len() example 以上例程会输出： Column Names Column Types Column Maxlen host_year integer 11 event_code integer 11 athlete_code integer 11 stadium_code integer 11 nation_code char 3 medal char 1 game_date date 10
