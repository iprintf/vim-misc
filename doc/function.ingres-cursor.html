ingres_cursor (PECL ingres >= 1.1.0) ingres_cursor — Get a cursor name for a given result resource 说明 string ingres_cursor ( resource $result ) Returns a string with the active cursor name. If no cursor is active then NULL is returned. 参数 result The query result identifier 返回值 Returns a string containing the active cursor name.
