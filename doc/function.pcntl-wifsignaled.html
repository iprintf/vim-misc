pcntl_wifsignaled (PHP 4 >= 4.1.0, PHP 5, PHP 7) pcntl_wifsignaled — 检查子进程状态码是否代表由于某个信号而中断 说明 bool pcntl_wifsignaled ( int $status ) 检查子进程是否是由于某个未捕获的信号退出的。 参数 status 参数 status 是提供给成功调用 pcntl_waitpid() 时的状态参数。 返回值 如果子进程是由于某个未捕获的信号退出的返回 TRUE ，其他情况返回 FALSE 。 参见 • pcntl_waitpid() - 等待或返回fork的子进程状态 • pcntl_signal() - 安装一个信号处理器
