MongoBinData::__construct (PECL mongo >= 0.8.1) MongoBinData::__construct — 创建一个新的二进制数据对象 说明 public MongoBinData::__construct ( string $data [, int $type = 2 ] ) 创建一个新的二进制数据对象。 当前有五种类型的二进制数据能被 BSON 规格所识别： function (0x01)、 byte array (0x02)、 UUID (0x03)、 MD5 (0x05)，和用户定义的 (0x80)。默认类型是 byte array (0x02)。驱动和服务器解释不同类型时没有什么特殊的不同之处，所以目前他们是无关紧 要的。任何数字（在0和255之间）能用于 type，只要用户愿意冒着数据库可能基于类型做 些处理的风险。 参数 data 二进制数据。 type 数据类型。 返回值 返回一个新的二进制数据对象。 更新日志 版本 说明 1.2.11 没有使用第二个参数时产生 E_DEPRECATED。 type 的默认值在近期的功能里可能 会改变。
