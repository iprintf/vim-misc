pg_options (PHP 4, PHP 5, PHP 7) pg_options — 获得和连接有关的选项 说明 string pg_options ( resource $connection ) pg_options() 以字符串形式返回指定 PostgreSQL connection 资源的连接选项。 User Contributed Notes 1 note =============================================================================== Apparently this function does not return the database name given in the connection string. I tried this on apache 2.x/php4.3.x/pg7.4.x and apache 1.3.x/php5.x/pg8rc1 and saw this behavior on both setups.
