fann_get_total_neurons (PECL fann >= 1.0.0) fann_get_total_neurons — Get the total number of neurons in the entire network 说明 int fann_get_total_neurons ( resource $ann ) Get the total number of neurons in the entire network. This number does also include the bias neurons, so a 2-4-2 network has 2+4+2 +2(bias) = 10 neurons. 参数 ann Neural network resource. 返回值 Total number of neurons in the entire network, or FALSE on error.
