fann_shuffle_train_data (PECL fann >= 1.0.0) fann_shuffle_train_data — Shuffles training data, randomizing the order 说明 bool fann_shuffle_train_data ( resource $train_data ) Shuffles training data, randomizing the order. This is recommended for incremental training, while it have no influence during batch training. 参数 train_data Neural network training data resource. 返回值 Returns TRUE on success, or FALSE otherwise.
