px_get_value (PECL paradox >= 1.1.0) px_get_value — Gets a value 说明 float px_get_value ( resource $pxdoc , string $name ) Gets various values. 参数 pxdoc Resource identifier of the paradox database as returned by px_new(). name name can be one of the following. numprimkeys The number of primary keys. Paradox databases always use the first numprimkeys fields for the primary index. 返回值 Returns the value of the parameter 或者在失败时返回 FALSE.
