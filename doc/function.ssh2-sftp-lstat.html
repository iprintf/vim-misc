ssh2_sftp_lstat (PECL ssh2 >= 0.9.0) ssh2_sftp_lstat — Stat a symbolic link 说明 array ssh2_sftp_lstat ( resource $sftp , string $path ) Stats a symbolic link on the remote filesystem without following the link. This function is similar to using the lstat() function with the ssh2.sftp:// wrapper in PHP 5 and returns the same values. 参数 sftp path Path to the remote symbolic link. 返回值 See the documentation for stat() for details on the values which may be returned. 范例 Example #1 Stating a symbolic link via SFTP 参见 • ssh2_sftp_stat() - Stat a file on a remote filesystem • lstat() - 给出一个文件或符号连接的信息 • stat() - 给出文件的信息
