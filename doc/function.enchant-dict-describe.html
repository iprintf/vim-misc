enchant_dict_describe (PHP 5 >= 5.3.0, PHP 7, PECL enchant >= 0.1.0 ) enchant_dict_describe — Describes an individual dictionary 说明 mixed enchant_dict_describe ( resource $dict ) Returns the details of the dictionary. 参数 dict Dictionary resource 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 A enchant_dict_describe() example Check if a dictionary exists using enchant_broker_dict_exists() and show the detail of it. 以上例程的输出类似于： Array ( [lang] => en_US [name] => aspell [desc] => Aspell Provider [file] => /usr/lib/enchant/libenchant_aspell.so )
