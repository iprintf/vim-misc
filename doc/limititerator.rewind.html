LimitIterator::rewind (PHP 5 >= 5.1.0, PHP 7) LimitIterator::rewind — Rewind the iterator to the specified starting offset 说明 public void LimitIterator::rewind ( void ) Rewinds the iterator to the starting offset specified in LimitIterator::__construct(). 参数 此函数没有参数。 返回值 没有返回值。 参见 • LimitIterator::current() - Get current element • LimitIterator::key() - Get current key • LimitIterator::next() - Move the iterator forward • LimitIterator::seek() - Seek to the given position • LimitIterator::valid() - Check whether the current element is valid
