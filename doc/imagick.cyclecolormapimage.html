Imagick::cycleColormapImage (PECL imagick 2.0.0) Imagick::cycleColormapImage — Displaces an image's colormap 说明 bool Imagick::cycleColormapImage ( int $displace ) Displaces an image's colormap by a given number of positions. If you cycle the colormap a number of times you can produce a psychedelic effect. 参数 displace The amount to displace the colormap. 返回值 成功时返回 TRUE。 错误／异常 错误时抛出 ImagickException。
