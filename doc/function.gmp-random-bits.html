gmp_random_bits (PHP 5 >= 5.6.3, PHP 7) gmp_random_bits — Random number 说明 GMP gmp_random_bits ( integer $bits ) Generate a random number. The number will be between 0 and (2 ** bits) - 1. bits must greater than 0, and the maximum value is restricted by available memory. 参数 bits The number of bits. 可以是一个 GMP 数据 resouce，或一个可以转换为数值的字符串。 返回值 A random GMP number. 范例 Example #1 gmp_random_bits() example 以上例程会输出： 3 15
