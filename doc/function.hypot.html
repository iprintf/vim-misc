hypot (PHP 4 >= 4.1.0, PHP 5, PHP 7) hypot — 计算一直角三角形的斜边长度 说明 float hypot ( float $x , float $y ) hypot() 函数将会跟据直角三角形的两直解边长度 x 和 y 计算其斜边的长度。或者是从 标点 (x, y) 到原点的距离。该函数的算法等同于 sqrt(x*x + y*y)。 参数 x 第一条边的长度 y 第二条边的长度 返回值 计算斜边的长度 User Contributed Notes 1 note =============================================================================== A simpler approach would be to allow an arbitrary number of parameters. That would allow for whatever number of dimensions you want *and* it would be backwards compatible with the current implementation.
