ifx_free_result (PHP 4, PHP <=5.2.0) ifx_free_result — Releases resources for the query 说明 bool ifx_free_result ( resource $result_id ) Releases resources for the query associated with result_id. 参数 result_id result_id is a valid resultid returned by ifx_query() or ifx_prepare() (select type queries only!). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ifx_do() - Execute a previously prepared SQL-statement
