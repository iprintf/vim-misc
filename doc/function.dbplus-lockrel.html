dbplus_lockrel (PHP 4 <= 4.1.0, PECL dbplus >= 0.9) dbplus_lockrel — Request write lock on relation 说明 int dbplus_lockrel ( resource $relation ) Requests a write lock on the given relation. Other clients may still query the relation, but can't alter it while it is locked. 参数 relation A relation opened by dbplus_open(). 返回值 注释 Warning 此函数是实验性的。此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本 中未通知就被修改。使用本函数风险自担。
