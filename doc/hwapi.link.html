hw_api::link (PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN) hw_api::link — Creates a link to an object 说明 bool hw_api::link ( array $parameter ) Creates a link to an object. Accessing this link is like accessing the object to links points to. 参数 parameter The parameter array contains the required elements 'objectIdentifier' and 'destinationParentIdentifier'. 'destinationParentIdentifier' is the target collection. 返回值 The function returns TRUE on success or an error object. 参见 • hw_api::copy
