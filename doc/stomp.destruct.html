Stomp::__destruct stomp_close (PECL stomp >= 0.1.0) Stomp::__destruct -- stomp_close — Closes stomp connection 说明 面向对象风格 (destructor): public bool Stomp::__destruct ( void ) 过程化风格: bool stomp_close ( resource $link ) Closes a previously opened connection. 参数 link 仅对过程化样式：由 stomp_connect() 返回的 stomp 连接标识符。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 See stomp_connect().
