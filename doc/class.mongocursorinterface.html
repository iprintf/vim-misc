The MongoCursorInterface interface (PECL mongo >=1.5.0) Introduction Interface for cursors, which can be used to iterate through results of a database query or command. This interface is implemented by the MongoCursor and MongoCommandCursor classes. Note: Similar to Traversable, this interface cannot be implemented in PHP scripts.
