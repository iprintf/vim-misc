pcntl_wexitstatus (PHP 4 >= 4.1.0, PHP 5, PHP 7) pcntl_wexitstatus — 返回一个中断的子进程的返回代码 说明 int pcntl_wexitstatus ( int $status ) 返回一个中断的子进程的返回代码。这个函数仅在函数pcntl_wifexited()返回 TRUE.时有 效。 参数 status 参数 status 是提供给成功调用 pcntl_waitpid() 时的状态参数。 返回值 返回整形的子进程返回代码。 参见 • pcntl_waitpid() - 等待或返回fork的子进程状态 • pcntl_wifexited() - 检查状态代码是否代表一个正常的退出。
