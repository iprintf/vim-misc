HaruPage::getLineJoin (PECL haru >= 0.0.1) HaruPage::getLineJoin — Get the current line join style 说明 int HaruPage::getLineJoin ( void ) Get the current line join style. 参数 此函数没有参数。 返回值 Returns the current line join style. The result value is one of the following: • HaruPage::MITER_JOIN • HaruPage::ROUND_JOIN • HaruPage::BEVEL_JOIN 错误／异常 发生错误时抛出 HaruException 异常。 参见 • HaruPage::setLineJoin() - Set line join style for the page
