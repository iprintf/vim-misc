SDO_DAS_XML::loadString (^) SDO_DAS_XML::loadString — Returns SDO_DAS_XML_Document for a given xml instance string 说明 SDO_DAS_XML_Document SDO_DAS_XML::loadString ( string $xml_string ) Constructs the tree of SDO_DataObjects from the given xml instance string. Returns SDO_DAS_XML_Document Object. Use SDO_DAS_XML_Document::getRootDataObject method to get root data object. 参数 xml_string xml string. 返回值 Returns SDO_DAS_XML_Document object on Success or throws exception as described. 错误／异常 SDO_TypeNotFoundException Thrown if a type is not defined by the underlying model. SDO_PropertyNotFoundException Thrown if the a property within a type is not defined in the underlying model. SDO_DAS_XML_ParserException Thrown for any problems while parsing the given XSD File.
