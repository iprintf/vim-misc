curl_reset (PHP 5 >= 5.5.0, PHP 7) curl_reset — 重置一个 libcurl 会话句柄的所有的选项 说明 void curl_reset ( resource $ch ) 该函数将给定的 cURL 句柄所有选项重新设置为默认值。 参数 ch 由 curl_init() 返回的 cURL 句柄。 返回值 没有返回值。 范例 Example #1 curl_reset() 示例 注释 Note: curl_reset() also resets the URL given as the curl_init() parameter. 参见 • curl_setopt() - 设置一个cURL传输选项
