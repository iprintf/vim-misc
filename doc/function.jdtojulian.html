JDToJulian (PHP 4, PHP 5, PHP 7) JDToJulian — 转变一个Julian Day计数到Julian历法的日期 说明 string jdtojulian ( int $julianday ) 转变一个Julian Day计数到以“月/日/年”形式显示的Julian历法的日期。 参数 julianday 一个julian天数 返回值 以“月/日/年”形式显示的Julian历法的日期 参见 • juliantojd() - 转变一个Julian历法的日期为Julian Day计数 • cal_from_jd() - 转换Julian Day计数到一个支持的历法。
