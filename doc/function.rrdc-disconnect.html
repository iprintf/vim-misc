rrdc_disconnect (PECL rrd >= 1.1.2) rrdc_disconnect — Close any outstanding connection to rrd caching daemon 说明 void rrdc_disconnect ( void ) Close any outstanding connection to rrd caching daemon. This function is automatically called when the whole PHP process is terminated. It depends on used SAPI. For example, it's called automatically at the end of command line script. It's up user whether he wants to call this function at the end of every request or otherwise. 参数 此函数没有参数。 返回值 没有返回值。
