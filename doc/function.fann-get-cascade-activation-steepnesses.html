fann_get_cascade_activation_steepnesses (PECL fann >= 1.0.0) fann_get_cascade_activation_steepnesses — Returns the cascade activation steepnesses 说明 array fann_get_cascade_activation_steepnesses ( resource $ann ) The cascade activation steepnesses array is an array of the different activation functions used by the candidates. See fann_get_cascade_num_candidates() for a description of which candidate neurons will be generated by this array. The default activation steepnesses are {0.25, 0.50, 0.75, 1.00}. 参数 ann Neural network resource. 返回值 The cascade activation steepnesses, or FALSE on error. 参见 • fann_get_cascade_activation_steepnesses_count() - The number of activation steepnesses • fann_set_cascade_activation_steepnesses() - Sets the array of cascade candidate activation steepnesses
