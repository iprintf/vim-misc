gupnp_service_action_return_error (PECL gupnp >= 0.1.0) gupnp_service_action_return_error — Return error code 说明 bool gupnp_service_action_return_error ( resource $action , int $error_code [, string $error_description ] ) Return error code. 参数 action A service action identifier. error_code The error code. Signal can be one of the following values or user defined: GUPNP_CONTROL_ERROR_INVALID_ACTION The action name was invalid. GUPNP_CONTROL_ERROR_INVALID_ARGS The action arguments were invalid. GUPNP_CONTROL_ERROR_OUT_OF_SYNC Out of sync (deprecated). GUPNP_CONTROL_ERROR_ACTION_FAILED The action failed. error_description The error description. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • gupnp_service_action_return() - Return successfully
