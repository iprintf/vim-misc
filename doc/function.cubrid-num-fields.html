cubrid_num_fields (PECL CUBRID >= 8.3.0) cubrid_num_fields — Return the number of columns in the result set 说明 int cubrid_num_fields ( resource $result ) This function returns the number of columns in the result set, on success, or it returns FALSE on failure. 参数 result result comes from a call to cubrid_execute(), cubrid_query() and cubrid_prepare() 返回值 Number of columns, on success. -1 if SQL sentence is not SELECT. FALSE when process is unsuccessful. 范例 Example #1 cubrid_num_fields() example 以上例程会输出： Row Num: 6 Column Num: 2
