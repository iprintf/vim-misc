fbsql_free_result (PHP 4 >= 4.0.6, PHP 5, PHP 7) fbsql_free_result — Free result memory 说明 bool fbsql_free_result ( resource $result ) Frees all memory associated with the given result identifier. fbsql_free_result() only needs to be called if you are concerned about how much memory is being used for queries that return large result sets. All associated result memory is automatically freed at the end of the script's execution. 参数 result 由 fbsql_query() 或 fbsql_db_query() 返回的结果标识符。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
