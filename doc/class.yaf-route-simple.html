The Yaf_Route_Simple class (Yaf >=1.0.0) 简介 Yaf_Route_Simple 会匹配请求中的query string，然后找到路由信息。 你需要做的只是告诉 Yaf_Route_Simple，在$_GET中哪个是Module，哪个是Controller， 哪个是Action。 Yaf_Route_Simple::route() 总是会返回TRUE，所以把Yaf_Route_Simple放在路由堆栈前 面是很重要的，否则其他所有的路由都可能不会被调用到。 类摘要 Yaf_Route_Simple implements Yaf_Route_Interface { /* 属性 */ protected $controller ; protected $module ; protected $action ; /* 方法 */ public string assemble ( array $info [, array $query ] ) public__construct ( string $module_name , string $controller_name , string $action_name ) public bool route ( Yaf_Request_Abstract $request ) } 属性 controller module action Table of Contents • Yaf_Route_Simple::assemble — 组合url • Yaf_Route_Simple::__construct — Yaf_Route_Simple constructor • Yaf_Route_Simple::route — Route a request
