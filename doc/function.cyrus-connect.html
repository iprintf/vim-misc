cyrus_connect (PHP 4 >= 4.1.0, PECL cyrus 1.0) cyrus_connect — Connect to a Cyrus IMAP server 说明 resource cyrus_connect ([ string $host [, string $port [, int $flags ]]] ) Connects to a Cyrus IMAP server. 参数 host The Cyrus IMAP host name. port The port number. flags 返回值 Returns a connection handler on success 或者在失败时返回 FALSE.
