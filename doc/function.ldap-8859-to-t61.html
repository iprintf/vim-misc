ldap_8859_to_t61 (PHP 4 >= 4.0.2, PHP 5, PHP 7) ldap_8859_to_t61 — Translate 8859 characters to t61 characters 说明 string ldap_8859_to_t61 ( string $value ) Translate ISO-8859 characters to t61 characters. This function is useful if you have to talk to a legacy LDAPv2 server. 参数 value The text to be translated. 返回值 Return the t61 translation of value. 参见 • ldap_t61_to_8859() - Translate t61 characters to 8859 characters User Contributed Notes 1 note =============================================================================== Description: ldap_8859_to_t61() assists in the conversion of strings between ISO 8859-1 and t61 encodings. If the string cannot be converted successfully, the function returns FALSE. Example: $converted = ldap_8859_to_t61($data); echo $converted;
