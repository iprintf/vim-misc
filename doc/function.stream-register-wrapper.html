User Contributed Notes 1 note =============================================================================== To get the https wrapper working on Windows, you need to replace your php4ts.dll with the one provided at http://ftp.proventum.net/pub/php/win32/misc /openssl/. This is instead of using the php_openssl.dll, so be sure to comment this extension in your php.ini.
