OCI-Collection::getElem (PHP 5, PHP 7, PECL OCI8 >= 1.1.0) OCI-Collection::getElem — Returns value of the element 说明 mixed OCI-Collection::getElem ( int $index ) Returns element's value with the index index (0-based). 参数 index The element index. First index is 0. 返回值 Returns FALSE if such element doesn't exist; NULL if element is NULL; string if element is column of a string datatype or number if element is numeric field. 参见 • OCI-Collection::assignElem
