fbsql_rows_fetched (PHP 5 >= 5.1.0, PHP 7) fbsql_rows_fetched — Get the number of rows affected by the last statement 说明 int fbsql_rows_fetched ( resource $result ) Gets the number of rows affected by the last statement. 参数 result 由 fbsql_query() 或 fbsql_db_query() 返回的结果标识符。 返回值 Returns the number of rows, as an integer.
