ldap_next_attribute (PHP 4, PHP 5, PHP 7) ldap_next_attribute — Get the next attribute in result 说明 string ldap_next_attribute ( resource $link_identifier , resource $result_entry_identifier ) Retrieves the attributes in an entry. The first call to ldap_next_attribute() is made with the result_entry_identifier returned from ldap_first_attribute(). 参数 link_identifier An LDAP link identifier, returned by ldap_connect(). result_entry_identifier ber_identifier The internal state of the pointer is maintained by this parameter. Note: This parameter is no longer used as this is now handled automatically by PHP. For backwards compatibility PHP will not throw an error if this parameter is passed. 返回值 Returns the next attribute in an entry on success and FALSE on error. 更新日志 版本 说明 5.2.4 The ber_identifier was removed. This is now handled automatically by PHP. 参见 • ldap_get_attributes() - Get attributes from a search result entry
