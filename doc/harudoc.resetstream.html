HaruDoc::resetStream (PECL haru >= 0.0.1) HaruDoc::resetStream — Rewind the temporary stream 说明 bool HaruDoc::resetStream ( void ) Rewinds the temporary stream of the document. 参数 此函数没有参数。 返回值 Returns TRUE on success. 错误／异常 发生错误时抛出 HaruException 异常。 参见 • HaruDoc::saveToStream() - Save the document into a temporary stream • HaruDoc::getStreamSize() - Get the size of the temporary stream • HaruDoc::readFromStream() - Read data from the temporary stream
