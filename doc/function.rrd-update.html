rrd_update (PECL rrd >= 0.9.0) rrd_update — Updates the RRD database. 说明 bool rrd_update ( string $filename , array $options ) Updates the RRD database file. The input data is time interpolated according to the properties of the RRD database file. 参数 filename RRD database file name. This database will be updated. options Options for updating the RRD database. This is list of strings. See man page of rrd update for whole list of options. 返回值 Returns TRUE on success, FALSE when error occurs.
