streamWrapper::dir_rewinddir (PHP 4 >= 4.3.2, PHP 5, PHP 7) streamWrapper::dir_rewinddir — Rewind directory handle 说明 public bool streamWrapper::dir_rewinddir ( void ) This method is called in response to rewinddir(). Should reset the output generated by streamWrapper::dir_readdir(). i.e.: The next call to streamWrapper::dir_readdir() should return the first entry in the location returned by streamWrapper::dir_opendir(). 参数 此函数没有参数。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • rewinddir() - 倒回目录句柄 • streamWrapper::dir_readdir() - Read entry from directory handle
