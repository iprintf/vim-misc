ps_close (PECL ps >= 1.1.0) ps_close — Closes a PostScript document 说明 bool ps_close ( resource $psdoc ) Closes the PostScript document. This function writes the trailer of the PostScript document. It also writes the bookmark tree. ps_close() does not free any resources, which is done by ps_delete(). This function is also called by ps_delete() if it has not been called before. 参数 psdoc Resource identifier of the postscript file as returned by ps_new(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ps_open_file() - Opens a file for output • ps_delete() - Deletes all resources of a PostScript document
