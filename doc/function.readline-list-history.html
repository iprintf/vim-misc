readline_list_history (PHP 4, PHP 5, PHP 7) readline_list_history — 获取命令历史列表 说明 array readline_list_history ( void ) 获取整个命令行历史. 返回值 返回整个命令行历史的数组.元素的整数索引从零开始 User Contributed Notes 1 note =============================================================================== Note this function is only available when PHP is compiled with libreadline, not if it is compiled with libedit.
