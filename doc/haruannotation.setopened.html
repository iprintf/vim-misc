HaruAnnotation::setOpened (PECL haru >= 0.0.1) HaruAnnotation::setOpened — Set the initial state of the annotation 说明 bool HaruAnnotation::setOpened ( bool $opened ) Defines whether the annotation is initially displayed open. This function may be used with text annotations only. 参数 opened TRUE means the annotation is initially displayed open, FALSE - means it's closed. 返回值 Returns TRUE on success. 错误／异常 发生错误时抛出 HaruException 异常。
