The CairoLineCap class (No version information available, might only be in Git) 简介 Specifies how to render the endpoints of the path when stroking. The default line cap style is CairoLineCap::BUTT. 类摘要 CairoLineCap { /* 常量 */ const integer BUTT = 0 ; const integer ROUND = 1 ; const integer SQUARE = 2 ; } 预定义常量 CairoLineCap::BUTT Start(stop) the line exactly at the start(end) point CairoLineCap::ROUND Use a round ending, the center of the circle is the end point CairoLineCap::SQUARE Use squared ending, the center of the square is the end point
