pspell_save_wordlist (PHP 4 >= 4.0.2, PHP 5, PHP 7) pspell_save_wordlist — Save the personal wordlist to a file 说明 bool pspell_save_wordlist ( int $dictionary_link ) pspell_save_wordlist() saves the personal wordlist from the current session. The location of files to be saved specified with pspell_config_personal() and (optionally) pspell_config_repl(). 参数 dictionary_link A dictionary link identifier opened with pspell_new_personal(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 pspell_add_to_personal() 注释 Note: This function will not work unless you have pspell .11.2 and aspell .32.5 or later.
