xhprof_disable (PECL xhprof >= 0.9.0) xhprof_disable — 停止 xhprof 分析器 说明 array xhprof_disable ( void ) 停止性能分析，并返回此次运行的 xhprof 数据。 参数 此函数没有参数。 返回值 本次运行的 array 类型的 xhprof 数据。 范例 Example #1 xhprof_disable() 范例 以上例程的输出类似于： Array ( [main()==>strlen] => Array ( [ct] => 1 [wt] => 279 ) [main()==>xhprof_disable] => Array ( [ct] => 1 [wt] => 9 ) [main()] => Array ( [ct] => 1 [wt] => 610 ) ) User Contributed Notes 1 note =============================================================================== Raw output snippet with, XHPROF_FLAGS_CPU + XHPROF_FLAGS_MEMORY.
