SoapClient::__getFunctions (PHP 5 >= 5.0.1, PHP 7) SoapClient::__getFunctions — Returns list of available SOAP functions 说明 public array SoapClient::__getFunctions ( void ) Returns an array of functions described in the WSDL for the Web service. Note: 此函数仅在 WSDL 模式下生效。 参数 此函数没有参数。 返回值 The array of SOAP function prototypes, detailing the return type, the function name and type-hinted paramaters. 范例 Example #1 SoapClient::__getFunctions() example __getFunctions()); ?> 以上例程会输出： array(26) { [0]=> string(70) "ProductInfo KeywordSearchRequest(KeywordRequest $KeywordSearchRequest)" [1]=> string(79) "ProductInfo TextStreamSearchRequest(TextStreamRequest $TextStreamSearchRequest)" [2]=> string(64) "ProductInfo PowerSearchRequest(PowerRequest $PowerSearchRequest)" ...
