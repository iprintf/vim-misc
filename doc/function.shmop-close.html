shmop_close (PHP 4 >= 4.0.4, PHP 5, PHP 7) shmop_close — Close shared memory block 说明 void shmop_close ( int $shmid ) shmop_close() is used to close a shared memory block. 参数 shmid The shared memory block identifier created by shmop_open() 返回值 没有返回值。 范例 Example #1 Closing shared memory block This example will close shared memory block identified by $shm_id. 参见 • shmop_open() - Create or open shared memory block User Contributed Notes 1 note =============================================================================== shmop_close doesn't delete the memory segment, it just detaches from it. If you have created the block and need to delete it you must call shmop_delete **BEFORE** calling shmop_close (for reasons outlined in shmop_delete help page notes).
