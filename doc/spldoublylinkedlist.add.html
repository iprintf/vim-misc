SplDoublyLinkedList::add (PHP 5 >= 5.5.0, PHP 7) SplDoublyLinkedList::add — Add/insert a new value at the specified index 说明 public void SplDoublyLinkedList::add ( mixed $index , mixed $newval ) Insert the value newval at the specified index, shuffling the previous value at that index (and all subsequent values) up through the list. 参数 index The index where the new value is to be inserted. newval The new value for the index. 返回值 没有返回值。 错误／异常 Throws OutOfRangeException when index is out of bounds or when index cannot be parsed as an integer.
