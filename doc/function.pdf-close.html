PDF_close (PHP 4, PECL pdflib >= 1.0.0) PDF_close — Close pdf resource [deprecated] 说明 bool PDF_close ( resource $p ) Closes the generated PDF file, and frees all document-related resources. 成功时 返回 TRUE，或者在失败时返回 FALSE。 This function is deprecated since PDFlib version 6, use PDF_end_document() instead.
