SolrUtils::digestXmlResponse (PECL solr >= 0.9.2) SolrUtils::digestXmlResponse — Parses an response XML string into a SolrObject 说明 public static SolrObject SolrUtils::digestXmlResponse ( string $xmlresponse [, int $parse_mode = 0 ] ) This method parses an response XML string from the Apache Solr server into a SolrObject. It throws a SolrException if there was an error. 参数 xmlresponse The XML response string from the Solr server. parse_mode Use SolrResponse::PARSE_SOLR_OBJ or SolrResponse::PARSE_SOLR_DOC 返回值 Returns the SolrObject representing the XML response. If the parse_mode parameter is set to SolrResponse::PARSE_SOLR_OBJ Solr documents will be parses as SolrObject instances. If it is set to SolrResponse::PARSE_SOLR_DOC, they will be parsed as SolrDocument instances.
