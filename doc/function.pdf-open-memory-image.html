PDF_open_memory_image (PHP 4, PECL pdflib >= 1.0.0) PDF_open_memory_image — Open image created with PHP's image functions [not supported] 说明 int PDF_open_memory_image ( resource $p , resource $image ) This function is not supported by PDFlib GmbH. User Contributed Notes 1 note =============================================================================== note that in PHP 4.2.2 and PDF_lib 4.0.1 this function does not support more than 256 colors (only GD 1.x it seems). WORKS: $im = ImageCreate(100, 100); $pim = pdf_open_memory_image($pdf, $im); DOES NOT WORK: $im = ImageCreateTrueColor(100, 100); $pim = pdf_open_memory_image($pdf, $im); DOES NOT WORK: $im = ImageCreateFromJPEG("file.jpg"); $pim = pdf_open_memory_image($pdf, $im);
