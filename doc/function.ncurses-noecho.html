ncurses_noecho (PHP 4 >= 4.0.7, PHP 5 < 5.3.0, PECL ncurses >= 1.0.0) ncurses_noecho — Switch off keyboard input echo 说明 bool ncurses_noecho ( void ) Warning 此函数是实验性的。此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本 中未通知就被修改。使用本函数风险自担。 Prevents echoing of user typed characters. 返回值 Returns TRUE if any error occurred, FALSE otherwise. 参见 • ncurses_echo() - Activate keyboard input echo • ncurses_getch() - Read a character from keyboard User Contributed Notes 1 note =============================================================================== The ncurses_noecho() function basically disables user input from being echoed to your present cursor location. Re-enable with the ncurses_echo() function. Echo is enabled by default in a new ncurses session.
