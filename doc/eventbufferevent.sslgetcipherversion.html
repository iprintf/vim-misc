EventBufferEvent::sslGetCipherVersion (PECL event >= 1.10.0) EventBufferEvent::sslGetCipherVersion — Returns version of cipher used by current SSL connection. 说明 public string EventBufferEvent::sslGetCipherVersion ( void ) Retrieves version of cipher used by current SSL connection. Note: This function is available only if Event is compiled with OpenSSL support. 参数 此函数没有参数。 返回值 Returns the current cipher version of the SSL connection, or FALSE on error.
