ZMQSocket::recvMulti (PECL zmq >= 0.8.0) ZMQSocket::recvMulti — Receives a multipart message 说明 public string ZMQSocket::recvMulti ([ integer $mode = 0 ] ) Receive an array multipart message from a socket. By default receiving will block until a message is available unless ZMQ::MODE_NOBLOCK flag is used. 参数 mode Pass mode flags to receive multipart messages or non-blocking operation. See ZMQ::MODE_* constants. 返回值 Returns the array of message parts. Throws ZMQSocketException in error. If ZMQ::MODE_NOBLOCK is used and the operation would block boolean false shall be returned.
