SplFileInfo::getPathname (PHP 5 >= 5.1.2, PHP 7) SplFileInfo::getPathname — Gets the path to the file 说明 public string SplFileInfo::getPathname ( void ) Returns the path to the file. 参数 此函数没有参数。 返回值 The path to the file. 范例 Example #1 SplFileInfo::getPathname() example getPathname()); ?> 以上例程的输出类似于： string(12) "/usr/bin/php" 参见 • SplFileInfo::getRealPath() - Gets absolute path to file User Contributed Notes 1 note =============================================================================== In Windows, getPathname might return a path with a mix of backslashes and forward slashes ('C:/some/path/to\a\file.txt'). Observed in PHP 5.2.9. By contrast, getRealPath will return a consistent result (all backslashes).
