gupnp_context_get_port (PECL gupnp >= 0.1.0) gupnp_context_get_port — Get the port 说明 int gupnp_context_get_port ( resource $context ) Get the port that the SOAP server is running on. 参数 context A context identifier, returned by gupnp_context_new(). 返回值 Returns the port number for the current context and FALSE on error. 范例 Example #1 Create new UPnP context and get port number 参见 • gupnp_context_new() - Create a new context • gupnp_context_get_host_ip() - Get the IP address
