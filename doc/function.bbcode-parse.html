bbcode_parse (PECL bbcode >= 0.9.0) bbcode_parse — Parse a string following a given rule set 说明 string bbcode_parse ( resource $bbcode_container , string $to_parse ) This function parse the string to_parse following the rules in the bbcode_container created by bbcode_create() 参数 bbcode_container BBCode_Container resource returned by bbcode_create(). to_parse The string we need to parse. 返回值 Returns the parsed string, 或者在失败时返回 FALSE.
