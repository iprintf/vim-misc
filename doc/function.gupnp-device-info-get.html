gupnp_device_info_get (PECL gupnp >= 0.1.0) gupnp_device_info_get — Get info of root device 说明 array gupnp_device_info_get ( resource $root_device ) Get info of root device. 参数 root_device A root device identifier, returned by gupnp_root_device_new(). 返回值 Return array wich contains the information of the root device (like location, url, udn and etc).
