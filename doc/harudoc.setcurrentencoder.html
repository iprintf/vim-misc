HaruDoc::setCurrentEncoder (PECL haru >= 0.0.1) HaruDoc::setCurrentEncoder — Set the current encoder for the document 说明 bool HaruDoc::setCurrentEncoder ( string $encoding ) Defines the encoder currently used in the document. 参数 encoding The name of the encoding to use. See Builtin Encodings for the list of allowed values. 返回值 Returns TRUE on success. 错误／异常 发生错误时抛出 HaruException 异常。
