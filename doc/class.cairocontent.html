The CairoContent class (No version information available, might only be in Git) 简介 CairoContent is used to describe the content that a surface will contain, whether color information, alpha information (translucence vs. opacity), or both. Note: The large values here are designed to keep CairoContent values distinct from CairoContent values so that the implementation can detect the error if users confuse the two types. 类摘要 CairoContent { /* 常量 */ const integer COLOR = 4096 ; const integer ALPHA = 8192 ; const integer COLOR_ALPHA = 12288 ; } 预定义常量 CairoContent::COLOR The surface will hold color content only. CairoContent::ALPHA The surface will hold alpha content only. CairoContent::COLOR_ALPHA The surface will hold color and alpha content.
