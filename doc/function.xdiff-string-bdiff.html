xdiff_string_bdiff (PECL xdiff >= 1.5.0) xdiff_string_bdiff — Make binary diff of two strings 说明 string xdiff_string_bdiff ( string $old_data , string $new_data ) Makes a binary diff of two strings and returns the result. This function works with both text and binary data. Resulting patch can be later applied using xdiff_string_bpatch()/xdiff_file_bpatch(). 参数 old_data First string with binary data. It acts as "old" data. new_data Second string with binary data. It acts as "new" data. 返回值 Returns string with binary diff containing differences between "old" and "new" data or FALSE if an internal error occurred. 参见 • xdiff_string_bpatch() - Patch a string with a binary diff
