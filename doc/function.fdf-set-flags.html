fdf_set_flags (PHP 4 >= 4.0.2, PHP 5, PHP 7) fdf_set_flags — Sets a flag of a field 说明 bool fdf_set_flags ( resource $fdf_document , string $fieldname , int $whichFlags , int $newFlags ) Sets certain flags of the given field. 参数 fdf_document The FDF document handle, returned by fdf_create(), fdf_open() or fdf_open_string(). fieldname Name of the FDF field, as a string. whichFlags newFlags 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • fdf_set_opt() - Sets an option of a field
