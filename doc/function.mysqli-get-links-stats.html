mysqli_get_links_stats (PHP 5 >= 5.6.0, PHP 7) mysqli_get_links_stats — Return information about open and cached links 说明 array mysqli_get_links_stats ( void ) mysqli_get_links_stats() returns information about open and cached MySQL links. 参数 此函数没有参数。 返回值 mysqli_get_links_stats() returns an associative array with three elements, keyed as follows: total An integer indicating the total number of open links in any state. active_plinks An integer representing the number of active persistent connections. cached_plinks An integer representing the number of inactive persistent connections.
