pg_lo_read (PHP 4 >= 4.2.0, PHP 5, PHP 7) pg_lo_read — 从大型对象中读入数据 说明 string pg_lo_read ( resource $large_object , int $len ) pg_lo_read() 从大型对象中读入最多 len 字节的数据并以字符串返回。large_object 指 定了有效的大型对象的资源号，len 则指定了大型对象的段所允许的最大长度。如果出错 则返回 FALSE。 要使用大型对象（lo）接口，需要将其放置在事务块中。 Note: 本函数以前的名字为 pg_loread()。 参见 pg_lo_read_all()。 User Contributed Notes 1 note =============================================================================== Be careful: pg_lo_export() create empty file without "begin"/"end". See docs: "...it is necessary to enclose it within a transaction block..."
