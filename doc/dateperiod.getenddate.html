DatePeriod::getEndDate (PHP 5 >= 5.6.5, PHP 7) DatePeriod::getEndDate — Gets the end date 说明 面向对象风格 public DateTimeInterface DatePeriod::getEndDate ( void ) Gets the end date of the period. 参数 此函数没有参数。 返回值 Returns NULL if the DatePeriod does not have an end date. For example, when initialized with the recurrences parameter, or the isostr parameter without an end date. Returns a DateTimeImmutable object when the DatePeriod is initialized with a DateTimeImmutable object as the start parameter.
