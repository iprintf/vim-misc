ImagickPixel::getHSL (PECL imagick 2.0.0) ImagickPixel::getHSL — Returns the normalized HSL color of the ImagickPixel object 说明 array ImagickPixel::getHSL ( void ) Returns the normalized HSL color described by the ImagickPixel object, with each of the three values as floating point numbers between 0.0 and 1.0. 返回值 Returns the HSL value in an array with the keys "hue", "saturation", and "luminosity". Throws ImagickPixelException on failure. 范例 Example #1 Basic Imagick::getHSL() example getHSL(); print_r($colorInfo); ?> 以上例程会输出： Array ( [hue] => 0 [saturation] => 0.80001220740379 [luminosity] => 0.50000762951095 ) 注释 Note: Available with ImageMagick library version 6.2.9 and higher.
