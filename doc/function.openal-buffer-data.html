openal_buffer_data (PECL openal >= 0.1.0) openal_buffer_data — Load a buffer with data 说明 bool openal_buffer_data ( resource $buffer , int $format , string $data , int $freq ) 参数 buffer An Open AL(Buffer) resource (previously created by openal_buffer_create()). format Format of data, one of: AL_FORMAT_MONO8, AL_FORMAT_MONO16, AL_FORMAT_STEREO8 和 AL_FORMAT_STEREO16 data Block of binary audio data in the format and freq specified. freq Frequency of data given in Hz. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • openal_buffer_loadwav() - Load a .wav file into a buffer • openal_stream() - Begin streaming on a source
