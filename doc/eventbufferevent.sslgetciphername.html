EventBufferEvent::sslGetCipherName (PECL event >= 1.10.0) EventBufferEvent::sslGetCipherName — Returns the current cipher name of the SSL connection. 说明 public string EventBufferEvent::sslGetCipherName ( void ) Retrieves name of cipher used by current SSL connection. Note: This function is available only if Event is compiled with OpenSSL support. 参数 此函数没有参数。 返回值 Returns the current cipher name of the SSL connection, or FALSE on error.
