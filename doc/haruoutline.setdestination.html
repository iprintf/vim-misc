HaruOutline::setDestination (PECL haru >= 0.0.1) HaruOutline::setDestination — Set the destination for the outline 说明 bool HaruOutline::setDestination ( object $destination ) Sets a destination object which becomes a target to jump to when the outline is clicked. 参数 destination A valid HaruDestination instance. 返回值 Returns TRUE on success. 错误／异常 发生错误时抛出 HaruException 异常。
