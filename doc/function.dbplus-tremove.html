dbplus_tremove (PHP 4 <= 4.1.0, PECL dbplus >= 0.9) dbplus_tremove — Remove tuple and return new current tuple 说明 int dbplus_tremove ( resource $relation , array $tuple [, array &$current ] ) dbplus_tremove() removes tuple from relation if it perfectly matches a tuple within the relation. current, if given, will contain the data of the new current tuple after calling dbplus_tremove(). 参数 relation A relation opened by dbplus_open(). tuple current 返回值 注释 Warning 此函数是实验性的。此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本 中未通知就被修改。使用本函数风险自担。
