MongoLog::setModule (PECL mongo >= 1.2.3) MongoLog::setModule — Sets the module(s) to be logged 说明 public static void MongoLog::setModule ( int $module ) This function can be used to set which driver modules should be logged. The MongoLog module constants may be used with bitwise operators to specify multiple modules. Note that you must also call MongoLog::setLevel() to enable logging. 参数 module The module(s) you would like to log.
