chdb::get (PECL chdb >= 0.1.0) chdb::get — Gets the value associated with a key 说明 public string chdb::get ( string $key ) Gets the value associated with a key from a chdb database. 参数 key The key for which to get the value. 返回值 Returns a string containing the value associated with the given key, or NULL if not found. 范例 Example #1 chdb::get() example 'value1', 'key2' => 'value2', // ... ); chdb_create('data.chdb', $data); $chdb = new chdb('data.chdb'); $value1 = $chdb->get('key1'); $value2 = $chdb->get('key2'); echo $value1, PHP_EOL; echo $value2, PHP_EOL; ?> 以上例程的输出类似于： value1 value2
