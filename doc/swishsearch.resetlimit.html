SwishSearch::resetLimit (PECL swish >= 0.1.0) SwishSearch::resetLimit — Reset the search limits 说明 void SwishSearch::resetLimit ( void ) Warning 此函数是实验性的。此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本 中未通知就被修改。使用本函数风险自担。 Reset the search limits previous set by SwishSearch::setLimit. 返回值 没有返回值。 范例 Example #1 Basic SwishSearch::resetLimit() example prepare(); $results = $search->execute("time"); echo "First query found: ", $results->hits, " hits\n"; $search->setLimit("swishdocsize", "3000", "6000"); // limit by document size, from 3000 to 6000 bytes $results = $search->execute("time"); echo "Second query found: ", $results->hits, " hits\n"; $search->resetLimit(); $results = $search->execute("time"); echo "Third query found: ", $results->hits, " hits\n"; } catch (SwishException $e) { echo $e->getMessage(), "\n"; } ?> 以上例程的输出类似于： First query found: 5 hits Second query found: 2 hits Third query found: 5 hits
