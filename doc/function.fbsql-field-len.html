fbsql_field_len (PHP 4 >= 4.0.6, PHP 5, PHP 7) fbsql_field_len — Returns the length of the specified field 说明 int fbsql_field_len ( resource $result [, int $field_offset ] ) Returns the length of the specified field. 参数 result A result pointer returned by fbsql_list_fields(). field_offset The numerical offset of the field. The field index starts at 0. 返回值 Returns the length of the specified field.
