DOMXPath::__construct (PHP 5, PHP 7) DOMXPath::__construct — Creates a new DOMXPath object 说明 public DOMXPath::__construct ( DOMDocument $doc ) Creates a new DOMXPath object. 参数 doc The DOMDocument associated with the DOMXPath. User Contributed Notes 2 notes =============================================================================== The DOMXPath object doesn't create a reference to the document. If you load a new xml using either load, loadXML or loadHTML functions the DOMXPath object will still query and evaluate over the source in the DOMDocument when it was created. =============================================================================== As of PHP 5.2.0, DOMXPath object implements XPath 1.0.
