trader_mult (PECL trader >= 0.2.0) trader_mult — Vector Arithmetic Mult 说明 array trader_mult ( array $real0 , array $real1 ) Calculates the vector dot product of real0 with real1 and returns the resulting vector. 参数 real0 浮点数数组。 real1 浮点数数组。 返回值 Returns an array with calculated data or false on failure.
