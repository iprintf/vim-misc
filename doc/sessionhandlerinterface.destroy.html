SessionHandlerInterface::destroy (PHP 5 >= 5.4.0, PHP 7) SessionHandlerInterface::destroy — Destroy a session 说明 abstract public bool SessionHandlerInterface::destroy ( string $session_id ) Destroys a session. Called by session_regenerate_id() (with $destroy = TRUE), session_destroy() and when session_decode() fails. 参数 session_id The session ID being destroyed. 返回值 会话存储的返回值（通常成功返回 0，失败返回 1）。
