gnupg_export (PECL gnupg >= 0.1) gnupg_export — Exports a key 说明 string gnupg_export ( resource $identifier , string $fingerprint ) Exports the key fingerprint. 参数 identifier gnupg 标识符，由对 gnupg_init() 或 gnupg 的调用生成。 fingerprint 指纹键名。 返回值 On success, this function returns the keydata.
