event_base_reinit (PECL libevent >= 0.1.0) event_base_reinit — Reinitialize the event base after a fork 说明 bool event_base_reinit ( resource $event_base ) Some event mechanisms do not survive across fork. The event_base needs to be reinitialized with this function. 参数 event_base Valid event base resource that needs to be re-initialized. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
