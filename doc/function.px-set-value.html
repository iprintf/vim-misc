px_set_value (PECL paradox >= 1.1.0) px_set_value — Sets a value 说明 bool px_set_value ( resource $pxdoc , string $name , float $value ) Sets various values. 参数 pxdoc Resource identifier of the paradox database as returned by px_new(). name name can be one of the following. numprimkeys The number of primary keys. Paradox databases always use the first numprimkeys fields for the primary index. value 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 px_set_parameter() - Sets a parameter
