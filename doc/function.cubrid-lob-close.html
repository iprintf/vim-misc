cubrid_lob_close (PECL CUBRID >= 8.3.1) cubrid_lob_close — Close BLOB/CLOB data 说明 bool cubrid_lob_close ( array $lob_identifier_array ) cubrid_lob_close() is used to close all BLOB/CLOB returned from cubrid_lob_get (). 参数 lob_identifier_array LOB identifier array return from cubrid_lob_get. 返回值 TRUE, when process is successful. FALSE, when process is unsuccessful. 范例 Example #1 cubrid_lob_close() example 参见 • cubrid_lob_get() - Get BLOB/CLOB data • cubrid_lob_size() - Get BLOB/CLOB data size • cubrid_lob_export() - Export BLOB/CLOB data to file • cubrid_lob_send() - Read BLOB/CLOB data and send straight to browser
