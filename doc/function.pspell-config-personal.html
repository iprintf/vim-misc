pspell_config_personal (PHP 4 >= 4.0.2, PHP 5, PHP 7) pspell_config_personal — Set a file that contains personal wordlist 说明 bool pspell_config_personal ( int $dictionary_link , string $file ) Set a file that contains personal wordlist. The personal wordlist will be loaded and used in addition to the standard one after you call pspell_new_config(). The file is also the file where pspell_save_wordlist() will save personal wordlist to. pspell_config_personal() should be used on a config before calling pspell_new_config(). 参数 dictionary_link file The personal wordlist. If the file does not exist, it will be created. The file should be writable by whoever PHP runs as (e.g. nobody). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 pspell_config_personal() 注释 Note: This function will not work unless you have pspell .11.2 and aspell .32.5 or later. User Contributed Notes 1 note =============================================================================== There is no error if the file does not exists. So if you are debugging, check wheter the file really exists.
