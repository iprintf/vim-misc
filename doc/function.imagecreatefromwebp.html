imagecreatefromwebp (PHP 5 >= 5.5.0, PHP 7) imagecreatefromwebp — 由文件或 URL 创建一个新图象。 说明 resource imagecreatefromwebp ( string $filename ) imagecreatefromwebp() 返回从给定文件名创建的图像标示符。 Tip 如已启用fopen 包装器，在此函数中， URL 可作为文件名。关于如何指定文件名详见 fopen()。各种 wapper 的不同功能请参见支持的协议和封装协议，注意其用法及其可提供 的预定义变量。 参数 filename WebP 图像文件路径。 返回值 成功后返回图象资源,失败后返回 FALSE 。 范例 Example #1 Convert an WebP image to a jpeg image using imagecreatefromwebp()
