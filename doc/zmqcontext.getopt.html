ZMQContext::getOpt (PECL zmq >= 1.0.4) ZMQContext::getOpt — Get context option 说明 public mixed ZMQContext::getOpt ( string $key ) Returns the value of a context option. 参数 key An integer representing the option. See the ZMQ::CTXOPT_* constants. 返回值 Returns either a string or an integer depending on key. Throws ZMQContextException on error.
