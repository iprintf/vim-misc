rrd_xport (PECL rrd >= 0.9.0) rrd_xport — Exports the information about RRD database. 说明 array rrd_xport ( array $options ) Exports the information about RRD database file. This data can be converted to XML file via user space PHP script and then restored back as RRD database file. 参数 options Array of options for the export, see rrd xport man page. 返回值 Array with information about RRD database file, FALSE when error occurs.
