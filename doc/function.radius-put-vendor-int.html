radius_put_vendor_int (PECL radius >= 1.1.0) radius_put_vendor_int — Attaches a vendor specific integer attribute 说明 bool radius_put_vendor_int ( resource $radius_handle , int $vendor , int $type , int $value [, int $options = 0 [, int $tag ]] ) Attaches a vendor specific integer attribute to the current RADIUS request. Note: A request must be created via radius_create_request() before this function can be called. 参数 radius_handle The RADIUS resource. vendor The vendor ID. type The attribute type. value The attribute value. options A bitmask of the attribute options. The available options include RADIUS_OPTION_TAGGED and RADIUS_OPTION_SALT. tag The attribute tag. This parameter is ignored unless the RADIUS_OPTION_TAGGED option is set. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 更新日志 版本 说明 PECL radius 1.3.0 The options and tag parameters were added. 参见 • radius_put_vendor_string() - Attaches a vendor specific string attribute
