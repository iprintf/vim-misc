yaz_close (PHP 4 >= 4.0.1, PECL yaz >= 0.9.0) yaz_close — Close YAZ connection 说明 bool yaz_close ( resource $id ) Closes the connection given by parameter id. Note: This function will only close a non-persistent connection opened by setting the persistent option to FALSE with yaz_connect(). 参数 id The connection resource returned by yaz_connect(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • yaz_connect() - Prepares for a connection to a Z39.50 server
