ZipArchive::getCommentName (PHP 5 >= 5.2.0, PHP 7, PECL zip >= 1.4.0) ZipArchive::getCommentName — Returns the comment of an entry using the entry name 说明 string ZipArchive::getCommentName ( string $name [, int $flags ] ) Returns the comment of an entry using the entry name. 参数 name Name of the entry flags If flags is set to ZipArchive::FL_UNCHANGED, the original unchanged comment is returned. 返回值 Returns the comment on success 或者在失败时返回 FALSE. 范例 Example #1 Dump an entry comment open('test1.zip'); if ($res === TRUE) { var_dump($zip->getCommentName('test/entry1.txt')); } else { echo 'failed, code:' . $res; } ?>
