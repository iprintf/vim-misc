readline_read_history (PHP 4, PHP 5, PHP 7) readline_read_history — 读取命令历史 说明 bool readline_read_history ([ string $filename ] ) 这个函数从一个文件读取命令历史 参数 filename 保存了命令历史的文件的路径. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 User Contributed Notes 1 note =============================================================================== Note that the first line in the history file must contain the string: _HiStOrY_V2_ Else it wont work.
