getmygid (PHP 4 >= 4.1.0, PHP 5, PHP 7) getmygid — 获取当前 PHP 脚本拥有者的 GID 说明 int getmygid ( void ) 获取当前 PHP 脚本拥有者的用户组 ID。 返回值 返回当前 PHP 脚本拥有者的用户组 ID，或在错误时返回 FALSE。 参见 • getmyuid() - 获取 PHP 脚本所有者的 UID • getmypid() - 获取 PHP 进程的 ID • get_current_user() - 获取当前 PHP 脚本所有者名称 • getmyinode() - 获取当前脚本的索引节点（inode） • getlastmod() - 获取页面最后修改的时间
