The direction of rotation is defined such that positive angles rotate in the direction from the positive X axis toward the positive Y axis. With the default axis orientation of cairo, positive angles rotate in a clockwise direction. 返回值 Returns a new CairoMatrix object that can be used with surfaces, contexts, and patterns. 范例 Example #1 面向对象风格 Example #2 过程化风格 参见 • CairoMatrix::__construct() - Creates a new CairoMatrix object • CairoMatrix::initIdentity() - Creates a new identity matrix • CairoMatrix::initScale() - Creates a new scaling matrix • CairoMatrix::initTranslate() - Creates a new translation matrix
