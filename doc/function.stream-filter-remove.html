stream_filter_remove (PHP 5 >= 5.1.0, PHP 7) stream_filter_remove — 从资源流里移除某个过滤器 说明 bool stream_filter_remove ( resource $stream_filter ) 移除之前通过 stream_filter_prepend() 或者 stream_filter_append() 添加到资源流里 面的过滤器。在移除之前，残留在过滤器内部缓冲区里的所有数据刷新到下一个过滤器。 参数 stream_filter 需要被移除的资源流过滤器。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 动态地重新过滤一个资源流 以上例程会输出： Guvf vf a test 参见 • stream_filter_register() - Register a user defined stream filter • stream_filter_append() - Attach a filter to a stream • stream_filter_prepend() - Attach a filter to a stream
