EvLoop::nowUpdate (PECL ev >= 0.2.0) EvLoop::nowUpdate — Establishes the current time by querying the kernel, updating the time returned by EvLoop::now in the progress. 说明 public void EvLoop::nowUpdate ( void ) Establishes the current time by querying the kernel, updating the time returned by EvLoop::now() in the progress. This is a costly operation and is usually done automatically within EvLoop::run() . 参数 此函数没有参数。 返回值 没有返回值。 参见 • EvLoop::now() - Returns the current "event loop time" • Ev::nowUpdate() - Establishes the current time by querying the kernel, updating the time returned by Ev::now in the progress.
