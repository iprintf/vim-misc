cubrid_error_code_facility (PECL CUBRID >= 8.3.0) cubrid_error_code_facility — Get the facility code of error 说明 int cubrid_error_code_facility ( void ) The cubrid_error_code_facility() function is used to get the facility code (level in which the error occurred) from the error code of the error that occurred during the API execution. Usually, you can get the error code when API returns false as its return value. 参数 此函数没有参数。 返回值 Facility code of the error code that occurred: CUBRID_FACILITY_DBMS, CUBRID_FACILITY_CAS, CUBRID_FACILITY_CCI, CUBRID_FACILITY_CLIENT 范例 Example #1 cubrid_error_code_facility() example 以上例程会输出： Error facility: 1 Error code: -493 Error msg: Syntax: In line 1, column 15 before END OF STATEMENT Syntax error: unexpected 'unknown' 参见 • cubrid_error_code() - Get error code for the most recent function call • cubrid_error_msg() - Get last error message for the most recent function call
