EvTimer::set (PECL ev >= 0.2.0) EvTimer::set — Configures the watcher 说明 public void EvTimer::set ( double $after , double $repeat ) Configures the watcher 参数 after Configures the timer to trigger after after seconds. repeat If repeat is 0.0 , then it will automatically be stopped once the timeout is reached. If it is positive, then the timer will automatically be configured to trigger again every repeat seconds later, until stopped manually. 返回值 没有返回值。
