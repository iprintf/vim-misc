Gender\Gender::country (PECL gender >= 0.8.0) Gender\Gender::country — Get textual country representation 说明 public array Gender\Gender::country ( integer $country ) Returns the textual representation of a country from a Gender class constant. 参数 country A country ID specified by a Gender\Gender class constant. 返回值 Returns an array with the short and full names of the country on success 或者在 失败时返回 FALSE. 范例 Example #1 Using Gender\Gender::country() $gender = new Gender\Gender; var_dump($gender->country(Gender\Gender::BRITAIN)); 以上例程会输出： array(2) { 'country_short' => string(2) "UK" 'country' => string(13) "Great Britain" }
