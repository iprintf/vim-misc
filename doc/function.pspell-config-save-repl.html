pspell_config_save_repl (PHP 4 >= 4.0.2, PHP 5, PHP 7) pspell_config_save_repl — Determine whether to save a replacement pairs list along with the wordlist 说明 bool pspell_config_save_repl ( int $dictionary_link , bool $flag ) pspell_config_save_repl() determines whether pspell_save_wordlist() will save the replacement pairs along with the wordlist. Usually there is no need to use this function because if pspell_config_repl() is used, the replacement pairs will be saved by pspell_save_wordlist() anyway, and if it is not, the replacement pairs will not be saved. pspell_config_save_repl() should be used on a config before calling pspell_new_config(). 参数 dictionary_link flag TRUE if replacement pairs should be saved, FALSE otherwise. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 注释 Note: This function will not work unless you have pspell .11.2 and aspell .32.5 or later.
