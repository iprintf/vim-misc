HaruPage::eoFillStroke (PECL haru >= 0.0.1) HaruPage::eoFillStroke — Fill current path using even-odd rule, then paint the path 说明 bool HaruPage::eoFillStroke ([ bool $close_path = false ] ) Fills current path using even-odd rule, then paints the path. 参数 close_path Optional parameter. When set to TRUE, the function closes the current path. Default to FALSE. 返回值 Returns TRUE on success. 错误／异常 发生错误时抛出 HaruException 异常。
