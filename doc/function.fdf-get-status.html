fdf_get_status (PHP 4, PHP 5, PHP 7) fdf_get_status — Get the value of the /STATUS key 说明 string fdf_get_status ( resource $fdf_document ) Gets the value of the /STATUS key. 参数 fdf_document The FDF document handle, returned by fdf_create(), fdf_open() or fdf_open_string(). 返回值 Returns the key value, as a string. 参见 • fdf_set_status() - Set the value of the /STATUS key
