fdf_set_version (PHP 4 >= 4.3.0, PHP 5, PHP 7) fdf_set_version — Sets version number for a FDF file 说明 bool fdf_set_version ( resource $fdf_document , string $version ) Sets the FDF version for the given document. Some features supported by this extension are only available in newer FDF versions. 参数 fdf_document The FDF document handle, returned by fdf_create(), fdf_open() or fdf_open_string(). version The version number. For the current FDF toolkit 5.0, this may be either 1.2 , 1.3 or 1.4. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • fdf_get_version() - Gets version number for FDF API or file
