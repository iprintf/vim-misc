geoip_id_by_name (PECL geoip >= 0.2.0) geoip_id_by_name — Get the Internet connection type 说明 int geoip_id_by_name ( string $hostname ) The geoip_id_by_name() function will return the Internet connection type corresponding to a hostname or an IP address. The return value is numeric and can be compared to the following constants: • GEOIP_UNKNOWN_SPEED • GEOIP_DIALUP_SPEED • GEOIP_CABLEDSL_SPEED • GEOIP_CORPORATE_SPEED 参数 hostname The hostname or IP address whose connection type is to be looked-up. 返回值 Returns the connection type. 范例 Example #1 A geoip_id_by_name() example This will output the connection type of the host example.com. 以上例程会输出： The connection type is corporate
