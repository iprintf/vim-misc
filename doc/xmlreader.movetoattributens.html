XMLReader::moveToAttributeNs (PHP 5 >= 5.1.0, PHP 7) XMLReader::moveToAttributeNs — Move cursor to a named attribute 说明 public bool XMLReader::moveToAttributeNs ( string $localName , string $namespaceURI ) Positions cursor on the named attribute in specified namespace. 参数 localName The local name. namespaceURI The namespace URI. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • XMLReader::moveToElement() - Position cursor on the parent Element of current Attribute • XMLReader::moveToAttribute() - Move cursor to a named attribute • XMLReader::moveToAttributeNo() - Move cursor to an attribute by index • XMLReader::moveToFirstAttribute() - Position cursor on the first Attribute
