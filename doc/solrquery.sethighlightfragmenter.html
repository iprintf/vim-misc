SolrQuery::setHighlightFragmenter (PECL solr >= 0.9.2) SolrQuery::setHighlightFragmenter — Sets a text snippet generator for highlighted text 说明 public SolrQuery SolrQuery::setHighlightFragmenter ( string $fragmenter [, string $field_override ] ) Specify a text snippet generator for highlighted text. 参数 fragmenter The standard fragmenter is gap.
