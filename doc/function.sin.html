sin (PHP 4, PHP 5, PHP 7) sin — 正弦 说明 float sin ( float $arg ) sin() 返回参数 arg 的正弦值。参数 arg 的单位为弧度。 参数 arg 单位是弧度的值。 返回值 arg 的正弦值 范例 Example #1 sin() 例子 参见 • asin() - 反正弦 • sinh() - 双曲正弦 • cos() - 余弦 • tan() - 正切 • deg2rad() - 将角度转换为弧度 User Contributed Notes 1 note =============================================================================== You can use SIN & COS to create/draw circles. Please beware that the circle's center is also your base point (0,0). So without specifying an offset, some coordinates will be negative.
