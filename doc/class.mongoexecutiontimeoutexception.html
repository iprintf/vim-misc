The MongoExecutionTimeoutException class (PECL mongo >= 1.5.0) 简介 Thrown when a operation times out server side (i.e. in MongoDB). To configure the operation timeout threshold, use MongoCursor::maxTimeMS() or the "maxTimeMS" command option. 类摘要 MongoExecutionTimeoutException extends MongoException { /* 继承的属性 */ protected string $message ; protected int $code ; protected string $file ; protected int $line ; }
