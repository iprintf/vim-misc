pg_connection_reset (PHP 4 >= 4.2.0, PHP 5, PHP 7) pg_connection_reset — 重置连接（再次连接） 说明 bool pg_connection_reset ( resource $connection ) pg_connection_reset() 重置一个连接，用于错误恢复。 参数 connection PostgreSQL database connection resource. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 pg_connection_reset() 例子 参见 • pg_connect() - 打开一个 PostgreSQL 连接 • pg_pconnect() - 打开一个持久的 PostgreSQL 连接 • pg_connection_status() - 获得连接状态
