zip_entry_read (PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PECL zip >= 1.0.0) zip_entry_read — 读取一个打开了的压缩目录实体 说明 string zip_entry_read ( resource $zip_entry [, int $length = 1024 ] ) 读取一个打开了的压缩目录实体。 参数 zip_entry 由函数zip_read() 返回的目录实体。 length 需要返回的字节数。 Note: 这字节数应该是你所要读取的未压缩的字节数。 返回值 成功的时候返回读取到的数据；到达文件末尾的时候返回一个空的字符串；读取出错的时 候则会返回FALSE 参见 • zip_entry_open() - 打开用于读取的目录实体 • zip_entry_close() - 关闭目录项 • zip_entry_filesize() - 检索目录实体的实际大小
