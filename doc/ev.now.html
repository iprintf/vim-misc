Ev::now (PECL ev >= 0.2.0) Ev::now — Returns the time when the last iteration of the default event loop has started. 说明 final public static double Ev::now ( void ) Returns the time when the last iteration of the default event loop has started. This is the time that timers( EvTimer and EvPeriodic ) are based on, and referring to it is usually faster then calling Ev::time() . 参数 此函数没有参数。 返回值 Returns number of seconds(fractional) representing the time when the last iteration of the default event loop has started. 参见 • Ev::nowUpdate() - Establishes the current time by querying the kernel, updating the time returned by Ev::now in the progress.
