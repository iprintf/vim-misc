RecursiveArrayIterator::hasChildren (PHP 5 >= 5.1.0, PHP 7) RecursiveArrayIterator::hasChildren — Returns whether current entry is an array or an object. 说明 public bool RecursiveArrayIterator::hasChildren ( void ) Returns whether current entry is an array or an object for which an iterator can be obtained via RecursiveArrayIterator::getChildren(). 参数 此函数没有参数。 返回值 Returns TRUE if the current entry is an array or an object, otherwise FALSE is returned. 范例 Example #1 RecursiveArrayIterator::hasChildren() example "lemon", "b" => "orange", array("a" => "apple", "p" => "pear")); $iterator = new RecursiveArrayIterator($fruits); while ($iterator->valid()) { // Check if there are children if ($iterator->hasChildren()) { // print all children foreach ($iterator->getChildren() as $key => $value) { echo $key . ' : ' . $value . "\n"; } } else { echo "No children.\n"; } $iterator->next(); } ?> 以上例程会输出： No children. No children. a : apple p : pear 参见 • RecursiveArrayIterator::getChildren() - Returns an iterator for the current entry if it is an array or an object.
