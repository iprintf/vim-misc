LimitIterator::valid (PHP 5 >= 5.1.0, PHP 7) LimitIterator::valid — Check whether the current element is valid 说明 public bool LimitIterator::valid ( void ) Checks whether the current element is valid. 参数 此函数没有参数。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • LimitIterator::current() - Get current element • LimitIterator::key() - Get current key • LimitIterator::rewind() - Rewind the iterator to the specified starting offset • LimitIterator::next() - Move the iterator forward • LimitIterator::seek() - Seek to the given position
