HaruPage::endPath (PECL haru >= 0.0.1) HaruPage::endPath — End current path object without filling and painting operations 说明 bool HaruPage::endPath ( void ) Ends current path object without performing filling and painting operations. 参数 此函数没有参数。 返回值 Returns TRUE on success. 错误／异常 发生错误时抛出 HaruException 异常。
