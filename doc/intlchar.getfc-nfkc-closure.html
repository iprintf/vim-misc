IntlChar::getFC_NFKC_Closure (PHP 7) IntlChar::getFC_NFKC_Closure — Get the FC_NFKC_Closure property for a code point 说明 public static string IntlChar::getFC_NFKC_Closure ( mixed $codepoint ) Gets the FC_NFKC_Closure property string for a character. 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 Returns the FC_NFKC_Closure property string for the codepoint, or an empty string if there is none. 范例 Example #1 Testing different code points 以上例程会输出： string(3) "tel" string(2) "æ"
