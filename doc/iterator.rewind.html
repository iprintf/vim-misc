Iterator::rewind (PHP 5 >= 5.0.0, PHP 7) Iterator::rewind — 返回到迭代器的第一个元素 说明 abstract public void Iterator::rewind ( void ) 返回到迭代器的第一个元素。 Note: 当开始一个 foreach 循环时，这是第一个被调用的方法。它将不会在 foreach 循环 之后被调用。 参数 此函数没有参数。 返回值 任何返回都将被忽略。 User Contributed Notes 1 note =============================================================================== Builtin PHP function is »reset«. Don't mix this with builtin function »rewind« which operates on file handle.
