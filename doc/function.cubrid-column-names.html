cubrid_column_names (PECL CUBRID >= 8.3.0) cubrid_column_names — Get the column names in result 说明 array cubrid_column_names ( resource $req_identifier ) The cubrid_column_names() function is used to get the column names of the query result by using req_identifier. 参数 req_identifier Request identifier. 返回值 Array of string values containing the column names, when process is successful. FALSE, when process is unsuccessful. 范例 Example #1 cubrid_column_names() example 以上例程会输出： Column Names Column Types Column Maxlen host_year integer 11 event_code integer 11 athlete_code integer 11 stadium_code integer 11 nation_code char 3 medal char 1 game_date date 10 参见 • cubrid_prepare() - Prepare a SQL statement for execution • cubrid_execute() - Execute a prepared SQL statement • cubrid_column_types() - Get column types in result
