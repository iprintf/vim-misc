event_buffer_enable (PECL libevent >= 0.0.1) event_buffer_enable — Enable a buffered event 说明 bool event_buffer_enable ( resource $bevent , int $events ) Enables the specified buffered event. 参数 bevent Valid buffered event resource. events Any combination of EV_READ and EV_WRITE. 返回值 event_buffer_enable() returns TRUE on success or FALSE on error.
