ps_continue_text (PECL ps >= 1.1.0) ps_continue_text — Continue text in next line 说明 bool ps_continue_text ( resource $psdoc , string $text ) Output a text one line below the last line. The line spacing is taken from the value "leading" which must be set with ps_set_value(). The actual position of the text is determined by the values "textx" and "texty" which can be requested with ps_get_value() 参数 psdoc Resource identifier of the postscript file as returned by ps_new(). text The text to output. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ps_show() - Output text • ps_show_xy() - Output text at given position • ps_show_boxed() - Output text in a box
