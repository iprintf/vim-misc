fileowner (PHP 4, PHP 5, PHP 7) fileowner — 取得文件的所有者 说明 int fileowner ( string $filename ) 取得文件的所有者。 参数 filename 文件的路径。 返回值 返回文件所有的用户 ID，如果出错则返回 FALSE。用户 ID 以数字格式返回，用 posix_getpwuid() 来将其解析为用户名。 范例 Example #1 找到文件的所有者 错误／异常 失败时抛出E_WARNING警告。 注释 Note: 此函数的结果会被缓存。参见 clearstatcache() 以获得更多细节。 Tip 自 PHP 5.0.0 起, 此函数也用于某些 URL 包装器。请参见支持的协议和封装协议以获得 支持 stat() 系列函数功能的包装器列表。 参见 • filegroup() - 取得文件的组 • stat() - 给出文件的信息 • posix_getpwuid() - Return info about a user by user id User Contributed Notes 3 notes =============================================================================== Small note: the function resolves symbolic links.
