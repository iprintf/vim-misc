DOMElement::hasAttribute (PHP 5, PHP 7) DOMElement::hasAttribute — Checks to see if attribute exists 说明 public bool DOMElement::hasAttribute ( string $name ) Indicates whether attribute named name exists as a member of the element. 参数 name The attribute name. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • DOMElement::hasAttributeNS() - Checks to see if attribute exists • DOMElement::getAttribute() - Returns value of attribute • DOMElement::setAttribute() - Adds new attribute • DOMElement::removeAttribute() - Removes attribute
