fdf_get_ap (PHP 4 >= 4.3.0, PHP 5, PHP 7) fdf_get_ap — Get the appearance of a field 说明 bool fdf_get_ap ( resource $fdf_document , string $field , int $face , string $filename ) Gets the appearance of a field (i.e. the value of the /AP key) and stores it in a file. 参数 fdf_document The FDF document handle, returned by fdf_create(), fdf_open() or fdf_open_string(). field face The possible values are FDFNormalAP, FDFRolloverAP and FDFDownAP. filename The appearance will be stored in this parameter. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
