fann_get_cascade_max_cand_epochs (PECL fann >= 1.0.0) fann_get_cascade_max_cand_epochs — Returns the maximum candidate epochs 说明 int fann_get_cascade_max_cand_epochs ( resource $ann ) The maximum candidate epochs determines the maximum number of epochs the input connections to the candidates may be trained before adding a new candidate neuron. The default max candidate epochs is 150. 参数 ann Neural network resource. 返回值 The maximum candidate epochs, or FALSE on error. 参见 • fann_set_cascade_max_cand_epochs() - Sets the max candidate epochs
