mysqli_stmt::attr_get mysqli_stmt_attr_get (PHP 5, PHP 7) mysqli_stmt::attr_get -- mysqli_stmt_attr_get — Used to get the current value of a statement attribute 说明 面向对象风格 int mysqli_stmt::attr_get ( int $attr ) 过程化风格 int mysqli_stmt_attr_get ( mysqli_stmt $stmt , int $attr ) Gets the current value of a statement attribute. 参数 stmt 仅以过程化样式：由 mysqli_stmt_init() 返回的 statement 标识。 attr The attribute that you want to get. 返回值 Returns FALSE if the attribute is not found, otherwise returns the value of the attribute.
