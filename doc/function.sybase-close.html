sybase_close (PHP 4, PHP 5, PHP 7) sybase_close — Closes a Sybase connection 说明 bool sybase_close ([ resource $link_identifier ] ) sybase_close() closes the link to a Sybase database that's associated with the specified link link_identifier. Note that this isn't usually necessary, as non-persistent open links are automatically closed at the end of the script's execution. sybase_close() will not close persistent links generated by sybase_pconnect(). 参数 link_identifier If the link identifier isn't specified, the last opened link is assumed. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • sybase_connect() - Opens a Sybase server connection • sybase_pconnect() - Open persistent Sybase connection
