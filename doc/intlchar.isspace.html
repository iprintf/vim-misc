IntlChar::isspace (PHP 7) IntlChar::isspace — Check if code point is a space character 说明 public static bool IntlChar::isspace ( mixed $codepoint ) Determines if the specified character is a space character or not. 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 Returns TRUE if codepoint is a space character, FALSE if not. 范例 Example #1 Testing different code points 以上例程会输出： bool(false) bool(true) bool(true) bool(true) bool(true) 参见 • IntlChar::isJavaSpaceChar() - Check if code point is a space character according to Java • IntlChar::isWhitespace() - Check if code point is a whitespace character according to ICU • IntlChar::isUWhiteSpace() - Check if code point has the White_Space Unicode property
