deg2rad (PHP 4, PHP 5, PHP 7) deg2rad — 将角度转换为弧度 说明 float deg2rad ( float $number ) 本函数把 number 从角度转换成弧度。 参数 number 以角度为单位的值 返回值 number 等量的弧度值 范例 Example #1 deg2rad() 例子 参见 • rad2deg() - 将弧度数转换为相应的角度数 User Contributed Notes 1 note =============================================================================== Never use deg2rad() in loops, use $rad = ($deg * M_PI / 180) instead which is faster!
