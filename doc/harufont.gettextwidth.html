HaruFont::getTextWidth (PECL haru >= 0.0.1) HaruFont::getTextWidth — Get the total width of the text, number of characters, number of words and number of spaces 说明 array HaruFont::getTextWidth ( string $text ) Get the total width of the text, number of characters, number of words and number of spaces. 参数 text The text to measure. 返回值 Returns the total width of the text, number of characters, number of words and number of spaces in the given text. 错误／异常 发生错误时抛出 HaruException 异常。
