MongoResultException 类 (PECL mongo >=1.3.0) 简介 MongoResultException 由几个助手命令（例如 MongoCollection::findAndModify()）在 发生失败事件时抛出。原始的结果文档可以通过 MongoResultException::getDocument() 获取。 类摘要 MongoResultException extends MongoException { /* 属性 */ public $document ; /* 继承的属性 */ protected string $message ; protected int $code ; protected string $file ; protected int $line ; /* 方法 */ public array getDocument ( void ) } 属性 document array 的原始结果文档。 Table of Contents • MongoResultException::getDocument — 获取完整的结果文档
