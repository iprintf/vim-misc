SphinxClient::getLastWarning (PECL sphinx >= 0.1.0) SphinxClient::getLastWarning — Get the last warning 说明 public string SphinxClient::getLastWarning ( void ) Returns last warning message. If there were no warnings during the previous API call, empty string is returned. This method doesn't reset the warning, so you can safely call it several times. 参数 此函数没有参数。 返回值 Returns the last warning message or an empty string if there were no warnings. 参见 • SphinxClient::getLastError
