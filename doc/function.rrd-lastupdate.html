rrd_lastupdate (PECL rrd >= 0.9.0) rrd_lastupdate — Gets information about last updated data. 说明 array rrd_lastupdate ( string $filename ) Gets array of the UNIX timestamp and the values stored for each date in the most recent update of the RRD database file. 参数 file RRD database file name. 返回值 Array of information about last update, FALSE when error occurs.
