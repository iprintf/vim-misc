ReflectionClass::getMethod (PHP 5, PHP 7) ReflectionClass::getMethod — 获取一个类方法的 ReflectionMethod。 说明 public ReflectionMethod ReflectionClass::getMethod ( string $name ) 获取一个类方法的 ReflectionMethod。 参数 name 要反射的方法名称。 返回值 一个 ReflectionMethod。 错误／异常 如果方法不存在则会抛出 ReflectionException 异常。 范例 Example #1 ReflectionClass::getMethod() 的基本用法 getMethod('getMethod'); var_dump($method); ?> 以上例程会输出： object(ReflectionMethod)#2 (2) { ["name"]=> string(9) "getMethod" ["class"]=> string(15) "ReflectionClass" } 参见 • ReflectionClass::getMethods() - 获取方法的数组 User Contributed Notes 2 notes =============================================================================== If you ever need to get the type hint of a parameter in a method use this. getMethod('FireCannon')->getParameters(); //Loop through each parameter and get the type foreach($parameters as $param) { //Before you call getClass() that class must be defined! echo $param->getClass()->name; } ?> =============================================================================== If the method doesn't exist within the class that the ReflectionClass is reflecting an Exception is thrown.
