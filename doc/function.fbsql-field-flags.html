fbsql_field_flags (PHP 4 >= 4.0.6, PHP 5, PHP 7) fbsql_field_flags — Get the flags associated with the specified field in a result 说明 string fbsql_field_flags ( resource $result [, int $field_offset ] ) Gets the flags associated with the specified field in a result. 参数 result A result pointer returned by fbsql_list_fields(). field_offset The numerical offset of the field. The field index starts at 0. 返回值 Returns the field flags of the specified field as a single word per flag separated by a single space, so that you can split the returned value using explode().
