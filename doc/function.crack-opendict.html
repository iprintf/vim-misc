crack_opendict (PECL crack >= 0.1) crack_opendict — Opens a new CrackLib dictionary 说明 resource crack_opendict ( string $dictionary ) crack_opendict() opens the specified CrackLib dictionary for use with crack_check(). Warning 此函数是实验性的。此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本 中未通知就被修改。使用本函数风险自担。 Note: Only one dictionary may be open at a time. 参数 dictionary The path to the Cracklib dictionary. 返回值 Returns a dictionary resource identifier on success 或者在失败时返回 FALSE. 参见 • crack_check() - Performs an obscure check with the given password • crack_closedict() - Closes an open CrackLib dictionary
