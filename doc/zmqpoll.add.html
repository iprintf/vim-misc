ZMQPoll::add (PECL zmq >= 0.5.0) ZMQPoll::add — Add item to the poll set 说明 public string ZMQPoll::add ( mixed $entry , integer $type ) Adds a new item to the poll set and returns the internal id of the added item. The item can be removed from the poll set using the returned string id. 参数 entry ZMQSocket object or a PHP stream resource type Defines what activity the socket is polled for. See ZMQ::POLL_IN and ZMQ::POLL_OUT constants. 返回值 Returns a string id of the added item which can be later used to remove the item. Throws ZMQPollException on error.
