svn_revert (PECL svn >= 0.3.0) svn_revert — Revert changes to the working copy 说明 bool svn_revert ( string $path [, bool $recursive = false ] ) Revert any local changes to the path in a working copy. 参数 path The path to the working repository. recursive Optionally make recursive changes. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • svn_delete() - Delete items from a working copy or repository. • svn_export() - Export the contents of a SVN directory
