Weakref::valid (PECL weakref >= 0.1.0) Weakref::valid — Checks whether the object referenced still exists 说明 public bool Weakref::valid ( void ) Checks whether the object referenced still exists. 参数 此函数没有参数。 返回值 Returns TRUE if the object still exists and is thus still accessible via Weakref::get(), FALSE otherwise. 参见 • Weakref::get() - Returns the object pointed to by the weak reference
