posix_setpgid (PHP 4, PHP 5, PHP 7) posix_setpgid — Set process group id for job control 说明 bool posix_setpgid ( int $pid , int $pgid ) Let the process pid join the process group pgid. 参数 pid The process id. pgid The process group id. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • See POSIX.1 and the setsid(2) manual page on the POSIX system for more informations on process groups and job control.
