XMLReader::moveToAttributeNo (PHP 5 >= 5.1.0, PHP 7) XMLReader::moveToAttributeNo — Move cursor to an attribute by index 说明 public bool XMLReader::moveToAttributeNo ( int $index ) Positions cursor on attribute based on its position. 参数 index The position of the attribute. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • XMLReader::moveToElement() - Position cursor on the parent Element of current Attribute • XMLReader::moveToAttribute() - Move cursor to a named attribute • XMLReader::moveToAttributeNs() - Move cursor to a named attribute • XMLReader::moveToFirstAttribute() - Position cursor on the first Attribute
