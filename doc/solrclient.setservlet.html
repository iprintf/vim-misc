SolrClient::setServlet (PECL solr >= 0.9.2) SolrClient::setServlet — Changes the specified servlet type to a new value 说明 public bool SolrClient::setServlet ( int $type , string $value ) Changes the specified servlet type to a new value 参数 type One of the following : - SolrClient::SEARCH_SERVLET_TYPE - SolrClient::UPDATE_SERVLET_TYPE - SolrClient::THREADS_SERVLET_TYPE - SolrClient::PING_SERVLET_TYPE - SolrClient::TERMS_SERVLET_TYPE value The new value for the servlet 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
