sqlite_close (PHP 5 < 5.4.0, PECL sqlite >= 1.0.0) sqlite_close — Closes an open SQLite database 说明 void sqlite_close ( resource $dbhandle ) Closes the given db_handle database handle. If the database was persistent, it will be closed and removed from the persistent list. 参数 dbhandle The SQLite Database resource; returned from sqlite_open() when used procedurally. 返回值 没有返回值。 范例 Example #1 sqlite_close() example 参见 • sqlite_open() - Opens an SQLite database and create the database if it does not exist • sqlite_popen() - Opens a persistent handle to an SQLite database and create the database if it does not exist
