gupnp_root_device_start (PECL gupnp >= 0.1.0) gupnp_root_device_start — Start main loop 说明 bool gupnp_root_device_start ( resource $root_device ) Start root server's main loop. 参数 root_device A root device identifier, returned by gupnp_root_device_new(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • gupnp_root_device_stop() - Stop main loop
