ftp_close (PHP 4 >= 4.2.0, PHP 5, PHP 7) ftp_close — 关闭一个 FTP 连接 说明 bool ftp_close ( resource $ftp_stream ) ftp_close() 关闭给出的连接标识符并释放资源。 Note: 调用本函数后，将不能再使用 FTP 连接，必须用 ftp_connect() 建立一个新连接。 参数 ftp_stream FTP 连接的标识符。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 ftp_close() 例子 参见 • ftp_connect() - 建立一个新的 FTP 连接
