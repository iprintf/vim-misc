ifx_do (PHP 4, PHP <=5.2.0) ifx_do — Execute a previously prepared SQL-statement 说明 bool ifx_do ( resource $result_id ) Executes a previously prepared query or opens a cursor for it. Does NOT free result_id on error. Also sets the real number of ifx_affected_rows() for non-select statements for retrieval by ifx_affected_rows(). 参数 result_id result_id is a valid resultid returned by ifx_query() or ifx_prepare() (select type queries only!). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 Example #1 ifx_do() Example 参见 • ifx_prepare() - Prepare an SQL-statement for execution
