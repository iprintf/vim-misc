PDF_set_border_dash (PHP 4 >= 4.0.1, PECL pdflib >= 1.0.0) PDF_set_border_dash — Set border dash style of annotations [deprecated] 说明 bool PDF_set_border_dash ( resource $pdfdoc , float $black , float $white ) Sets the border dash style for all kinds of annotations. 成功时返回 TRUE，或者 在失败时返回 FALSE。 This function is deprecated since PDFlib version 6, use the option dasharray in PDF_create_annotation() instead.
