gupnp_context_get_subscription_timeout (PECL gupnp >= 0.1.0) gupnp_context_get_subscription_timeout — Get the event subscription timeout 说明 int gupnp_context_get_subscription_timeout ( resource $context ) Get the event subscription timeout (in seconds), or 0 meaning there is no timeout. 参数 context A context identifier, returned by gupnp_context_new(). 返回值 The event subscription timeout in seconds. 参见 • gupnp_context_new() - Create a new context • gupnp_context_set_subscription_timeout() - Sets the event subscription timeout
