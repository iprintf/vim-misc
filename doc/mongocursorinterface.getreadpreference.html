MongoCursorInterface::getReadPreference (PECL mongo >=1.6.0) MongoCursorInterface::getReadPreference — Get the read preference for this query Description abstract public array MongoCursorInterface::getReadPreference ( void ) Parameters This function has no parameters. Return Values This function returns an array describing the read preference. The array contains the values type for the string read preference mode (corresponding to the MongoClient constants), and tagsets containing a list of all tag set criteria. If no tag sets were specified, tagsets will not be present in the array.
