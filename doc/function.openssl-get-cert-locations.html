openssl_get_cert_locations (PHP 5 >= 5.6.0, PHP 7) openssl_get_cert_locations — Retrieve the available certificate locations 说明 array openssl_get_cert_locations ( void ) openssl_get_cert_locations() returns an array with information about the available certificate locations that will be searched for SSL certificates. 参数 此函数没有参数。 返回值 Returns an array with the available certificate locations. 范例 Example #1 openssl_get_cert_locations() example 以上例程会输出： array(8) { ["default_cert_file"]=> string(21) "/usr/lib/ssl/cert.pem" ["default_cert_file_env"]=> string(13) "SSL_CERT_FILE" ["default_cert_dir"]=> string(18) "/usr/lib/ssl/certs" ["default_cert_dir_env"]=> string(12) "SSL_CERT_DIR" ["default_private_dir"]=> string(20) "/usr/lib/ssl/private" ["default_default_cert_area"]=> string(12) "/usr/lib/ssl" ["ini_cafile"]=> string(0) "" ["ini_capath"]=> string(0) "" }
