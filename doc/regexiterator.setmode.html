RegexIterator::setMode (PHP 5 >= 5.2.0, PHP 7) RegexIterator::setMode — Sets the operation mode. 说明 public void RegexIterator::setMode ( int $mode ) Sets the operation mode. 参数 mode The operation mode. The available modes are listed below. The actual meanings of these modes are described in the predefined constants. ' => ' . $value[1] . PHP_EOL; } ?> 以上例程的输出类似于： str1 => 1 str3 => 123 参见 • RegexIterator::getMode() - Returns operation mode.
