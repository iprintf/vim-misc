SplFileObject::fgetss (PHP 5 >= 5.1.0, PHP 7) SplFileObject::fgetss — Gets line from file and strip HTML tags 说明 public string SplFileObject::fgetss ([ string $allowable_tags ] ) Identical to SplFileObject::fgets(), except that SplFileObject::fgetss() attempts to strip any HTML and PHP tags from the text it reads. 参数 allowable_tags Optional parameter to specify tags which should not be stripped. 返回值 Returns a string containing the next line of the file with HTML and PHP code stripped, or FALSE on error. 范例 Example #1 SplFileObject::fgetss() example
