HaruDoc::loadRaw (PECL haru >= 0.0.1) HaruDoc::loadRaw — Load a RAW image 说明 object HaruDoc::loadRaw ( string $filename , int $width , int $height , int $color_space ) Loads a RAW image. 参数 filename The name of a RAW image file. width The width of the image. height The height of the image. color_space The color space of the image. Can be one of the following values: □ HaruDoc::CS_DEVICE_GRAY □ HaruDoc::CS_DEVICE_RGB □ HaruDoc::CS_DEVICE_CMYK 返回值 Returns a HaruImage instance. 错误／异常 发生错误时抛出 HaruException 异常。 参见 • HaruDoc::loadJPEG() - Load a JPEG image • HaruDoc::loadPNG() - Load PNG image and return HaruImage instance
