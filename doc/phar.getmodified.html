Phar::getModified (PHP >= 5.3.0, PECL phar >= 1.0.0) Phar::getModified — Return whether phar was modified 说明 public bool Phar::getModified ( void ) This method can be used to determine whether a phar has either had an internal file deleted, or contents of a file changed in some way. 参数 No parameters. 返回值 TRUE if the phar has been modified since opened, FALSE if not.
