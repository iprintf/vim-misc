fbsql_drop_db (PHP 4 >= 4.0.6, PHP 5, PHP 7) fbsql_drop_db — Drop (delete) a FrontBase database 说明 bool fbsql_drop_db ( string $database_name [, resource $link_identifier ] ) fbsql_drop_db() attempts to drop (remove) an entire database from the server associated with the specified link identifier. 参数 database_name The database name, as a string. link_identifier 由 fbsql_connect() 或 fbsql_pconnect() 返回的 FrontBase 链接标识符. 如可选并未指定，函数会尝试查找一个到 FrontBase 服务器的开放链接，且如没有找 到这样的链接，会尝试通过不带参数调用 fbsql_connect() 来创建一个。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • fbsql_create_db() - Create a FrontBase database
