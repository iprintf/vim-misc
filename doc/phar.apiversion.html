Phar::apiVersion (PHP >= 5.3.0, PECL phar >= 1.0.0) Phar::apiVersion — Returns the api version 说明 final public static string Phar::apiVersion ( void ) Return the API version of the phar file format that will be used when creating phars. The Phar extension supports reading API version 1.0.0 or newer. API version 1.1.0 is required for SHA-256 and SHA-512 hash, and API version 1.1.1 is required to store empty directories. 参数 返回值 The API version string as in "1.0.0". 范例 Example #1 A Phar::apiVersion() example 以上例程会输出： 1.1.1
