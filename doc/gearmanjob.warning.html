GearmanJob::warning (PECL gearman <= 0.5.0) GearmanJob::warning — Send a warning (deprecated) 说明 public bool GearmanJob::warning ( string $warning ) Sends a warning for this job while it is running. Note: This method has been replaced by GearmanJob::sendWarning() in the 0.6.0 release of the Gearman extension. 参数 warning A warning messages. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • GearmanJob::sendComplete() - Send the result and complete status • GearmanJob::sendException() - Send exception for running job (exception) • GearmanJob::sendFail() - Send fail status
