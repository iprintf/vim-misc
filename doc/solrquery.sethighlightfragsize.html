SolrQuery::setHighlightFragsize (PECL solr >= 0.9.2) SolrQuery::setHighlightFragsize — The size of fragments to consider for highlighting 说明 public SolrQuery SolrQuery::setHighlightFragsize ( int $size [, string $field_override ] ) Sets the size, in characters, of fragments to consider for highlighting. "0" indicates that the whole field value should be used (no fragmenting). 参数 size The size, in characters, of fragments to consider for highlighting field_override The name of the field. 返回值 Returns the current SolrQuery object, if the return value is used.
