IntlChar::toupper (PHP 7) IntlChar::toupper — Make Unicode character uppercase 说明 public static mixed IntlChar::toupper ( mixed $codepoint ) The given character is mapped to its uppercase equivalent. If the character has no uppercase equivalent, the character itself is returned. 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 Returns the Simple_Uppercase_Mapping of the code point, if any; otherwise the code point itself. The return type will be integer unless the code point was passed as a UTF-8 string, in which case a string will be returned. 范例 Example #1 Testing different code points 以上例程会输出： string(1) "A" string(1) "A" string(2) "Φ" string(2) "Φ" string(1) "1" int(65) int(65) 参见 • IntlChar::tolower() - Make Unicode character lowercase • IntlChar::totitle() - Make Unicode character titlecase • mb_strtoupper() - 使字符串大写
