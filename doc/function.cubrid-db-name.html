cubrid_db_name (PECL CUBRID >= 8.3.1) cubrid_db_name — Get db name from results of cubrid_list_dbs 说明 string cubrid_db_name ( array $result , int $index ) Retrieve the database name from a call to cubrid_list_dbs(). 参数 result The result pointer from a call to cubrid_list_dbs(). index The index into the result set. 返回值 Returns the database name on success, and FALSE on failure. If FALSE is returned, use cubrid_error() to determine the nature of the error. 范例 Example #1 cubrid_db_name() example 以上例程会输出： demodb 参见 • cubrid_list_dbs() - Return an array with the list of all existing CUBRID databases
