runkit_function_remove (PECL runkit >= 0.7.0) runkit_function_remove — Remove a function definition 说明 bool runkit_function_remove ( string $funcname ) Note: 默认情况下，仅在用户空间可删除，重命名，或者修改函数。为了覆盖内部函 数，必须启用 php.ini 中的 runkit.internal_override 设置。 参数 funcname Name of function to be deleted 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • runkit_function_add() - Add a new function, similar to create_function • runkit_function_copy() - Copy a function to a new function name • runkit_function_redefine() - Replace a function definition with a new implementation • runkit_function_rename() - Change a function's name
