IntlChar::istitle (PHP 7) IntlChar::istitle — Check if code point is a titlecase letter 说明 public static bool IntlChar::istitle ( mixed $codepoint ) Determines whether the specified code point is a titlecase letter. TRUE for general category "Lt" (titlecase letter). 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 Returns TRUE if codepoint is a titlecase letter, FALSE if not. 范例 Example #1 Testing different code points 以上例程会输出： bool(false) bool(true) bool(false) bool(true) bool(false) 参见 • IntlChar::isupper() - Check if code point has the general category "Lu" (uppercase letter) • IntlChar::islower() - Check if code point is a lowercase letter • IntlChar::totitle() - Make Unicode character titlecase
