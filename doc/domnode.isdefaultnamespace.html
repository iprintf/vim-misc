DOMNode::isDefaultNamespace (PHP 5, PHP 7) DOMNode::isDefaultNamespace — Checks if the specified namespaceURI is the default namespace or not 说明 public bool DOMNode::isDefaultNamespace ( string $namespaceURI ) Tells whether namespaceURI is the default namespace. 参数 namespaceURI The namespace URI to look for. 返回值 Return TRUE if namespaceURI is the default namespace, FALSE otherwise.
