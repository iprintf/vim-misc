OCI-Lob::append (PHP 5, PHP 7, PECL OCI8 >= 1.1.0) OCI-Lob::append — Appends data from the large object to another large object 说明 bool OCI-Lob::append ( OCI-Lob $lob_from ) Appends data from the large object to the end of another large object. Writing to the large object with this method will fail if buffering was previously enabled. You must disable buffering before appending. You may need to flush buffers with OCI-Lob::flush before disabling buffering. 参数 lob_from The copied LOB. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • OCI-Lob::flush • OCI-Lob::setBuffering • OCI-Lob::getBuffering
