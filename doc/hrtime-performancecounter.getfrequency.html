HRTime\PerformanceCounter::getFrequency (PECL hrtime >= 0.4.3) HRTime\PerformanceCounter::getFrequency — Timer frequency in ticks per second 说明 public int HRTime\PerformanceCounter::getFrequency ( void ) Returns the timer frequency in ticks per second.
