imap_num_msg (PHP 4, PHP 5, PHP 7) imap_num_msg — Gets the number of messages in the current mailbox 说明 int imap_num_msg ( resource $imap_stream ) Gets the number of messages in the current mailbox. 参数 imap_stream 由 imap_open() 返回的 IMAP 流。 返回值 Return the number of messages in the current mailbox, as an integer. 参见 • imap_num_recent() - Gets the number of recent messages in current mailbox • imap_status() - Returns status information on a mailbox User Contributed Notes 1 note =============================================================================== While imap_status sets "read"-Flag when used, imap_num_msg does not touch any flag.
