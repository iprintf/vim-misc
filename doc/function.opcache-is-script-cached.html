opcache_is_script_cached (PHP 5 >= 5.6.0, PHP 7, PECL ZendOpcache >= 7.0.4) opcache_is_script_cached — Tells whether a script is cached in OPCache 说明 boolean opcache_is_script_cached ( string $file ) This function checks if a PHP script has been cached in OPCache. This can be used to more easily detect the "warming" of the cache for a particular script. 参数 file The path to the PHP script to be checked. 返回值 Returns TRUE if file is cached in OPCache, FALSE otherwise. 参见 • opcache_compile_file() - 无需运行，即可编译并缓存 PHP 脚本
