HaruDoc::readFromStream (PECL haru >= 0.0.1) HaruDoc::readFromStream — Read data from the temporary stream 说明 string HaruDoc::readFromStream ( int $bytes ) Read data from the temporary stream. 参数 bytes The bytes parameter specifies how many bytes to read, though the stream may contain less bytes than requested. 返回值 Returns data from the temporary stream. 错误／异常 发生错误时抛出 HaruException 异常。 参见 • HaruDoc::saveToStream() - Save the document into a temporary stream • HaruDoc::resetStream() - Rewind the temporary stream • HaruDoc::getStreamSize() - Get the size of the temporary stream
