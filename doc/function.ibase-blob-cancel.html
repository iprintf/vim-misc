ibase_blob_cancel (PHP 5, PHP 7) ibase_blob_cancel — Cancel creating blob 说明 bool ibase_blob_cancel ( resource $blob_handle ) This function will discard a BLOB if it has not yet been closed by ibase_blob_close(). 参数 blob_handle A BLOB handle opened with ibase_blob_create(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ibase_blob_close() - Close blob • ibase_blob_create() - Create a new blob for adding data • ibase_blob_import() - Create blob, copy file in it, and close it
