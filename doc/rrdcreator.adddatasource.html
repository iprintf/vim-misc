RRDCreator::addDataSource (PECL rrd >= 0.9.0) RRDCreator::addDataSource — Adds data source definition for RRD database. 说明 public void RRDCreator::addDataSource ( string $description ) RRD can accept input from several data sources (DS), e.g incomming and outgoing traffic. This method adds data source by description. You need call this method for each data source. 参数 description Definition of data source - DS. This has same format as DS definition in rrd create command. See man page of rrd create for more details. 返回值 没有返回值。
