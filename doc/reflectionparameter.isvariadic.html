ReflectionParameter::isVariadic (PHP 5 >= 5.6.0, PHP 7) ReflectionParameter::isVariadic — Checks if the parameter is variadic 说明 public bool ReflectionParameter::isVariadic ( void ) Checks if the parameter was declared as a variadic parameter. 参数 此函数没有参数。 返回值 Returns TRUE if the parameter is variadic, otherwise FALSE.
