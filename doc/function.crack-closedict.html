crack_closedict (PECL crack >= 0.1) crack_closedict — Closes an open CrackLib dictionary 说明 bool crack_closedict ([ resource $dictionary ] ) crack_closedict() closes the specified dictionary identifier. Warning 此函数是实验性的。此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本 中未通知就被修改。使用本函数风险自担。 参数 dictionary The dictionary to close. If not specified, the current dictionary is closed. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • crack_opendict() - Opens a new CrackLib dictionary
