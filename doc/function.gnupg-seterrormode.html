gnupg_seterrormode (PECL gnupg >= 0.6) gnupg_seterrormode — Sets the mode for error_reporting 说明 void gnupg_seterrormode ( resource $identifier , int $errormode ) Sets the mode for error_reporting. 参数 identifier gnupg 标识符，由对 gnupg_init() 或 gnupg 的调用生成。 errormode The error mode. errormode takes a constant indicating what type of error_reporting should be used. The possible values are GNUPG_ERROR_WARNING, GNUPG_ERROR_EXCEPTION and GNUPG_ERROR_SILENT. By default GNUPG_ERROR_SILENT is used. 返回值 没有返回值。 范例 Example #1 Procedural gnupg_seterrormode() example Example #2 OO gnupg_seterrormode() example seterrormode(gnupg::ERROR_EXCEPTION); // throw an exception in case of an error ?>
