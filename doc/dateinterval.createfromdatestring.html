DateInterval::createFromDateString (PHP 5 >= 5.3.0, PHP 7) DateInterval::createFromDateString — Sets up a DateInterval from the relative parts of the string 说明 public static DateInterval DateInterval::createFromDateString ( string $time ) Uses the normal date parsers and sets up a DateInterval from the relative parts of the parsed string. 参数 time A date with relative parts. Specifically, the relative formats supported by the parser used for strtotime() and DateTime will be used to construct the DateInterval. 范例 Example #1 Parsing valid date intervals 返回值 Returns a new DateInterval instance. User Contributed Notes 1 note =============================================================================== DateInterval::createFromDateString ( string $time ) When the manual says "Uses the normal date parsers" it means that this function cannot take $time = ISO8601 strings like "P7D". If you want to use those, you must use the constructor.
