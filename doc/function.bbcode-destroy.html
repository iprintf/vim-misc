bbcode_destroy (PECL bbcode >= 0.9.0) bbcode_destroy — Close BBCode_container resource 说明 bool bbcode_destroy ( resource $bbcode_container ) This function closes the resource opened by bbcode_create(). 参数 bbcode_container BBCode_Container resource returned by bbcode_create(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
