sqlite_prev SQLiteResult::prev (PHP 5 < 5.4.0) sqlite_prev -- SQLiteResult::prev — Seek to the previous row number of a result set 说明 bool sqlite_prev ( resource $result ) 面向对象风格 (method): bool SQLiteResult::prev ( void ) sqlite_prev() seeks back the result handle to the previous row. 参数 result The SQLite result resource. This parameter is not required when using the object-oriented method. Note: 此函数不能用于未缓冲的结果句柄。 返回值 Returns TRUE on success, or FALSE if there are no more previous rows. 参见 • sqlite_has_prev() - Returns whether or not a previous row is available • sqlite_rewind() - Seek to the first row number • sqlite_next() - Seek to the next row number
