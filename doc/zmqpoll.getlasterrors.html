ZMQPoll::getLastErrors (PECL zmq >= 0.5.0) ZMQPoll::getLastErrors — Get poll errors 说明 public array ZMQPoll::getLastErrors ( void ) Returns the ids of the objects that had errors in the last poll. 参数 此函数没有参数。 返回值 Returns an array containing ids for the items that had errors in the last poll. Empty array is returned if there were no errors.
