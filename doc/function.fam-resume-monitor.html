fam_resume_monitor (PHP 5 <= 5.0.5) fam_resume_monitor — Resume suspended monitoring 说明 bool fam_resume_monitor ( resource $fam , resource $fam_monitor ) Resumes monitoring of a resource previously suspended using fam_suspend_monitor (). 参数 fam A resource representing a connection to the FAM service returned by fam_open() fam_monitor A resource returned by one of the fam_monitor_XXX functions 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • fam_suspend_monitor() - Temporarily suspend monitoring
