msg_remove_queue (PHP 4 >= 4.3.0, PHP 5, PHP 7) msg_remove_queue — Destroy a message queue 说明 bool msg_remove_queue ( resource $queue ) msg_remove_queue() destroys the message queue specified by the queue. Only use this function when all processes have finished working with the message queue and you need to release the system resources held by it. 参数 queue Message queue resource handle 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • msg_get_queue() - Create or attach to a message queue • msg_receive() - Receive a message from a message queue • msg_stat_queue() - Returns information from the message queue data structure • msg_set_queue() - Set information in the message queue data structure
