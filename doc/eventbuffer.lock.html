EventBuffer::lock (PECL event >= 1.2.6-beta) EventBuffer::lock — Acquires a lock on buffer 说明 public void EventBuffer::lock ( void ) Acquires a lock on buffer. Can be used in pair with EventBuffer::unlock() to make a set of operations atomic, i.e. thread-safe. Note, it is not needed to lock buffers for individual operations. When locking is enabled(see EventBuffer::enableLocking() ), individual operations on event buffers are already atomic. 参数 此函数没有参数。 返回值 没有返回值。 参见 • EventBuffer::unlock() - Releases lock acquired by EventBuffer::lock
