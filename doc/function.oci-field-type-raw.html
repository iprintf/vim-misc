oci_field_type_raw (PHP 5, PHP 7, PECL OCI8 >= 1.1.0) oci_field_type_raw — 返回字段的原始 Oracle 数据类型 说明 int oci_field_type_raw ( resource $statement , int $field ) oci_field_type_raw() 返回字段的原始 Oracle 数据类型。 Note: 在 PHP 5.0.0 之前的版本必须使用 ocicolumntyperaw() 替代本函数。该函数名仍然 可用，为向下兼容作为 oci_field_type_raw() 的别名。不过其已被废弃，不推荐使 用。 不过，如果需要取得字段的类型，oci_field_type() 更为适合。更多信息见 oci_field_type()。
