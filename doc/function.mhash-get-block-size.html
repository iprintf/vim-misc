mhash_get_block_size (PHP 4, PHP 5, PHP 7) mhash_get_block_size — Gets the block size of the specified hash 说明 int mhash_get_block_size ( int $hash ) Gets the size of a block of the specified hash. 参数 hash The hash ID. One of the MHASH_hashname constants. 返回值 Returns the size in bytes or FALSE, if the hash does not exist. 范例 Example #1 mhash_get_block_size() Example
