Imagick::opaquePaintImage (No version information available, might only be in Git) Imagick::opaquePaintImage — Changes the color value of any pixel that matches target 说明 bool Imagick::opaquePaintImage ( mixed $target , mixed $fill , float $fuzz , bool $invert [, int $channel = Imagick::CHANNEL_DEFAULT ] ) Changes any pixel that matches color with the color defined by fill. 此方法在 Imagick基于ImageMagick 6.3.8以上版本编译时可用。 参数 target ImagickPixel object or a string containing the color to change fill The replacement color fuzz The amount of fuzz. For example, set fuzz to 10 and the color red at intensities of 100 and 102 respectively are now interpreted as the same color. invert If TRUE paints any pixel that does not match the target color. channel Provide any channel constant that is valid for your channel mode. To apply to more than one channel, combine channel constants using bitwise operators. 缺省为Imagick::CHANNEL_DEFAULT. 参考此通道常数列表 返回值 成功时返回 TRUE。
