Imagick::sampleImage (PECL imagick 2.0.0) Imagick::sampleImage — Scales an image with pixel sampling 说明 bool Imagick::sampleImage ( int $columns , int $rows ) Scales an image to the desired dimensions with pixel sampling. Unlike other scaling methods, this method does not introduce any additional color into the scaled image. 参数 columns rows 返回值 成功时返回 TRUE。
