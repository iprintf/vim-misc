gzeof (PHP 4, PHP 5, PHP 7) gzeof — Test for EOF on a gz-file pointer 说明 int gzeof ( resource $zp ) Tests the given GZ file pointer for EOF. 参数 zp The gz-file pointer. It must be valid, and must point to a file successfully opened by gzopen(). 返回值 Returns TRUE if the gz-file pointer is at EOF or an error occurs; otherwise returns FALSE. 范例 Example #1 gzeof() example User Contributed Notes 2 notes =============================================================================== Be careful with this example. if gzopen doesn't return a valid handler, gzeof will do a nice loop. ===============================================================================
