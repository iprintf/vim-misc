IntlChar::charDigitValue (PHP 7) IntlChar::charDigitValue — Get the decimal digit value of a decimal digit character 说明 public static int IntlChar::charDigitValue ( mixed $codepoint ) Returns the decimal digit value of a decimal digit character. Such characters have the general category "Nd" (decimal digit numbers) and a Numeric_Type of Decimal. 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 The decimal digit value of codepoint, or -1 if it is not a decimal digit character. 范例 Example #1 Testing different code points 以上例程会输出： int(1) int(2) int(3) 参见 • IntlChar::getNumericValue() - Get the numeric value for a Unicode code point
