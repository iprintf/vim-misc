ssh2_sftp_realpath (PECL ssh2 >= 0.9.0) ssh2_sftp_realpath — Resolve the realpath of a provided path string 说明 string ssh2_sftp_realpath ( resource $sftp , string $filename ) Translates filename into the effective real path on the remote filesystem. 参数 sftp An SSH2 SFTP resource opened by ssh2_sftp(). filename 返回值 Returns the real path as a string. 范例 Example #1 Resolving a pathname 参见 • realpath() - 返回规范化的绝对路径名 • ssh2_sftp_symlink() - Create a symlink • ssh2_sftp_readlink() - Return the target of a symbolic link
