ncurses_clrtobot (PHP 4 >= 4.0.7, PHP 5 < 5.3.0, PECL ncurses >= 1.0.0) ncurses_clrtobot — Clear screen from current position to bottom 说明 bool ncurses_clrtobot ( void ) Warning 此函数是实验性的。此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本 中未通知就被修改。使用本函数风险自担。 Erases all lines from cursor to end of screen and creates blanks. Blanks created by ncurses_clrtobot() have the current background rendition. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ncurses_clear() - Clear screen • ncurses_clrtoeol() - Clear screen from current position to end of line
