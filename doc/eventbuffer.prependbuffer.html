EventBuffer::prependBuffer (PECL event >= 1.2.6-beta) EventBuffer::prependBuffer — Moves all data from source buffer to the front of current buffer 说明 public bool EventBuffer::prependBuffer ( EventBuffer $buf ) Behaves as EventBuffer::addBuffer() , except that it moves data to the front of the buffer. 参数 buf Source buffer. 返回值 Returns TRUE on success. Otherwise FALSE. 参见 • EventBuffer::add() - Append data to the end of an event buffer • EventBuffer::addBuffer() - Move all data from a buffer provided to the current instance of EventBuffer • EventBuffer::prepend() - Prepend data to the front of the buffer
