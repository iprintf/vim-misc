fann_get_cascade_max_out_epochs (PECL fann >= 1.0.0) fann_get_cascade_max_out_epochs — Returns the maximum out epochs 说明 int fann_get_cascade_max_out_epochs ( resource $ann ) The maximum out epochs determines the maximum number of epochs the output connections may be trained after adding a new candidate neuron. The default max out epochs is 150. 参数 ann Neural network resource. 返回值 The maximum out epochs, or FALSE on error. 参见 • fann_set_cascade_max_out_epochs() - Sets the maximum out epochs
