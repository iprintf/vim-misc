gupnp_context_unhost_path (PECL gupnp >= 0.1.0) gupnp_context_unhost_path — Stop hosting 说明 bool gupnp_context_unhost_path ( resource $context , string $server_path ) Stop hosting the file or folder at server_path. 参数 context A context identifier, returned by gupnp_context_new(). server_path Web server path where the file or folder is hosted. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • gupnp_context_new() - Create a new context • gupnp_context_host_path() - Start hosting
