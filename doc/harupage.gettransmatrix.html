HaruPage::getTransMatrix (PECL haru >= 0.0.1) HaruPage::getTransMatrix — Get the current transformation matrix of the page 说明 array HaruPage::getTransMatrix ( void ) Get the current transformation matrix of the page. 参数 此函数没有参数。 返回值 Returns the current transformation matrix of the page as an array of 6 elements: "a", "b", "c", "d", "x" and "y". 错误／异常 发生错误时抛出 HaruException 异常。 参见 • HaruPage::concat() - Concatenate current transformation matrix of the page and the specified matrix
