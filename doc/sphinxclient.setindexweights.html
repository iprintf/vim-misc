SphinxClient::setIndexWeights (PECL sphinx >= 0.1.0) SphinxClient::setIndexWeights — Set per-index weights 说明 public bool SphinxClient::setIndexWeights ( array $weights ) Sets per-index weights and enables weighted summing of match weights across different indexes. 参数 weights An associative array mapping string index names to integer weights. Default is empty array, i.e. weighting summing is disabled. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
