Gmagick::edgeimage (PECL gmagick >= Unknown) Gmagick::edgeimage — Enhance edges within the image 说明 public Gmagick Gmagick::edgeimage ( float $radius ) Enhance edges within the image with a convolution filter of the given radius. Use radius 0 and it will be auto-selected. 参数 radius The radius of the operation. 返回值 The Gmagick object with edges enhanced. 错误／异常 错误时抛出 GmagickException。
