Gmagick::profileimage (PECL gmagick >= Unknown) Gmagick::profileimage — Adds or removes a profile from an image 说明 public Gmagick Gmagick::profileimage ( string $name , string $profile ) Adds or removes a ICC, IPTC, or generic profile from an image. If the profile is NULL, it is removed from the image otherwise added. Use a name of '*' and a profile of NULL to remove all profiles from the image. 参数 name Name of profile to add or remove: ICC, IPTC, or generic profile. profile The profile. 返回值 The Gmagick object on success 错误／异常 错误时抛出 GmagickException。
