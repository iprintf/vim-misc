fann_get_rprop_delta_zero (PECL fann >= 1.0.0) fann_get_rprop_delta_zero — Returns the initial step-size 说明 ReturnType fann_get_rprop_delta_zero ( resource $ann ) The initial step-size is a positive number determining the initial step size.
