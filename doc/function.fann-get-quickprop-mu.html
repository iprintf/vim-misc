fann_get_quickprop_mu (PECL fann >= 1.0.0) fann_get_quickprop_mu — Returns the mu factor 说明 float fann_get_quickprop_mu ( resource $ann ) The mu factor is used to increase and decrease the step-size during quickprop training. The mu factor should always be above 1, since it would otherwise decrease the step-size when it was suppose to increase it. The default mu factor is 1.75. 参数 ann Neural network resource. 返回值 The mu factor, or FALSE on error. 参见 • fann_set_quickprop_mu() - Sets the quickprop mu factor
