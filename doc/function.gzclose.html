gzclose (PHP 4, PHP 5, PHP 7) gzclose — Close an open gz-file pointer 说明 bool gzclose ( resource $zp ) Closes the given gz-file pointer. 参数 zp The gz-file pointer. It must be valid, and must point to a file successfully opened by gzopen(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 gzclose() example 参见 • gzopen() - Open gz-file
