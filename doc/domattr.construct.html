DOMAttr::__construct (PHP 5, PHP 7) DOMAttr::__construct — Creates a new DOMAttr object 说明 public DOMAttr::__construct ( string $name [, string $value ] ) Creates a new DOMAttr object. This object is read only. It may be appended to a document, but additional nodes may not be appended to this node until the node is associated with a document.
