SplFixedArray::valid (PHP 5 >= 5.3.0, PHP 7) SplFixedArray::valid — Check whether the array contains more elements 说明 public bool SplFixedArray::valid ( void ) Checks if the array contains any more elements. 参数 此函数没有参数。 返回值 Returns TRUE if the array contains any more elements, FALSE otherwise. User Contributed Notes 1 note =============================================================================== This function will return TRUE whenever the iterator is within the array bounds, even if unset() has been called on the current element.
