IntlChar::isdigit (PHP 7) IntlChar::isdigit — Check if code point is a digit character 说明 public static bool IntlChar::isdigit ( mixed $codepoint ) Determines whether the specified code point is a digit character. TRUE for characters with general category "Nd" (decimal digit numbers). Beginning with Unicode 4, this is the same as testing for the Numeric_Type of Decimal. 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 Returns TRUE if codepoint is a digit character, FALSE if not. 范例 Example #1 Testing different code points 以上例程会输出： bool(false) bool(true) bool(false) 参见 • IntlChar::isalpha() - Check if code point is a letter character • IntlChar::isalnum() - Check if code point is an alphanumeric character • IntlChar::isxdigit() - Check if code point is a hexadecimal digit
