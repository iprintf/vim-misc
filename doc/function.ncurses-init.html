ncurses_init (PHP 4 >= 4.0.7, PHP 5 < 5.3.0, PECL ncurses >= 1.0.0) ncurses_init — Initialize ncurses 说明 void ncurses_init ( void ) Initializes the ncurses interface. This function must be used before any other ncurses function call. Note that ncurses_end() must be called before exiting from the program, or the terminal will not be restored to its proper non-visual mode. 参数 此函数没有参数。 返回值 没有返回值。 参见 • ncurses_end() - Stop using ncurses, clean up the screen
