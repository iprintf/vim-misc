ifx_textasvarchar (PHP 4, PHP <=5.2.0) ifx_textasvarchar — Set the default text mode 说明 bool ifx_textasvarchar ( int $mode ) Sets the default text mode for all select-queries. 参数 mode Mode "0" will return a blob id, and mode "1" will return a varchar with text content. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ifx_byteasvarchar() - Set the default byte mode
