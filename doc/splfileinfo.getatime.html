SplFileInfo::getATime (PHP 5 >= 5.1.2, PHP 7) SplFileInfo::getATime — Gets last access time of the file 说明 public int SplFileInfo::getATime ( void ) Gets the last access time for the file. 参数 此函数没有参数。 返回值 Returns the time the file was last accessed. 错误／异常 Throws RunTimeException on error. 参见 • fileatime() - 取得文件的上次访问时间
