event_base_loop (PECL libevent >= 0.0.1) event_base_loop — Handle events 说明 int event_base_loop ( resource $event_base [, int $flags = 0 ] ) Starts event loop for the specified event base. 参数 event_base Valid event base resource. flags Optional parameter, which can take any combination of EVLOOP_ONCE and EVLOOP_NONBLOCK. 返回值 event_base_loop() returns 0 on success, -1 on error and 1 if no events were registered.
