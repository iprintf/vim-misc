event_buffer_disable (PECL libevent >= 0.0.1) event_buffer_disable — Disable a buffered event 说明 bool event_buffer_disable ( resource $bevent , int $events ) Disables the specified buffered event. 参数 bevent Valid buffered event resource. events Any combination of EV_READ and EV_WRITE. 返回值 event_buffer_disable() returns TRUE on success or FALSE on error.
