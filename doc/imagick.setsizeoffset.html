Imagick::setSizeOffset (PECL imagick 2.0.0) Imagick::setSizeOffset — Sets the size and offset of the Imagick object 说明 bool Imagick::setSizeOffset ( int $columns , int $rows , int $offset ) Sets the size and offset of the Imagick object. Set it before you read a raw image format such as RGB, GRAY, or CMYK. 此方法在Imagick基于ImageMagick 6.2.9以 上版本编译时可用。 参数 columns The width in pixels. rows The height in pixels. offset The image offset. 返回值 成功时返回 TRUE。
