ReflectionProperty::export (PHP 5, PHP 7) ReflectionProperty::export — Export Description public static string ReflectionProperty::export ( mixed $class , string $name [, bool $return ] ) Exports a reflection. Warning This function is currently not documented; only its argument list is available. Parameters argument The reflection to export. name The property name. return Setting to TRUE will return the export, as opposed to emitting it. Setting to FALSE (the default) will do the opposite.
