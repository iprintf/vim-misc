jpeg2wbmp (PHP 4 >= 4.0.5, PHP 5, PHP 7) jpeg2wbmp — 将 JPEG 图像文件转换为 WBMP 图像文件 说明 bool jpeg2wbmp ( string $jpegname , string $wbmpname , int $dest_height , int $dest_width , int $threshold ) 将 JPEG 图像文件转换为 WBMP 图像文件。 参数 jpegname JPEG 文件的路径。 wbmpname 目标 WBMP 文件的路径。 dest_height 目标图像高度。 dest_width 目标图像宽度。 threshold 阈值，在 0 和 8 之间（含）。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 jpeg2wbmp() 例子 注释 Note: WBMP 支持仅在 PHP 与 GD-1.8 或更高版本一起编译时可用。 Note: JPEG 支持仅在 PHP 与 GD-1.8 或更高版本一起编译时可用。 参见 • png2wbmp() - 将 PNG 图像文件转换为 WBMP 图像文件 User Contributed Notes 1 note =============================================================================== Actually this produces black and white images only, as does the jpeg2wbmp function.
