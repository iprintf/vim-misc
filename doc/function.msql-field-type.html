msql_field_type (PHP 4, PHP 5, PHP 7) msql_field_type — Get field type 说明 string msql_field_type ( resource $result , int $field_offset ) msql_field_type() gets the type of the specified field index. 参数 result resource 型的结果集。此结果集来自对 msql_query() 的调用。 field_offset 数值型字段偏移量。 field_offset 从 1 开始。 返回值 The type of the field. One of int, char, real, ident, null or unknown. This functions will return FALSE on failure.
