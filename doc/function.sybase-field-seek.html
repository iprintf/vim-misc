sybase_field_seek (PHP 4, PHP 5, PHP 7) sybase_field_seek — Sets field offset 说明 bool sybase_field_seek ( resource $result , int $field_offset ) Seeks to the specified field offset. If the next call to sybase_fetch_field() won't include a field offset, this field would be returned. 参数 result field_offset 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • sybase_fetch_field() - Get field information from a result
