fbsql_set_transaction (PHP 4 >= 4.2.0, PHP 5, PHP 7) fbsql_set_transaction — Set the transaction locking and isolation 说明 void fbsql_set_transaction ( resource $link_identifier , int $locking , int $isolation ) Sets the transaction locking and isolation. 参数 link_identifier 由 fbsql_connect() 或 fbsql_pconnect() 返回的 FrontBase 链接标识符. 如可选并未指定，函数会尝试查找一个到 FrontBase 服务器的开放链接，且如没有找 到这样的链接，会尝试通过不带参数调用 fbsql_connect() 来创建一个。 locking The type of locking to be set. It can be one of the following constants: FBSQL_LOCK_DEFERRED, FBSQL_LOCK_OPTIMISTIC, or FBSQL_LOCK_PESSIMISTIC. isolation The type of isolation to be set. It can be one of the following constants: FBSQL_ISO_READ_UNCOMMITTED, FBSQL_ISO_READ_COMMITTED, FBSQL_ISO_REPEATABLE_READ, FBSQL_ISO_SERIALIZABLE, or FBSQL_ISO_VERSIONED. 返回值 没有返回值。
