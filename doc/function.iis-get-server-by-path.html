iis_get_server_by_path (PECL iisfunc SVN) iis_get_server_by_path — Return the instance number associated with the Path 说明 int iis_get_server_by_path ( string $path ) Each virtual server in IIS is associated with an instance number. iis_get_server_by_path() finds the instance number from the actual path to the root directory. 参数 path The path to the root directory 返回值 Returns the server instance number.
