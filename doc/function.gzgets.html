gzgets (PHP 4, PHP 5, PHP 7) gzgets — Get line from file pointer 说明 string gzgets ( resource $zp [, int $length ] ) Gets a (uncompressed) string of up to length - 1 bytes read from the given file pointer. Reading ends when length - 1 bytes have been read, on a newline, or on EOF (whichever comes first). 参数 zp The gz-file pointer. It must be valid, and must point to a file successfully opened by gzopen(). length The length of data to get. 返回值 The uncompressed string, or FALSE on error. 范例 Example #1 gzgets() example 参见 • gzopen() - Open gz-file • gzgetc() - Get character from gz-file pointer • gzwrite() - Binary-safe gz-file write User Contributed Notes 2 notes =============================================================================== =============================================================================== For the above example by VIJAY, using gzgetc would be better, as I've encountered binary/text file incompatibilities (at least with PHP 4.0.4).
