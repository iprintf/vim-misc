cubrid_get_class_name (PECL CUBRID >= 8.3.0) cubrid_get_class_name — Get the class name using OID 说明 string cubrid_get_class_name ( resource $conn_identifier , string $oid ) The cubrid_get_class_name() function is used to get the class name from oid. It doesn't work when selecting data from the system tables, for example db_class. 参数 conn_identifier Connection identifier. oid OID of the instance that you want to check the existence. 返回值 Class name when process is successful. FALSE, when process is unsuccessful. 范例 Example #1 cubrid_get_class_name() example 以上例程会输出： code 参见 • cubrid_is_instance() - Check whether the instance pointed by OID exists • cubrid_drop() - Delete an instance using OID
