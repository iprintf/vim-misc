dbase_numrecords (PHP 4, PHP 5, PHP 7) dbase_numrecords — Gets the number of records in a database 说明 int dbase_numrecords ( int $dbase_identifier ) Gets the number of records (rows) in the specified database. Note: Record numbers are between 1 and dbase_numrecords($db), while field numbers are between 0 and dbase_numfields($db)-1. 参数 dbase_identifier The database link identifier, returned by dbase_open() or dbase_create(). 返回值 The number of records in the database, or FALSE if an error occurs. 范例 Example #1 Looping over all the records of the database 参见 • dbase_numfields() - Gets the number of fields of a database User Contributed Notes 1 note =============================================================================== Trying to get number of records in 1 of my "MySQL" tables.
