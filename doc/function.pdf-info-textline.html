User Contributed Notes 1 note =============================================================================== You can use PDF_info_textline to center some text in the page. Here's a little function the PDF handle // $text -> the text to center // $posy -> the vertical position of the text // $pageWidth -> the width of the page, defaults to 595 (A4) function center_text($p, $text, $posy, $pageWidth=595) { $width = PDF_info_textline($p, $text, 'width', ''); PDF_fit_textline($p, $text, ($pageWidth-$width)/2, $posy, ''); } ?>
