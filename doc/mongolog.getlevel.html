MongoLog::getLevel (PECL mongo >= 1.2.3) MongoLog::getLevel — Gets the level(s) currently being logged 说明 public static int MongoLog::getLevel ( void ) This function can be used to see which log levels are currently enabled. The returned integer may be compared with the MongoLog level constants using bitwise operators to check for specific log levels. 参数 此函数没有参数。 返回值 Returns the level(s) currently being logged.
