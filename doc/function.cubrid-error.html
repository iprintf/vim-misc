cubrid_error (PECL CUBRID >= 8.3.1) cubrid_error — Get the error message 说明 string cubrid_error ([ resource $connection ] ) The cubrid_error() function is used to get the error message that occurred during the use of CUBRID API. Usually, it gets error message when API returns false as its return value. 参数 connection The CUBRID connection. 返回值 Error message that occurred. 范例 Example #1 cubrid_error() example 以上例程会输出： Error Code: -493 Error Message: Syntax: Unknown class "person". select id, [name] from person 参见 • cubrid_errno() - Return the numerical value of the error message from previous CUBRID operation • cubrid_error_code() - Get error code for the most recent function call • cubrid_error_msg() - Get last error message for the most recent function call
