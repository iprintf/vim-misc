Yaf_Route_Interface::route (Yaf >=1.0.0) Yaf_Route_Interface::route — route a request 说明 abstract public bool Yaf_Route_Interface::route ( Yaf_Request_Abstract $request ) Yaf_Route_Interface::route() 是用户自定义路由唯一需要实现的方法。 如果这个方法返回TRUE，那么路由进程将会中止。否则，Yaf_Router 将会调用路由堆栈中 的下一个路由来路由请求。 这个方法会设置路由的结果给参数请求，通过调用 Yaf_Request_Abstract::setControllerName()，Yaf_Request_Abstract::setActionName () 和 Yaf_Request_Abstract::setModuleName() 这个方法也需要调用Yaf_Request_Abstract::setRouted()做最后的请求路由。 Warning 本函数还未编写文档，仅有参数列表。 参数 request A Yaf_Request_Abstract instance. 返回值
