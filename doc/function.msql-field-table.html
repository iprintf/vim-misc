msql_field_table (PHP 4, PHP 5, PHP 7) msql_field_table — Get table name for field 说明 int msql_field_table ( resource $result , int $field_offset ) Returns the name of the table that the specified field is in. 参数 result resource 型的结果集。此结果集来自对 msql_query() 的调用。 field_offset 数值型字段偏移量。 field_offset 从 1 开始。 返回值 The name of the table on success 或者在失败时返回 FALSE.
