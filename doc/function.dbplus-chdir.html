dbplus_chdir (PHP 4 <= 4.1.0, PECL dbplus >= 0.9) dbplus_chdir — Get/Set database virtual current directory 说明 string dbplus_chdir ([ string $newdir ] ) Changes the virtual current directory where relation files will be looked for by dbplus_open(). 参数 newdir The new directory for relation files. You can omit this parameter to query the current working directory. 返回值 Returns the absolute path of the current directory. 注释 Warning 此函数是实验性的。此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本 中未通知就被修改。使用本函数风险自担。
