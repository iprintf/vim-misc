ImagickDraw::getStrokeMiterLimit (PECL imagick 2.0.0) ImagickDraw::getStrokeMiterLimit — Returns the stroke miter limit 说明 int ImagickDraw::getStrokeMiterLimit ( void ) Warning 本函数还未编写文档，仅有参数列表。 Returns the miter limit. When two line segments meet at a sharp angle and miter joins have been specified for 'lineJoin', it is possible for the miter to extend far beyond the thickness of the line stroking the path. The 'miterLimit' imposes a limit on the ratio of the miter length to the 'lineWidth'. 返回值 Returns an int describing the miter limit and 0 if no miter limit is set.
