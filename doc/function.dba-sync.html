dba_sync (PHP 4, PHP 5, PHP 7) dba_sync — Synchronize database 说明 bool dba_sync ( resource $handle ) dba_sync() synchronizes the database. This will probably trigger a physical write to the disk, if supported. 参数 handle The database handler, returned by dba_open() or dba_popen(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • dba_optimize() - Optimize database
