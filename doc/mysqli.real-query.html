mysqli::real_query mysqli_real_query (PHP 5, PHP 7) mysqli::real_query -- mysqli_real_query — 执行一个mysql查询 说明 面向对象风格 bool mysqli::real_query ( string $query ) 过程化风格 bool mysqli_real_query ( mysqli $link , string $query ) 执行一个单条数据库查询, 其结果可以使用mysqli_store_result()或mysqli_use_result ()检索或存储. 为了确定给定的查询是否真的返回一个结果集, 可以查看mysqli_field_count(). 参数 link 仅以过程化样式：由mysqli_connect() 或 mysqli_init() 返回的链接标识。 query 查询字符串 查询中的数据可以进行属性转义. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • mysqli_query() - 对数据库执行一次查询 • mysqli_store_result() - Transfers a result set from the last query • mysqli_use_result() - Initiate a result set retrieval
