dbplus_errcode (PHP 4 <= 4.1.0, PECL dbplus >= 0.9) dbplus_errcode — Get error string for given errorcode or last error 说明 string dbplus_errcode ([ int $errno ] ) Returns a clear error string for the given error code. 参数 errno The error code. If not provided, the result code of the last db++ operation is assumed. 返回值 Returns the error message. 注释 Warning 此函数是实验性的。此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本 中未通知就被修改。使用本函数风险自担。
