RarEntry::isEncrypted (PECL rar >= 2.0.0) RarEntry::isEncrypted — Test whether an entry is encrypted 说明 public bool RarEntry::isEncrypted ( void ) Tests whether the current entry contents are encrypted. Note: The password used may differ between files inside the same RAR archive. 参数 此函数没有参数。 返回值 Returns TRUE if the current entry is encrypted and FALSE otherwise.
