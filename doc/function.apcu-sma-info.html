apcu_sma_info (PECL apcu >= 4.0.0) apcu_sma_info — Retrieves APCu Shared Memory Allocation information 说明 array apcu_sma_info ([ bool $limited = false ] ) Retrieves APCu Shared Memory Allocation information. 参数 limited When set to FALSE (default) apcu_sma_info() will return a detailed information about each segment. 返回值 Array of Shared Memory Allocation data; FALSE on failure. 范例 Example #1 A apcu_sma_info() example 以上例程的输出类似于： Array ( [num_seg] => 1 [seg_size] => 31457280 [avail_mem] => 31448408 [block_lists] => Array ( [0] => Array ( [0] => Array ( [size] => 31448408 [offset] => 8864 ) ) ) ) 参见 • APCu configuration directives
