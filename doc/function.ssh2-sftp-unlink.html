ssh2_sftp_unlink (PECL ssh2 >= 0.9.0) ssh2_sftp_unlink — Delete a file 说明 bool ssh2_sftp_unlink ( resource $sftp , string $filename ) Deletes a file on the remote filesystem. 参数 sftp An SSH2 SFTP resource opened by ssh2_sftp(). filename 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 Deleting a file 参见 • unlink() - 删除文件 User Contributed Notes 1 note =============================================================================== If you've already got a filepath of the form "ssh2.sftp://$sftp/path/to/file" you can just use unlink() on that path
