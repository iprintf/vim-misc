expect_popen (PECL expect >= 0.1.0) expect_popen — Execute command via Bourne shell, and open the PTY stream to the process 说明 resource expect_popen ( string $command ) Execute command via Bourne shell, and open the PTY stream to the process. 参数 command Command to execute. 返回值 Returns an open PTY stream to the processes stdio, stdout, and stderr. On failure this function returns FALSE. 范例 Example #1 expect_popen() example 参见 • popen() - 打开进程文件指针
