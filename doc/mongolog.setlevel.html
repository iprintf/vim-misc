MongoLog::setLevel (PECL mongo >= 1.2.3) MongoLog::setLevel — Sets the level(s) to be logged 说明 public static void MongoLog::setLevel ( int $level ) This function can be used to control logging verbosity and the types of activities that should be logged. The MongoLog level constants may be used with bitwise operators to specify multiple levels. Note that you must also call MongoLog::setModule() to specify which modules(s) of the driver should log. 参数 level The level(s) you would like to log.
