iconv_set_encoding (PHP 4 >= 4.0.5, PHP 5, PHP 7) iconv_set_encoding — 为字符编码转换设定当前设置 说明 bool iconv_set_encoding ( string $type , string $charset ) 将 type 设置的值从内部配置变量更改为 charset。 参数 type type 的值可以是以下其中任意一个： □ input_encoding □ output_encoding □ internal_encoding charset 字符集。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 iconv_set_encoding() 例子 参见 • iconv_get_encoding() - 获取 iconv 扩展的内部配置变量 • ob_iconv_handler() - 以输出缓冲处理程序转换字符编码 User Contributed Notes 2 notes =============================================================================== As of PHP 5.6, this throws a Deprecated level error, as is consistent with setting the iconv.*_encoding ini settings: iconv_set_encoding('internal_encoding', 'UTF-8'); triggers: PHP Deprecated: iconv_set_encoding(): Use of iconv.internal_encoding is deprecated in ... =============================================================================== On PHP 5.2.5 when iconv.*_encoding values are set using php_admin_value in the Apache configuration file, this call will fail and return false.
