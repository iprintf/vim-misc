posix_getgroups (PHP 4, PHP 5, PHP 7) posix_getgroups — Return the group set of the current process 说明 array posix_getgroups ( void ) Gets the group set of the current process. 返回值 Returns an array of integers containing the numeric group ids of the group set of the current process. 范例 Example #1 Example use of posix_getgroups() 以上例程的输出类似于： Array ( [0] => 4 [1] => 20 [2] => 24 [3] => 25 [4] => 29 [5] => 30 [6] => 33 [7] => 44 [8] => 46 [9] => 104 [10] => 109 [11] => 110 [12] => 1000 ) 参见 • posix_getgrgid() - Return info about a group by group id
