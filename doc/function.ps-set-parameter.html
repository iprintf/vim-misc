ps_set_parameter (PECL ps >= 1.1.0) ps_set_parameter — Sets certain parameters 说明 bool ps_set_parameter ( resource $psdoc , string $name , string $value ) Sets several parameters which are used by many functions. Parameters are by definition string values. 参数 psdoc Resource identifier of the postscript file as returned by ps_new(). name For a list of possible names see ps_get_parameter(). value The value of the parameter. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ps_get_parameters() • ps_set_value() - Sets certain values User Contributed Notes 1 note =============================================================================== Note that when setting a parameter such as "true" or "false", in this case "true" and "false" are strings.
