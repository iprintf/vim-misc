gmp_clrbit (PHP 4 >= 4.0.4, PHP 5, PHP 7) gmp_clrbit — Clear bit 说明 void gmp_clrbit ( GMP $a , int $index ) Clears (sets to 0) bit index in a. The index starts at 0. 参数 a 可以是一个 GMP 数据 resouce，或一个可以转换为数值的字符串。 index The index of the bit to clear. Index 0 represents the least significant bit. 返回值 GMP 数值资源. 范例 Example #1 gmp_clrbit() example 以上例程会输出： 254 注释 Note: Unlike most of the other GMP functions, gmp_clrbit() must be called with a GMP resource that already exists (using gmp_init() for example). One will not be automatically created. 参见 • gmp_setbit() - Set bit • gmp_testbit() - Tests if a bit is set
