mysqli::more_results mysqli_more_results (PHP 5, PHP 7) mysqli::more_results -- mysqli_more_results — Check if there are any more query results from a multi query 说明 面向对象风格 bool mysqli::more_results ( void ) 过程化风格 bool mysqli_more_results ( mysqli $link ) Indicates if one or more result sets are available from a previous call to mysqli_multi_query(). 参数 link 仅以过程化样式：由mysqli_connect() 或 mysqli_init() 返回的链接标识。 返回值 Returns TRUE if one or more result sets are available from a previous call to mysqli_multi_query(), otherwise FALSE. 范例 See mysqli_multi_query(). 参见 • mysqli_multi_query() - Performs a query on the database • mysqli_next_result() - Prepare next result from multi_query • mysqli_store_result() - Transfers a result set from the last query • mysqli_use_result() - Initiate a result set retrieval
