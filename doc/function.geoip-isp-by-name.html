geoip_isp_by_name (PECL geoip >= 1.0.2) geoip_isp_by_name — Get the Internet Service Provider (ISP) name 说明 string geoip_isp_by_name ( string $hostname ) The geoip_isp_by_name() function will return the name of the Internet Service Provider (ISP) that an IP is assigned to. This function is currently only available to users who have bought a commercial GeoIP ISP Edition. A warning will be issued if the proper database cannot be located. 参数 hostname The hostname or IP address. 返回值 Returns the ISP name on success, or FALSE if the address cannot be found in the database. 范例 Example #1 A geoip_isp_by_name() example This will print the ISP name of host example.com. 以上例程会输出： This host IP is from ISP: ICANN c/o Internet Assigned Numbers Authority
