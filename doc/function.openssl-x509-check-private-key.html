openssl_x509_check_private_key (PHP 4 >= 4.2.0, PHP 5, PHP 7) openssl_x509_check_private_key — Checks if a private key corresponds to a certificate 说明 bool openssl_x509_check_private_key ( mixed $cert , mixed $key ) Checks whether the given key is the private key that corresponds to cert. 参数 cert The certificate. key The private key. 返回值 Returns TRUE if key is the private key that corresponds to cert, or FALSE otherwise. User Contributed Notes 2 notes =============================================================================== This function will return FALSE if the private key requires a pass phrase. =============================================================================== Please note that this function only compares the public key component of the private key with the public key residing in the certificate.
