fbsql_commit (PHP 4 >= 4.0.6, PHP 5, PHP 7) fbsql_commit — Commits a transaction to the database 说明 bool fbsql_commit ([ resource $link_identifier ] ) Ends the current transaction by writing all inserts, updates and deletes to the disk and unlocking all row and table locks held by the transaction. This command is only needed if autocommit is set to false. 参数 link_identifier 由 fbsql_connect() 或 fbsql_pconnect() 返回的 FrontBase 链接标识符. 如可选并未指定，函数会尝试查找一个到 FrontBase 服务器的开放链接，且如没有找 到这样的链接，会尝试通过不带参数调用 fbsql_connect() 来创建一个。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • fbsql_autocommit() - Enable or disable autocommit • fbsql_rollback() - Rollback a transaction to the database
