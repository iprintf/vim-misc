Imagick::setImageOpacity (PECL imagick 2.0.0) Imagick::setImageOpacity — Sets the image opacity level 说明 bool Imagick::setImageOpacity ( float $opacity ) Sets the image to the specified opacity level. 此方法在Imagick基于ImageMagick 6.3.1以上版本编译时可用。 This method operates on all channels, which means that for example opacity value of 0.5 will set all transparent areas to partially opaque. To add transparency to areas that are not already transparent use Imagick::evaluateImage() 参数 opacity The level of transparency: 1.0 is fully opaque and 0.0 is fully transparent. 返回值 成功时返回 TRUE。 范例 Example #1 A Imagick::setImageOpacity() example An example of using Imagick::setImageOpacity() setImageOpacity(0.7); /* output the image */ header('Content-type: image/png'); echo $image; ?>
