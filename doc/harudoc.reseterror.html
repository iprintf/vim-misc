HaruDoc::resetError (PECL haru >= 0.0.1) HaruDoc::resetError — Reset error state of the document handle 说明 bool HaruDoc::resetError ( void ) Once an error code is set, most of the operations, including I/O processing functions cannot be performed. In case if you want to continue after the cause of the error has been fixed, you have to invoke this function in order to reset the document error state. 参数 此函数没有参数。 返回值 Always succeeds and returns TRUE.
