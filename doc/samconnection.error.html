SAMConnection::error (PECL sam >= 0.1.0) SAMConnection::error — Contains the text description of the last failed SAM operation. 说明 string $SAMConnection->error; Contains the text description of the last failed SAM operation on this connection. If the last operation completed successfully this property contains an empty string. 返回值 A string containing the text description of the last error type encountered on the connection.
