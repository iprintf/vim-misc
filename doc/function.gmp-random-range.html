gmp_random_range (PHP 5 >= 5.6.3, PHP 7) gmp_random_range — Random number 说明 GMP gmp_random_range ( GMP $min , GMP $max ) Generate a random number. The number will be between min and max. min and max can both be negative but min must always be less than max. 参数 min A GMP number representing the lower bound for the random number max A GMP number representing the upper bound for the random number 返回值 A random GMP number. 范例 Example #1 gmp_random_range() example 以上例程会输出： 42 -67
