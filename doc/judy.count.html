Judy::count (PECL judy >= 0.1.1) Judy::count — Count the number of elements in the Judy array 说明 public int Judy::count ([ int $index_start = 0 [, int $index_end = -1 ]] ) Count the number of elements in the Judy array. 参数 index_start Start counting from the given index. Default is first index. index_end Stop counting when reaching this index. Default is last index. 返回值 Return the number of elements.
