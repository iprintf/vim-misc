streamWrapper::stream_close (PHP 4 >= 4.3.2, PHP 5, PHP 7) streamWrapper::stream_close — Close a resource 说明 public void streamWrapper::stream_close ( void ) This method is called in response to fclose(). All resources that were locked, or allocated, by the wrapper should be released. 参数 此函数没有参数。 返回值 没有返回值。 参见 • fclose() - 关闭一个已打开的文件指针 • streamWrapper::dir_closedir() - Close directory handle
