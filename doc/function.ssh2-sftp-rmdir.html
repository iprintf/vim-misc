ssh2_sftp_rmdir (PECL ssh2 >= 0.9.0) ssh2_sftp_rmdir — Remove a directory 说明 bool ssh2_sftp_rmdir ( resource $sftp , string $dirname ) Removes a directory from the remote file server. This function is similar to using rmdir() with the ssh2.sftp:// wrapper. 参数 sftp An SSH2 SFTP resource opened by ssh2_sftp(). dirname 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 Removing a directory on a remote server 参见 • rmdir() - 删除目录 • ssh2_sftp_mkdir() - Create a directory
