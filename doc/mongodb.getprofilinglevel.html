MongoDB::getProfilingLevel (PECL mongo >=0.9.0) MongoDB::getProfilingLevel — Gets this database's profiling level 说明 public int MongoDB::getProfilingLevel ( void ) This returns the current database profiling level. The database profiler tracks query execution times. If you turn it on (say, using MongoDB::setProfilingLevel() or the shell), you can see how many queries took longer than a given number of milliseconds or the timing for all queries. Note that profiling slows down queries, so it is better to use in development or testing than in a time-sensitive application.
