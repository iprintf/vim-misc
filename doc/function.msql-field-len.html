msql_field_len (PHP 4, PHP 5, PHP 7) msql_field_len — Get field length 说明 int msql_field_len ( resource $result , int $field_offset ) msql_field_len() returns the length of the specified field. 参数 result resource 型的结果集。此结果集来自对 msql_query() 的调用。 field_offset 数值型字段偏移量。 field_offset 从 1 开始。 返回值 Returns the length of the specified field or FALSE on error.
