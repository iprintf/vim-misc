AppendIterator::next (PHP 5 >= 5.1.0, PHP 7) AppendIterator::next — Moves to the next element 说明 public void AppendIterator::next ( void ) Moves to the next element. If this means to another Iterator then it rewinds that Iterator. 参数 此函数没有参数。 返回值 没有返回值。 参见 • Iterator::next() - 向前移动到下一个元素 • AppendIterator::current() - Gets the current value • AppendIterator::key() - Gets the current key • AppendIterator::valid() - Checks validity of the current element • AppendIterator::rewind() - Rewinds the Iterator
