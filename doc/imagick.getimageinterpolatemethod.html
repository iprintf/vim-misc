Imagick::getImageInterpolateMethod (PECL imagick 2.0.0) Imagick::getImageInterpolateMethod — Returns the interpolation method 说明 int Imagick::getImageInterpolateMethod ( void ) Returns the interpolation method for the specified image. The method is one of the Imagick::INTERPOLATE_* constants. 返回值 Returns the interpolate method on success. 错误／异常 错误时抛出 ImagickException。
