DOMProcessingInstruction::__construct (PHP 5, PHP 7) DOMProcessingInstruction::__construct — Creates a new DOMProcessingInstruction object 说明 public DOMProcessingInstruction::__construct ( string $name [, string $value ] ) Creates a new DOMProcessingInstruction object. This object is read only. It may be appended to a document, but additional nodes may not be appended to this node until the node is associated with a document. To create a writeable node, use DOMDocument::createProcessingInstruction. 参数 name The tag name of the processing instruction. value The value of the processing instruction. 范例 Example #1 Creating a new DOMProcessingInstruction object appendChild(new DOMElement('html')); $body = $html->appendChild(new DOMElement('body')); $pinode = new DOMProcessingInstruction('php', 'echo "Hello World"; '); $body->appendChild($pinode); echo $dom->saveXML(); ?> 以上例程会输出： 参见 • DOMDocument::createProcessingInstruction() - Creates new PI node
