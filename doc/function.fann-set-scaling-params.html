fann_set_scaling_params (PECL fann >= 1.0.0) fann_set_scaling_params — Calculate input and output scaling parameters for future use based on training data 说明 bool fann_set_scaling_params ( resource $ann , resource $train_data , float $new_input_min , float $new_input_max , float $new_output_min , float $new_output_max ) Calculate input and output scaling parameters for future use based on training data. 参数 ann Neural network resource. train_data Neural network training data resource. new_input_min The desired lower bound in input data after scaling (not strictly followed) new_input_max The desired upper bound in input data after scaling (not strictly followed) new_output_min The desired lower bound in output data after scaling (not strictly followed) new_output_max The desired upper bound in output data after scaling (not strictly followed) 返回值 Returns TRUE on success, or FALSE otherwise. 参见 • fann_set_input_scaling_params() - Calculate input scaling parameters for future use based on training data • fann_set_output_scaling_params() - Calculate output scaling parameters for future use based on training data
