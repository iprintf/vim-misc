ldap_unbind (PHP 4, PHP 5, PHP 7) ldap_unbind — Unbind from LDAP directory 说明 bool ldap_unbind ( resource $link_identifier ) Unbinds from the LDAP directory. 参数 link_identifier An LDAP link identifier, returned by ldap_connect(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ldap_bind() - 绑定 LDAP 目录 User Contributed Notes 1 note =============================================================================== ldap_unbind kills the link descriptor. So, if you want to rebind as another user, just bind again; don't unbind. Otherwise, you'll have to open up a new connection.
