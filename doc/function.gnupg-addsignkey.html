gnupg_addsignkey (PECL gnupg >= 0.5) gnupg_addsignkey — Add a key for signing 说明 bool gnupg_addsignkey ( resource $identifier , string $fingerprint [, string $passphrase ] ) 参数 identifier gnupg 标识符，由对 gnupg_init() 或 gnupg 的调用生成。 fingerprint 指纹键名。 passphrase The pass phrase. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 Procedural gnupg_addsignkey() example Example #2 OO gnupg_addsignkey() example addsignkey("8660281B6051D071D94B5B230549F9DC851566DC","test"); ?> User Contributed Notes 1 note =============================================================================== As of gnupg version 2, it is not possible to pass a plain password any more. The parameter is simply ignored. Instead, a pinentry application will be launched in case of php running in cli mode. In cgi or apache mode, opening the key will fail. The simplest solution is to use keys without passwords.
