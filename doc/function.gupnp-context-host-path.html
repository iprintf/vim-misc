gupnp_context_host_path (PECL gupnp >= 0.1.0) gupnp_context_host_path — Start hosting 说明 bool gupnp_context_host_path ( resource $context , string $local_path , string $server_path ) Start hosting local_path at server_path. Files with the path local_path.LOCALE (if they exist) will be served up when LOCALE is specified in the request's Accept-Language header. 参数 context A context identifier, returned by gupnp_context_new(). local_path Path to the local file or folder to be hosted. server_path Web server path where local_path should be hosted. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 Create new UPnP context and set host path 参见 • gupnp_context_new() - Create a new context • gupnp_context_unhost_path() - Stop hosting
