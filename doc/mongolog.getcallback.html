MongoLog::getCallback (PECL mongo >= 1.3.0) MongoLog::getCallback — Gets the previously set callback function 说明 public static callable MongoLog::getCallback ( void ) Retrieves the callback function. 参数 此函数没有参数。 返回值 Returns the callback function, or FALSE if not set yet. 注释 Caution This function is only available with PHP 5.3.0 and later.
