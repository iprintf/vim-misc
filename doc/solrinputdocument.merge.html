SolrInputDocument::merge (PECL solr >= 0.9.2) SolrInputDocument::merge — Merges one input document into another 说明 public bool SolrInputDocument::merge ( SolrInputDocument $sourceDoc [, bool $overwrite = true ] ) Merges one input document into another. 参数 sourceDoc The source document. overwrite If this is TRUE it will replace matching fields in the destination document. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 In the future, this will be modified to return the number of fields in the new document.
