gmp_import (PHP 5 >= 5.6.1, PHP 7) gmp_import — Import from a binary string 说明 GMP gmp_import ( string $data , integer $word_size , integer $options ) Import a GMP number from a binary string 参数 data The binary string being imported word_size Default value is 1. The number of bytes in each chunk of binary data. This is mainly used in conjunction with the options parameter. options Default value is GMP_MSW_FIRST | GMP_NATIVE_ENDIAN. 返回值 Returns a GMP number 或者在失败时返回 FALSE. 范例 Example #1 gmp_import() example 以上例程会输出： 0 528
