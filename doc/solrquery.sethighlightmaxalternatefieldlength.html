SolrQuery::setHighlightMaxAlternateFieldLength (PECL solr >= 0.9.2) SolrQuery::setHighlightMaxAlternateFieldLength — Sets the maximum number of characters of the field to return 说明 public SolrQuery SolrQuery::setHighlightMaxAlternateFieldLength ( int $fieldLength [, string $field_override ] ) If SolrQuery::setHighlightAlternateField() was passed the value TRUE, this parameter specifies the maximum number of characters of the field to return Any value less than or equal to 0 means unlimited. 参数 fieldLength The length of the field field_override The name of the field. 返回值 Returns the current SolrQuery object, if the return value is used.
