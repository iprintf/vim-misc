Phar::getSupportedSignatures (PHP >= 5.3.0, PECL phar >= 1.1.0) Phar::getSupportedSignatures — Return array of supported signature types 说明 final public static array Phar::getSupportedSignatures ( void ) Return array of supported signature types 参数 No parameters. 返回值 Returns an array containing any of MD5, SHA-1, SHA-256, SHA-512, or OpenSSL. 参见 • Phar::getSignature() - Return MD5/SHA1/SHA256/SHA512/OpenSSL signature of a Phar archive • Phar::setSignatureAlgorithm() - set the signature algorithm for a phar and apply it.
