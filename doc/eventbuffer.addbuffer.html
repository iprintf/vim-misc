EventBuffer::addBuffer (PECL event >= 1.2.6-beta) EventBuffer::addBuffer — Move all data from a buffer provided to the current instance of EventBuffer 说明 public bool EventBuffer::addBuffer ( EventBuffer $buf ) Move all data from the buffer provided in buf parameter to the end of current EventBuffer . This is a destructive add. The data from one buffer moves into the other buffer. However, no unnecessary memory copies occur. 参数 buf The source EventBuffer object. 返回值 Returns TRUE on success. Otherwise FALSE. 参见 • EventBuffer::add() - Append data to the end of an event buffer
