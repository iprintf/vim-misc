mysqli::stmt_init mysqli_stmt_init (PHP 5, PHP 7) mysqli::stmt_init -- mysqli_stmt_init — 初始化一条语句并返回一个用于 mysqli_stmt_prepare(调用)的对象 说明 面向对象风格 mysqli_stmt mysqli::stmt_init ( void ) 过程化风格 mysqli_stmt mysqli_stmt_init ( mysqli $link ) 分配并初始化一个语句对象用于mysqli_stmt_prepare().
