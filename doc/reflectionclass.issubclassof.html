ReflectionClass::isSubclassOf (PHP 5, PHP 7) ReflectionClass::isSubclassOf — 检查是否为一个子类 说明 public bool ReflectionClass::isSubclassOf ( string $class ) 检查一个类是否为指定类的子类，或者实现了指定的接口。 参数 class 被检查的类名。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ReflectionClass::isInterface() - 检查类是否是一个接口（interface） • ReflectionClass::implementsInterface() - 接口的实现 • is_subclass_of() - 如果此对象是该类的子类，则返回 TRUE • get_parent_class() - 返回对象或类的父类名 User Contributed Notes 1 note =============================================================================== class A {} class B {} class C extends B {} $obj=new ReflectionClass('C'); var_dump($obj->isSubclassOf ('A')); //boolean false var_dump($obj->isSubclassOf ('B')); //boolean true
