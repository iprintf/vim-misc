IntlChar::isalpha (PHP 7) IntlChar::isalpha — Check if code point is a letter character 说明 public static bool IntlChar::isalpha ( mixed $codepoint ) Determines whether the specified code point is a letter character. TRUE for general categories "L" (letters). 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 Returns TRUE if codepoint is a letter character, FALSE if not. 范例 Example #1 Testing different code points 以上例程会输出： bool(true) bool(false) bool(false) 参见 • IntlChar::isalnum() - Check if code point is an alphanumeric character • IntlChar::isdigit() - Check if code point is a digit character
