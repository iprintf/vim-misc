dbase_close (PHP 4, PHP 5, PHP 7) dbase_close — Closes a database 说明 bool dbase_close ( int $dbase_identifier ) Closes the given database link identifier. 参数 dbase_identifier The database link identifier, returned by dbase_open() or dbase_create(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 Closing a dBase database file 参见 • dbase_open() - Opens a database • dbase_create() - Creates a database
