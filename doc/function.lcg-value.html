lcg_value (PHP 4, PHP 5, PHP 7) lcg_value — 组合线性同余发生器 说明 float lcg_value ( void ) lcg_value() 返回范围为 (0, 1) 的一个伪随机数。本函数组合了周期为 2^31 - 85 和 2 ^31 - 249 的两个同余发生器。本函数的周期等于这两个素数的乘积。 返回值 范围为 (0, 1) 的伪随机数。 参见 • rand() - 产生一个随机整数 • mt_rand() - 生成更好的随机数 User Contributed Notes 2 notes =============================================================================== Choose your weapon: =============================================================================== An elegant way to return random float between two numbers:
