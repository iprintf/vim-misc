GearmanJob::sendData (PECL gearman >= 0.6.0) GearmanJob::sendData — Send data for a running job 说明 public bool GearmanJob::sendData ( string $data ) Sends data to the job server (and any listening clients) for this job. 参数 data Arbitrary serialized data. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • GearmanJob::workload() - Get workload • GearmanTask::data() - Get data returned for a task User Contributed Notes 1 note =============================================================================== For Gearman PHP 1.1.1 sendData() only works for addTask. It does not work with doNormal, doBackground or addTaskBackground submission types
