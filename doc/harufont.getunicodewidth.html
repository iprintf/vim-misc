HaruFont::getUnicodeWidth (PECL haru >= 0.0.1) HaruFont::getUnicodeWidth — Get the width of the character in the font 说明 int HaruFont::getUnicodeWidth ( int $character ) Get the width of the character in the font. 参数 character The code of the character. 返回值 Returns the width of the character in the font. 错误／异常 发生错误时抛出 HaruException 异常。
