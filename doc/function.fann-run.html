fann_run (PECL fann >= 1.0.0) fann_run — Will run input through the neural network 说明 array fann_run ( resource $ann , array $input ) Will run input through the neural network, returning an array of outputs, the number of which being equal to the number of neurons in the output layer. 参数 ann Neural network resource. input Array of input values 返回值 Array of output values, or FALSE on error
