HaruPage::rectangle (PECL haru >= 0.0.1) HaruPage::rectangle — Append a rectangle to the current path 说明 bool HaruPage::rectangle ( float $x , float $y , float $width , float $height ) Appends a rectangle to the current drawing path. 参数 x The left border of the rectangle. y The lower border of the rectangle. width The width of the rectangle. height The height of the rectangle. 返回值 Returns TRUE on success. 错误／异常 发生错误时抛出 HaruException 异常。
