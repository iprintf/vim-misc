cubrid_list_dbs (PECL CUBRID >= 8.3.0) cubrid_list_dbs — Return an array with the list of all existing CUBRID databases 说明 array cubrid_list_dbs ([ resource $conn_identifier ] ) This function returns an array with the list of all existing Cubrid databases. 参数 conn_identifier The CUBRID connection. 返回值 An numeric array with all existing Cubrid databases; on success. FALSE on failure. 范例 Example #1 cubrid_list_dbs() example 以上例程会输出： array(1) { [0]=> string(6) "demodb" }
