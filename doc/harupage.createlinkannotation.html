HaruPage::createLinkAnnotation (PECL haru >= 0.0.1) HaruPage::createLinkAnnotation — Create new HaruAnnotation instance 说明 object HaruPage::createLinkAnnotation ( array $rectangle , object $destination ) Creates a new HaruAnnotation instance. 参数 rectangle An array with 4 coordinates of the clickable area. destination Valid HaruDestination instance. 返回值 Returns a new HaruAnnotation instance. 错误／异常 发生错误时抛出 HaruException 异常。
