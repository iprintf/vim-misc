dba_nextkey (PHP 4, PHP 5, PHP 7) dba_nextkey — Fetch next key 说明 string dba_nextkey ( resource $handle ) dba_nextkey() returns the next key of the database and advances the internal key pointer. 参数 handle The database handler, returned by dba_open() or dba_popen(). 返回值 Returns the key on success 或者在失败时返回 FALSE. 参见 • dba_firstkey() - Fetch first key • dba_key_split() - Splits a key in string representation into array representation • Example 2 in the DBA examples
