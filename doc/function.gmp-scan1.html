gmp_scan1 (PHP 4 >= 4.0.4, PHP 5, PHP 7) gmp_scan1 — Scan for 1 说明 int gmp_scan1 ( GMP $a , int $start ) Scans a, starting with bit start, towards more significant bits, until the first set bit is found. 参数 a The number to scan. 可以是一个 GMP 数据 resouce，或一个可以转换为数值的字符串。 start The starting bit. 返回值 Returns the index of the found bit, as an integer. If no set bit is found, -1 is returned. 范例 Example #1 gmp_scan1() example 以上例程会输出： 3 9
