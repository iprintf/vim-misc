PDF_attach_file (PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0) PDF_attach_file — Add file attachment for current page [deprecated] 说明 bool PDF_attach_file ( resource $pdfdoc , float $llx , float $lly , float $urx , float $ury , string $filename , string $description , string $author , string $mimetype , string $icon ) Adds a file attachment annotation. 成功时返回 TRUE，或者在失败时返回 FALSE。 This function is deprecated since PDFlib version 6, use PDF_create_annotation() with type=FileAttachment instead. User Contributed Notes 1 note =============================================================================== Only the 'Full' Acrobat software will be able to display file attachments. All others will either show nothing or display a question mark.
