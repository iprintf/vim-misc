ReflectionProperty::isStatic (PHP 5, PHP 7) ReflectionProperty::isStatic — Checks if property is static 说明 public bool ReflectionProperty::isStatic ( void ) Checks whether the property is static. 参数 此函数没有参数。 返回值 TRUE if the property is static, FALSE otherwise. 参见 • ReflectionProperty::isPublic() - Checks if property is public • ReflectionProperty::isProtected() - Checks if property is protected • ReflectionProperty::isPrivate() - Checks if property is private
