MongoId::isValid (PECL mongo >= 1.5.0) MongoId::isValid — Check if a value is a valid ObjectId This extension that defines this method is deprecated. Instead, the MongoDB extension should be used. There is no equivalent for this method in the new extension. 说明 public static bool MongoId::isValid ( mixed $value ) This method may be used to check a variable before passing it as an argument to MongoId::__construct(). 参数 value The value to check for validity. 返回值 Returns TRUE if value is a MongoId instance or a string consisting of exactly 24 hexadecimal characters; otherwise, FALSE is returned.
