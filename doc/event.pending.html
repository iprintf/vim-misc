Event::pending (PECL event >= 1.2.6-beta) Event::pending — Detects whether event is pending or scheduled 说明 public bool Event::pending ( int $flags ) Detects whether event is pending or scheduled 参数 flags One of, or a composition of the following constants: Event::READ , Event::WRITE , Event::TIMEOUT , Event::SIGNAL . 返回值 Returns TRUE if event is pending or scheduled. Otherwise FALSE.
