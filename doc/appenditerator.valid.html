AppendIterator::valid (PHP 5 >= 5.1.0, PHP 7) AppendIterator::valid — Checks validity of the current element 说明 public bool AppendIterator::valid ( void ) Checks validity of the current element. 参数 此函数没有参数。 返回值 Returns TRUE if the current iteration is valid, FALSE otherwise. 参见 • AppendIterator::current() - Gets the current value • AppendIterator::key() - Gets the current key • AppendIterator::next() - Moves to the next element • AppendIterator::rewind() - Rewinds the Iterator
