fdf_next_field_name (PHP 4, PHP 5, PHP 7) fdf_next_field_name — Get the next field name 说明 string fdf_next_field_name ( resource $fdf_document [, string $fieldname ] ) Gets the name of the field after the given field. This name can be used with several functions. 参数 fdf_document The FDF document handle, returned by fdf_create(), fdf_open() or fdf_open_string(). fieldname Name of the FDF field, as a string. If not given, the first field will be assumed. 返回值 Returns the field name as a string. 范例 Example #1 Detecting all fieldnames in a FDF 参见 • fdf_get_value() - Get the value of a field
