pcntl_wstopsig (PHP 4 >= 4.1.0, PHP 5, PHP 7) pcntl_wstopsig — 返回导致子进程停止的信号 说明 int pcntl_wstopsig ( int $status ) 返回导致子进程停止的信号编号。这个函数仅在pcntl_wifstopped()返回 TRUE 时有效。 参数 status 参数 status 是提供给成功调用 pcntl_waitpid() 时的状态参数。 返回值 返回信号编号。 参见 • pcntl_waitpid() - 等待或返回fork的子进程状态 • pcntl_wifstopped() - 检查子进程当前是否已经停止
