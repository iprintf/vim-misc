fdf_set_target_frame (PHP 4 >= 4.3.0, PHP 5, PHP 7) fdf_set_target_frame — Set target frame for form display 说明 bool fdf_set_target_frame ( resource $fdf_document , string $frame_name ) Sets the target frame to display a result PDF defined with fdf_save_file() in. 参数 fdf_document The FDF document handle, returned by fdf_create(), fdf_open() or fdf_open_string(). frame_name The frame name, as a string. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • fdf_save_file()
