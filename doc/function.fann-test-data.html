fann_test_data (PECL fann >= 1.0.0) fann_test_data — Test a set of training data and calculates the MSE for the training data 说明 float fann_test_data ( resource $ann , resource $data ) Test a set of training data and calculates the MSE for the training data. This function updates the MSE and the bit fail values. 参数 ann Neural network resource. data Neural network training data resource. 返回值 The updated MSE, or FALSE on error. 参见 • fann_train_on_data() - Trains on an entire dataset for a period of time • fann_train_epoch() - Train one epoch with a set of training data • fann_get_bit_fail() - The number of fail bits • fann_get_MSE() - Reads the mean square error from the network • fann_set_training_algorithm() - Sets the training algorithm
