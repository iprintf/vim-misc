IntlChar::getUnicodeVersion (PHP 7) IntlChar::getUnicodeVersion — Get the Unicode version 说明 public static array IntlChar::getUnicodeVersion ( void ) Gets the Unicode version information. The version array is filled in with the version information for the Unicode standard that is currently used by ICU. For example, Unicode version 3.1.1 is represented as an array with the values [3, 1, 1, 0]. 参数 此函数没有参数。 返回值 An array containing the Unicode version number. 范例 Example #1 Testing different properties 以上例程会输出： array(4) { [0]=> int(7) [1]=> int(0) [2]=> int(0) [3]=> int(0) } 参见 • IntlChar::charAge() - Get the "age" of the code point
