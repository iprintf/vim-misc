Gmagick::deconstructimages (PECL gmagick >= Unknown) Gmagick::deconstructimages — Returns certain pixel differences between images 说明 public void Gmagick::deconstructimages ( void ) Compares each image with the next in a sequence and returns the maximum bounding region of any pixel differences it discovers. 参数 此函数没有参数。 返回值 Returns self on success. 错误／异常 错误时抛出 GmagickException。
