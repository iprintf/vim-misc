ArrayAccess::offsetSet (PHP 5 >= 5.0.0, PHP 7) ArrayAccess::offsetSet — 设置一个偏移位置的值 说明 abstract public void ArrayAccess::offsetSet ( mixed $offset , mixed $value ) 为指定的偏移位置设置一个值。 参数 offset 待设置的偏移位置。 value 需要设置的值。 返回值 没有返回值。 注释 Note: 如果另一个值不可用，那么 offset 参数将被设置为 NULL，就像下面的例子。 以上例程会输出： Array ( [0] => first value [1] => second value )
