MongoLog::getModule (PECL mongo >= 1.2.3) MongoLog::getModule — Gets the module(s) currently being logged 说明 public static int MongoLog::getModule ( void ) This function can be used to see which driver modules are currently being logged. The returned integer may be compared with the MongoLog module constants using bitwise operators to check if specific modules are being logged. 参数 此函数没有参数。 返回值 Returns the module(s) currently being logged.
