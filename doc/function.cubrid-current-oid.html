cubrid_current_oid (PECL CUBRID >= 8.3.0) cubrid_current_oid — Get OID of the current cursor location 说明 string cubrid_current_oid ( resource $req_identifier ) The cubrid_current_oid() function is used to get the oid of the current cursor location from the query result. To use cubrid_current_oid(), the query executed must be a updatable query, and the CUBRID_INCLUDE_OID option must be included during the query execution. 参数 req_identifier Request identifier. 返回值 Oid of current cursor location, when process is successful FALSE, when process is unsuccessful. 范例 Example #1 cubrid_current_oid() example 以上例程会输出： Array ( [s_name] => X [f_name] => Mixed ) 参见 • cubrid_execute() - Execute a prepared SQL statement
