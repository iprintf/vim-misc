shmop_size (PHP 4 >= 4.0.4, PHP 5, PHP 7) shmop_size — Get size of shared memory block 说明 int shmop_size ( int $shmid ) shmop_size() is used to get the size, in bytes of the shared memory block. 参数 shmid The shared memory block identifier created by shmop_open() 返回值 Returns an int, which represents the number of bytes the shared memory block occupies. 范例 Example #1 Getting the size of the shared memory block This example will put the size of shared memory block identified by $shm_id into $shm_size.
