ImagickDraw::setFillPatternURL (PECL imagick 2.0.0) ImagickDraw::setFillPatternURL — Sets the URL to use as a fill pattern for filling objects 说明 bool ImagickDraw::setFillPatternURL ( string $fill_url ) Warning 本函数还未编写文档，仅有参数列表。 Sets the URL to use as a fill pattern for filling objects. Only local URLs ("# identifier") are supported at this time. These local URLs are normally created by defining a named fill pattern with DrawPushPattern/DrawPopPattern. 参数 fill_url URL to use to obtain fill pattern. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
