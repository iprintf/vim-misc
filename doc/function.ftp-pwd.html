ftp_pwd (PHP 4, PHP 5, PHP 7) ftp_pwd — 返回当前目录名 说明 string ftp_pwd ( resource $ftp_stream ) 参数 ftp_stream FTP 连接资源。 返回值 返回当前目录名称，发生错误则返回 FALSE。 范例 Example #1 ftp_pwd() 例子 参见 • ftp_chdir() - 在 FTP 服务器上改变当前目录 • ftp_cdup() - 切换到当前目录的父目录 User Contributed Notes 1 note =============================================================================== This function doesn't always go to the remote server for the PWD. Once called the PWD is cached, and until PHP has a reason to believe the directory has changed any call to ftp_pwd() will return from the cache, even if the remote server has gone away.
