getmyinode (PHP 4, PHP 5, PHP 7) getmyinode — 获取当前脚本的索引节点（inode） 说明 int getmyinode ( void ) 获取当前脚本的索引节点（inode）。 返回值 以整型返回当前脚本的索引节点（inode），或在错误时返回 FALSE。 参见 • getmygid() - 获取当前 PHP 脚本拥有者的 GID • getmyuid() - 获取 PHP 脚本所有者的 UID • getmypid() - 获取 PHP 进程的 ID • get_current_user() - 获取当前 PHP 脚本所有者名称 • getlastmod() - 获取页面最后修改的时间
