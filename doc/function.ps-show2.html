ps_show2 (PECL ps >= 1.1.0) ps_show2 — Output a text at current position 说明 bool ps_show2 ( resource $psdoc , string $text , int $len ) Output text at the current position. Do not print more than len characters. 参数 psdoc Resource identifier of the postscript file as returned by ps_new(). text The text to be output. len The maximum number of characters to print. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
