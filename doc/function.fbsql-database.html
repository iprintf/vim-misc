fbsql_database (PHP 4 >= 4.0.6, PHP 5, PHP 7) fbsql_database — Get or set the database name used with a connection 说明 string fbsql_database ( resource $link_identifier [, string $database ] ) Get or set the database name used with the connection. 参数 link_identifier 由 fbsql_connect() 或 fbsql_pconnect() 返回的 FrontBase 链接标识符. 如可选并未指定，函数会尝试查找一个到 FrontBase 服务器的开放链接，且如没有找 到这样的链接，会尝试通过不带参数调用 fbsql_connect() 来创建一个。 database The database name. If given, the default database of the connexion will be changed to database. 返回值 Returns the name of the database used with this connection.
