cubrid_lob2_close (PECL CUBRID >= 8.4.1) cubrid_lob2_close — Close LOB object. 说明 bool cubrid_lob2_close ( resource $lob_identifier ) The cubrid_lob2_close() function is used to close LOB object returned from cubrid_lob2_new() or got from the result set. 参数 lob_identifier Lob identifier as a result of cubrid_lob2_new() or get from the result set. 返回值 TRUE, on success. FALSE, on failure. 参见 • cubrid_lob2_new() - Create a lob object.
