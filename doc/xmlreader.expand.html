XMLReader::expand (PHP 5 >= 5.1.0, PHP 7) XMLReader::expand — Returns a copy of the current node as a DOM object 说明 public DOMNode XMLReader::expand ([ DOMNode $basenode ] ) This method copies the current node and returns the appropriate DOM object. 参数 basenode A DOMNode defining the target DOMDocument for the created DOM object. 返回值 The resulting DOMNode or FALSE on error. 更新日志 版本 说明 5.3.0 The parameter basenode was added.
