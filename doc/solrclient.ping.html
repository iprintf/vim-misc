SolrClient::ping (PECL solr >= 0.9.2) SolrClient::ping — Checks if Solr server is still up 说明 public SolrPingResponse SolrClient::ping ( void ) Checks if the Solr server is still alive. Sends a HEAD request to the Apache Solr server. 参数 此函数没有参数。 返回值 Returns a SolrPingResponse object on success and throws an exception on failure. 错误／异常 Throws SolrClientException if the client had failed, or there was a connection issue.
