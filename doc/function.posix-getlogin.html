posix_getlogin (PHP 4, PHP 5, PHP 7) posix_getlogin — Return login name 说明 string posix_getlogin ( void ) Returns the login name of the user owning the current process. 返回值 Returns the login name of the user, as a string. 范例 Example #1 Example use of posix_getlogin() 参见 • posix_getpwnam() - Return info about a user by username • POSIX man page GETLOGIN(3) User Contributed Notes 2 notes =============================================================================== posix_getlogin() will return boolean false when there is no shell, e.g. when running under an apache process =============================================================================== Note that this function returns the owner that the process *started* as. Even if you call posix_setuid (or posix_seteuid) it will still report the username the process started as. use get_current_user() to get the effective real user.
