HaruImage::getColorSpace (PECL haru >= 0.0.1) HaruImage::getColorSpace — Get the name of the color space 说明 string HaruImage::getColorSpace ( void ) Get the name of the color space. 参数 此函数没有参数。 返回值 Returns the name of the color space of the image. The name is one of the following values: • "DeviceGray" • "DeviceRGB" • "DeviceCMYK" • "Indexed" 错误／异常 发生错误时抛出 HaruException 异常。
