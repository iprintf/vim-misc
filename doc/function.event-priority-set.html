event_priority_set (PECL libevent >= 0.0.5) event_priority_set — Assign a priority to an event. 说明 bool event_priority_set ( resource $event , int $priority ) Assign a priority to the event. 参数 event Valid event resource. priority Priority level. Cannot be less than zero and cannot exceed maximum priority level of the event base (see event_base_priority_init()). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
