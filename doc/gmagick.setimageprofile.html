Gmagick::setimageprofile (PECL gmagick >= Unknown) Gmagick::setimageprofile — Adds a named profile to the Gmagick object 说明 public Gmagick Gmagick::setimageprofile ( string $name , string $profile ) Adds a named profile to the Gmagick object. If a profile with the same name already exists, it is replaced. This method differs from the Gmagick::ProfileImage() method in that it does not apply any CMS color profiles. 参数 name Name of profile to add or remove: ICC, IPTC, or generic profile. profile The profile. 返回值 The Gmagick object on success. 错误／异常 错误时抛出 GmagickException。
