SolrQuery::setEchoParams (PECL solr >= 0.9.2) SolrQuery::setEchoParams — Determines what kind of parameters to include in the response 说明 public SolrQuery SolrQuery::setEchoParams ( string $type ) Instructs Solr what kinds of Request parameters should be included in the response for debugging purposes, legal values include: - none - don't include any request parameters for debugging - explicit - include the parameters explicitly specified by the client in the request - all - include all parameters involved in this request, either specified explicitly by the client, or implicit because of the request handler configuration. 参数 type The type of parameters to include 返回值 Returns the current SolrQuery object, if the return value is used.
