gzgetc (PHP 4, PHP 5, PHP 7) gzgetc — Get character from gz-file pointer 说明 string gzgetc ( resource $zp ) Returns a string containing a single (uncompressed) character read from the given gz-file pointer. 参数 zp The gz-file pointer. It must be valid, and must point to a file successfully opened by gzopen(). 返回值 The uncompressed character or FALSE on EOF (unlike gzeof()). 范例 Example #1 gzgetc() example 参见 • gzopen() - Open gz-file • gzgets() - Get line from file pointer
