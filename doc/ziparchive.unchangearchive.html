ZipArchive::unchangeArchive (PHP 5 >= 5.2.0, PHP 7, PECL zip >= 1.1.0) ZipArchive::unchangeArchive — Revert all global changes done in the archive. 说明 bool ZipArchive::unchangeArchive ( void ) Revert all global changes to the archive. For now, this only reverts archive comment changes. 参数 此函数没有参数。 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
