mysqli::init mysqli_init (PHP 5, PHP 7) mysqli::init -- mysqli_init — Initializes MySQLi and returns a resource for use with mysqli_real_connect() 说明 面向对象风格 mysqli mysqli::init ( void ) 过程化风格 mysqli mysqli_init ( void ) Allocates or initializes a MYSQL object suitable for mysqli_options() and mysqli_real_connect(). Note: Any subsequent calls to any mysqli function (except mysqli_options()) will fail until mysqli_real_connect() was called. 返回值 Returns an object. 范例 See mysqli_real_connect(). 参见 • mysqli_options() - Set options • mysqli_close() - 关闭先前打开的数据库连接 • mysqli_real_connect() - 建立一个 MySQL 服务器连接 • mysqli_connect() - 别名 mysqli::__construct
