cubrid_lob_send (PECL CUBRID >= 8.3.1) cubrid_lob_send — Read BLOB/CLOB data and send straight to browser 说明 bool cubrid_lob_send ( resource $conn_identifier , resource $lob_identifier ) cubrid_lob_send() reads BLOB/CLOB data and passes it straight through to the browser. To use this function, you must use cubrid_lob_get() first to get BLOB/ CLOB info from CUBRID. 参数 conn_identifier Connection identifier. lob_identifier LOB identifier. 返回值 TRUE, when process is successful. FALSE, when process is unsuccessful. 范例 Example #1 cubrid_lob_send() example 参见 • cubrid_lob_get() - Get BLOB/CLOB data • cubrid_lob_close() - Close BLOB/CLOB data • cubrid_lob_size() - Get BLOB/CLOB data size • cubrid_lob_export() - Export BLOB/CLOB data to file
