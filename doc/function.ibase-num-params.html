ibase_num_params (PHP 5, PHP 7) ibase_num_params — Return the number of parameters in a prepared query 说明 int ibase_num_params ( resource $query ) This function returns the number of parameters in the prepared query specified by query. This is the number of binding arguments that must be present when calling ibase_execute(). 参数 query The prepared query handle. 返回值 Returns the number of parameters as an integer. 参见 • ibase_prepare() - Prepare a query for later binding of parameter placeholders and execution • ibase_param_info() - Return information about a parameter in a prepared query
