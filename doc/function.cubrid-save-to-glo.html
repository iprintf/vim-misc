cubrid_save_to_glo (PECL CUBRID >= 8.3.0) cubrid_save_to_glo — Save requested file in a GLO instance 说明 int cubrid_save_to_glo ( resource $conn_identifier , string $oid , string $file_name ) The cubrid_save_to_glo() function is used to save requested file in a glo instance. 参数 conn_identifier Connection identifier. oid Oid of the glo instance that you want to save a file in. file_name The name of the file that you want to save. 返回值 TRUE, when process is successful. FALSE, when process is unsuccessful. 范例 Example #1 cubrid_save_to_glo() example 注释 Note: 为了向下兼容，可以使用下列已废弃的别名： cubrid_save_to_glo() Note: This function is removed from CUBRID 3.1. 参见 • cubrid_new_glo() - Create a glo instance • cubrid_load_from_glo() - Read data from a GLO instance and save it in a file • cubrid_send_glo() - Read data from glo and send it to std output
