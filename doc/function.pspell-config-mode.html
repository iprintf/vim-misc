pspell_config_mode (PHP 4 >= 4.0.2, PHP 5, PHP 7) pspell_config_mode — Change the mode number of suggestions returned 说明 bool pspell_config_mode ( int $dictionary_link , int $mode ) pspell_config_mode() should be used on a config before calling pspell_new_config(). This function determines how many suggestions will be returned by pspell_suggest(). 参数 dictionary_link mode The mode parameter is the mode in which spellchecker will work. There are several modes available: □ PSPELL_FAST - Fast mode (least number of suggestions) □ PSPELL_NORMAL - Normal mode (more suggestions) □ PSPELL_BAD_SPELLERS - Slow mode (a lot of suggestions) 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 pspell_config_mode() Example
