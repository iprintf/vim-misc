mb_regex_encoding (PHP 4 >= 4.2.0, PHP 5, PHP 7) mb_regex_encoding — Set/Get character encoding for multibyte regex 说明 mixed mb_regex_encoding ([ string $encoding = mb_regex_encoding() ] ) Set/Get character encoding for a multibyte regex. 参数 encoding encoding 参数为字符编码。如果省略，则使用内部字符编码。 返回值 If encoding is set, then 成功时返回 TRUE，或者在失败时返回 FALSE。 In this case, the internal character encoding is NOT changed. If encoding is omitted, then the current character encoding name for a multibyte regex is returned. 更新日志 版本 说明 5.6.0 Default encoding is changed to UTF-8.
