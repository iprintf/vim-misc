stream_wrapper_restore (PHP 5 >= 5.1.0, PHP 7) stream_wrapper_restore — Restores a previously unregistered built-in wrapper 说明 bool stream_wrapper_restore ( string $protocol ) Restores a built-in wrapper previously unregistered with stream_wrapper_unregister(). 参数 protocol 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
