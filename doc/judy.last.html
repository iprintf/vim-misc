Judy::last (PECL judy >= 0.1.1) Judy::last — Search for the last index in the Judy array 说明 public void Judy::last ([ string $index ] ) Search (inclusive) for the last index present that is equal to or less than the passed Index. 参数 index The index can be an integer or a string corresponding to the index where to start the search. 返回值 Return the corresponding index in the array.
