bindtextdomain (PHP 4, PHP 5, PHP 7) bindtextdomain — Sets the path for a domain 说明 string bindtextdomain ( string $domain , string $directory ) The bindtextdomain() function sets the path for a domain. 参数 domain The domain directory The directory path 返回值 The full pathname for the domain currently being set. 范例 Example #1 bindtextdomain() example 以上例程会输出： /usr/share/myapp/locale User Contributed Notes 2 notes =============================================================================== The name of your .mo file must match the $domain, e.g. name your files messages.mo and call bindtextdomain("messages", $directory). =============================================================================== I recommend using absolute paths in the $directory parameter. This caused me several hours to debug as Ajax calls to my localization functions messed up the path.
