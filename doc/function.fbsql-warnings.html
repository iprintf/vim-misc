fbsql_warnings (PHP 4 >= 4.0.6, PHP 5, PHP 7) fbsql_warnings — Enable or disable FrontBase warnings 说明 bool fbsql_warnings ([ bool $OnOff ] ) Enables or disables FrontBase warnings. 参数 OnOff Whether to enable warnings or no. 返回值 Returns TRUE if warnings is turned on, FALSE otherwise. 参见 • fbsql_errno() - Returns the error number from previous operation • fbsql_error() - Returns the error message from previous operation
