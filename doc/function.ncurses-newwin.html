ncurses_newwin (PHP 4 >= 4.0.7, PHP 5 < 5.3.0, PECL ncurses >= 1.0.0) ncurses_newwin — Create a new window 说明 resource ncurses_newwin ( int $rows , int $cols , int $y , int $x ) Warning 此函数是实验性的。此函数的表象，包括名称及其相关文档都可能在未来的 PHP 发布版本 中未通知就被修改。使用本函数风险自担。 Creates a new window to draw elements in. When creating additional windows, remember to use ncurses_getmaxyx() to check for available space, as terminal size is individual and may vary. 参数 rows Number of rows cols Number of columns y y-coordinate of the origin x x-coordinate of the origin 返回值 Returns a resource ID for the new window.
