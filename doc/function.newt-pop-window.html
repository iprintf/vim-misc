newt_pop_window (PECL newt >= 0.1) newt_pop_window — Removes the top window from the display 说明 void newt_pop_window ( void ) Removes the top window from the display, and redraws the display areas which the window overwrote. 返回值 没有返回值。 参见 • newt_open_window() - Open a window of the specified size and position • newt_centered_window() - Open a centered window of the specified size
