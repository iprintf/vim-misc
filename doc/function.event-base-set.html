event_base_set (PECL libevent >= 0.0.1) event_base_set — Associate event base with an event 说明 bool event_base_set ( resource $event , resource $event_base ) Associates the event_base with the event. 参数 event Valid event resource. event_base Valid event base resource. 返回值 event_base_set() returns TRUE on success or FALSE on error.
