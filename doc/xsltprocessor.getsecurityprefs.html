XsltProcessor::getSecurityPrefs (PHP >= 5.4.0) XsltProcessor::getSecurityPrefs — Get security preferences 说明 public int XsltProcessor::getSecurityPrefs ( void ) Gets the security preferences. 参数 此函数没有参数。 返回值 A bitmask consisting of XSL_SECPREF_READ_FILE, XSL_SECPREF_WRITE_FILE, XSL_SECPREF_CREATE_DIRECTORY, XSL_SECPREF_READ_NETWORK, XSL_SECPREF_WRITE_NETWORK.
