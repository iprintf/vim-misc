=============================================================================== It is not correct that the same information can be obtained from the server. For example, the server will not give you information about client side memory usage. =============================================================================== While this function is only available if you're using the PHP MySQL Native Driver, you can get the same statistics without this function by using the MySQL SHOW STATUS Query. Use 'SHOW SESSION STATUS;' for statistics about the current MySQL session. Use 'SHOW GLOBAL STATUS;' for totals since the server was started.
