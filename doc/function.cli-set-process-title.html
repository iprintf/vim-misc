cli_set_process_title (PHP 5 >= 5.5.0, PHP 7) cli_set_process_title — Sets the process title 说明 bool cli_set_process_title ( string $title ) Sets the process title visible in tools such as top and ps. This function is available only in CLI mode. 参数 title The new title. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 错误／异常 An E_WARNING will be generated if the operating system is unsupported. 范例 Example #1 cli_set_process_title() example 参见 • cli_get_process_title() - Returns the current process title • setproctitle() - Set the process title User Contributed Notes 2 notes =============================================================================== Setting proc title on PHP based daemons is pretty sweet. =============================================================================== Beware that if you kill the process via console the original command line will appear.
