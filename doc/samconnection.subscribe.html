SAMConnection::subscribe (PECL sam >= 0.1.0) SAMConnection::subscribe — Create a subscription to a specified topic. 说明 string SAMConnection::subscribe ( string $targetTopic ) The "subscribe" method is used to create a new subscription to a specified topic. 参数 targetTopic The identity of the topic (topic://topicname) to subscribe to. 返回值 A subscription identifier that can be used in a subsequent receive call as a selector to obtain any topic data or FALSE if an error occurred.
