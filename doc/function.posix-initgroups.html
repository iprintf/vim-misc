posix_initgroups (PHP 5 >= 5.2.0, PHP 7) posix_initgroups — Calculate the group access list 说明 bool posix_initgroups ( string $name , int $base_group_id ) Calculates the group access list for the user specified in name. 参数 name The user to calculate the list for. base_group_id Typically the group number from the password file. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • The Unix manual page for initgroups(3). User Contributed Notes 1 note =============================================================================== The function does not calculate groups, it sets groups access list of the current process in accordance to user information in /etc/groups. Quote from Unix manual page for initgroups: NAME initgroups - initialize the supplementary group access list ... DESCRIPTION The initgroups() function initializes the group access list by reading the group database /etc/group and using all groups of which user is a member. The additional group group is also added to the list.
