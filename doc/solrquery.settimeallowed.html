SolrQuery::setTimeAllowed (PECL solr >= 0.9.2) SolrQuery::setTimeAllowed — The time allowed for search to finish 说明 public SolrQuery SolrQuery::setTimeAllowed ( int $timeAllowed ) The time allowed for a search to finish. This value only applies to the search and not to requests in general. Time is in milliseconds. Values less than or equal to zero implies no time restriction. Partial results may be returned, if there are any. 参数 timeAllowed The time allowed for a search to finish. 返回值 Returns the current SolrQuery object, if the return value is used.
