fbsql_set_password (PHP 5, PHP 7) fbsql_set_password — Change the password for a given user 说明 bool fbsql_set_password ( resource $link_identifier , string $user , string $password , string $old_password ) Changes the password for the given user. 参数 link_identifier 由 fbsql_connect() 或 fbsql_pconnect() 返回的 FrontBase 链接标识符. 如可选并未指定，函数会尝试查找一个到 FrontBase 服务器的开放链接，且如没有找 到这样的链接，会尝试通过不带参数调用 fbsql_connect() 来创建一个。 user The user name. password The new password to be set. old_password The old password to be replaced. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
