Imagick::setImageDispose (PECL imagick 2.0.0) Imagick::setImageDispose — Sets the image disposal method 说明 bool Imagick::setImageDispose ( int $dispose ) Sets the image disposal method. 参数 dispose 返回值 成功时返回 TRUE。 错误／异常 错误时抛出 ImagickException。 User Contributed Notes 1 note =============================================================================== The layer disposal method defines the way each the displayed image is to be modified after the current 'frame' of an animation has finished being displayed (after its 'delay' period), but before the next frame on an animation is to be overlaid onto the display. Value for $dispose: Undefined: 0 -> No disposal specified (equivalent to 'none'). None: 1 -> Do not dispose, just overlay next frame image. Background: 2 -> Clear the frame area with the background color. Previous: 3 -> Clear to the image prior to this frames overlay. from: http://www.imagemagick.org/script/command-line-options.php?#dispose
