ifxus_seek_slob (PHP 4, PHP <=5.2.0) ifxus_seek_slob — Sets the current file or seek position 说明 int ifxus_seek_slob ( int $bid , int $mode , int $offset ) Sets the current file or seek position of an open slob object. 参数 bid An existing slob id. mode 0 = LO_SEEK_SET, 1 = LO_SEEK_CUR, 2 = LO_SEEK_END. offset A byte offset. 返回值 Returns the seek position as an integer, or FALSE on errors. 参见 • ifxus_tell_slob() - Returns the current file or seek position
