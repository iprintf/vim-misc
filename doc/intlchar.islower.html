IntlChar::islower (PHP 7) IntlChar::islower — Check if code point is a lowercase letter 说明 public static bool IntlChar::islower ( mixed $codepoint ) Determines whether the specified code point has the general category "Ll" (lowercase letter). Note: This misses some characters that are also lowercase but have a different general category value. In order to include those, use IntlChar::isULowercase(). 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 Returns TRUE if codepoint is an Ll lowercase letter, FALSE if not. 范例 Example #1 Testing different code points 以上例程会输出： bool(false) bool(true) bool(false) bool(true) bool(false) 参见 • IntlChar::isupper() - Check if code point has the general category "Lu" (uppercase letter) • IntlChar::istitle() - Check if code point is a titlecase letter • IntlChar::tolower() - Make Unicode character lowercase • IntlChar::toupper() - Make Unicode character uppercase • IntlChar::PROPERTY_LOWERCASE
