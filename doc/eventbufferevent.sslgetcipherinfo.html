EventBufferEvent::sslGetCipherInfo (PECL event >= 1.10.0) EventBufferEvent::sslGetCipherInfo — Returns a textual description of the cipher. 说明 public string EventBufferEvent::sslGetCipherInfo ( void ) Retrieves description of the current cipher by means of the SSL_CIPHER_description SSL API function (see SSL_CIPHER_get_name(3) man page). Note: This function is available only if Event is compiled with OpenSSL support. 参数 此函数没有参数。 返回值 Returns a textual description of the cipher on success, or FALSE on error.
