Event::timer (PECL event >= 1.2.6-beta) Event::timer — Constructs timer event object 说明 public static Event Event::timer ( EventBase $base , callable $cb [, mixed $arg ] ) Constructs timer event object. This is a straightforward method to create a timer event. Note, the generic Event::__construct() method can contruct signal event objects too. 参数 base The associated event base object. cb The signal event callback. See Event callbacks . arg Custom data. If specified, it will be passed to the callback when event triggers. 返回值 Returns Event object on success. Otherwise FALSE.
