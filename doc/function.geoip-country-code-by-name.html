geoip_country_code_by_name (PECL geoip >= 0.2.0) geoip_country_code_by_name — Get the two letter country code 说明 string geoip_country_code_by_name ( string $hostname ) The geoip_country_code_by_name() function will return the two letter country code corresponding to a hostname or an IP address. 参数 hostname The hostname or IP address whose location is to be looked-up. 返回值 Returns the two letter ISO country code on success, or FALSE if the address cannot be found in the database. 范例 Example #1 A geoip_country_code_by_name() example This will print where the host example.com is located. 以上例程会输出： This host is located in: US 注释 Caution Please see » http://www.maxmind.com/en/iso3166 for a complete list of possible return values, including special codes. 参见 • geoip_country_code3_by_name() - Get the three letter country code • geoip_country_name_by_name() - Get the full country name
