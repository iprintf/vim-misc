XMLReader::close (PHP 5 >= 5.1.0, PHP 7) XMLReader::close — Close the XMLReader input 说明 public bool XMLReader::close ( void ) Closes the input the XMLReader object is currently parsing. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • XMLReader::open() - Set the URI containing the XML to parse • XMLReader::xml() - Set the data containing the XML to parse
