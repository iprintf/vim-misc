cubrid_num_cols (PECL CUBRID >= 8.3.0) cubrid_num_cols — Return the number of columns in the result set 说明 int cubrid_num_cols ( resource $result ) The cubrid_num_cols() function is used to get the number of columns from the query result. It can only be used when the query executed is a select statement. 参数 result Result. 返回值 Number of columns, when process is successful. FALSE, if SQL statement is not SELECT. 范例 Example #1 cubrid_num_cols() example 以上例程会输出： Row Num: 6 Column Num: 2 参见 • cubrid_execute() - Execute a prepared SQL statement • cubrid_num_rows() - Get the number of rows in the result set
