fbsql_get_autostart_info (PHP 4 >= 4.0.7, PHP 5, PHP 7) fbsql_get_autostart_info — 说明 array fbsql_get_autostart_info ([ resource $link_identifier ] ) Warning 本函数还未编写文档，仅有参数列表。 参数 link_identifier 由 fbsql_connect() 或 fbsql_pconnect() 返回的 FrontBase 链接标识符. 如可选并未指定，函数会尝试查找一个到 FrontBase 服务器的开放链接，且如没有找 到这样的链接，会尝试通过不带参数调用 fbsql_connect() 来创建一个。
