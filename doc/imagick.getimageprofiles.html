Imagick::getImageProfiles (PECL imagick 2.0.0) Imagick::getImageProfiles — Returns the image profiles 说明 array Imagick::getImageProfiles ([ string $pattern = "*" [, bool $only_names = true ]] ) Returns all associated profiles that match the pattern. If TRUE is passed as second parameter only the profile names are returned. 此方法在Imagick基于 ImageMagick 6.3.6以上版本编译时可用。 参数 pattern The pattern for profile names. only_names Whether to return only profile names. If FALSE then values are returned as well 返回值 Returns an array containing the image profiles or profile names.
