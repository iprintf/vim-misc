RRDGraph::save (PECL rrd >= 0.9.0) RRDGraph::save — Saves the result of query into image 说明 public array RRDGraph::save ( void ) Saves the result of RRD database query into image defined by RRDGraph::__construct(). 参数 此函数没有参数。 返回值 Array with information about generated image is returned, FALSE if error occurs.
