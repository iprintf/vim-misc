px_set_tablename (PECL paradox >= 1.0.0) px_set_tablename — Sets the name of a table (deprecated) 说明 void px_set_tablename ( resource $pxdoc , string $name ) Sets the table name of a paradox database, which was created with px_create_fp (). This function is deprecated use px_set_parameter() instead. 参数 pxdoc Resource identifier of the paradox database as returned by px_new(). tablename The name of the table. If it is not set explicitly it will be set to the name of the database file. 返回值 Returns NULL on success 或者在失败时返回 FALSE. 参见 px_set_parameter() - Sets a parameter
