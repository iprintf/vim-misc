maxdb_more_results maxdb::more_results (PECL maxdb >= 1.0) maxdb_more_results -- maxdb::more_results — Check if there any more query results from a multi query 说明 bool maxdb_more_results ( resource $link ) maxdb_more_results() indicates if one or more result sets are available from a previous call to maxdb_multi_query(). 返回值 Always FALSE. 范例 See maxdb_multi_query(). 参见 • maxdb_multi_query() - Performs a query on the database • maxdb_next_result() - Prepare next result from multi_query • maxdb_store_result() - Transfers a result set from the last query • maxdb_use_result() - Initiate a result set retrieval
