Imagick::setImageArtifact (No version information available, might only be in Git) Imagick::setImageArtifact — Set image artifact 说明 bool Imagick::setImageArtifact ( string $artifact , string $value ) Associates an artifact with the image. The difference between image properties and image artifacts is that properties are public and artifacts are private. 此 方法在Imagick基于ImageMagick 6.5.7以上版本编译时可用。 参数 artifact The name of the artifact value The value of the artifact 返回值 成功时返回 TRUE。 错误／异常 错误时抛出 ImagickException。 参见 • Imagick::getImageArtifact() - Get image artifact • Imagick::deleteImageArtifact() - Delete image artifact 范例 Example #1 Imagick::setImageArtifact() setImageVirtualPixelMethod(\Imagick::VIRTUALPIXELMETHOD_TRANSPARENT ); $src2->setImageArtifact('compose:args', "1,0,-0.5,0.5"); $src1->compositeImage($src2, Imagick::COMPOSITE_MATHEMATICS, 0, 0); $src1->setImageFormat('png'); header("Content-Type: image/png"); echo $src1->getImagesBlob(); } ?>
