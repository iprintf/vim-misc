XMLReader::setParserProperty (PHP 5 >= 5.1.0, PHP 7) XMLReader::setParserProperty — Set parser options 说明 public bool XMLReader::setParserProperty ( int $property , bool $value ) Set parser options. The options must be set after XMLReader::open() or XMLReader::xml() are called and before the first XMLReader::read() call. 参数 property One of the parser option constants. value If set to TRUE the option will be enabled otherwise will be disabled. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
