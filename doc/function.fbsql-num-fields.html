fbsql_num_fields (PHP 4 >= 4.0.6, PHP 5, PHP 7) fbsql_num_fields — Get number of fields in result 说明 int fbsql_num_fields ( resource $result ) Returns the number of fields in the given result set. 参数 result 由 fbsql_query() 或 fbsql_db_query() 返回的结果标识符。 返回值 Returns the number of fields, as an integer. 参见 • fbsql_db_query() - Send a FrontBase query • fbsql_query() - Send a FrontBase query • fbsql_fetch_field() - Get column information from a result and return as an object • fbsql_num_rows() - Get number of rows in result
