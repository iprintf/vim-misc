ifx_num_fields (PHP 4, PHP <=5.2.0) ifx_num_fields — Returns the number of columns in the query 说明 int ifx_num_fields ( resource $result_id ) After preparing or executing a query, this call gives you the number of columns in the query. 参数 result_id result_id is a valid resultid returned by ifx_query() or ifx_prepare() (select type queries only!). 返回值 Returns the number of columns in query for result_id, or FALSE on errors. 范例 Example #1 ifx_num_fields() Example 参见 • ifx_num_rows() - Count the rows already fetched from a query
