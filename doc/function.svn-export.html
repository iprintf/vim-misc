svn_export (PECL svn >= 0.3.0) svn_export — Export the contents of a SVN directory 说明 bool svn_export ( string $frompath , string $topath [, bool $working_copy = true [, int $revision_no = -1 ]] ) Export the contents of either a working copy or repository into a 'clean' directory. 参数 frompath The path to the current repository. topath The path to the new repository. working_copy If TRUE, it will export uncommitted files from the working copy. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 svn_export() example 参见 • svn_import() - Imports an unversioned path into a repository User Contributed Notes 2 notes =============================================================================== This functionality is very different from the actual command line svn export url [-rRevision] If you are thinking about checkout out a single file at a particular version, keep looking because this export can not perform that functionality. =============================================================================== To checkout a single file at a particular version use svn_cat .
