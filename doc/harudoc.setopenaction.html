HaruDoc::setOpenAction (PECL haru >= 0.0.1) HaruDoc::setOpenAction — Define which page is shown when the document is opened 说明 bool HaruDoc::setOpenAction ( object $destination ) Defines which page should be shown when the document is opened. 参数 destination A valid HaruDestination instance. 返回值 Returns TRUE on success. 错误／异常 发生错误时抛出 HaruException 异常。
