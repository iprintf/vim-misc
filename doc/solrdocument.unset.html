SolrDocument::__unset (PECL solr >= 0.9.2) SolrDocument::__unset — Removes a field from the document 说明 public bool SolrDocument::__unset ( string $fieldName ) Removes a field from the document when the field is access as an object property. 参数 fieldName The name of the field. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
