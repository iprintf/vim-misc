fdf_set_status (PHP 4, PHP 5, PHP 7) fdf_set_status — Set the value of the /STATUS key 说明 bool fdf_set_status ( resource $fdf_document , string $status ) Sets the value of the /STATUS key. When a client receives a FDF with a status set it will present the value in an alert box. 参数 fdf_document The FDF document handle, returned by fdf_create(), fdf_open() or fdf_open_string(). status 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • fdf_get_status() - Get the value of the /STATUS key
