cubrid_get_charset (PECL CUBRID >= 8.3.0) cubrid_get_charset — Return the current CUBRID connection charset 说明 string cubrid_get_charset ( resource $conn_identifier ) This function returns the current CUBRID connection charset and is similar to the CUBRID MySQL compatible function cubrid_client_encoding(). 参数 conn_identifier The CUBRID connection. 返回值 A string that represents the CUBRID connection charset; on success. FALSE on failure. 范例 Example #1 cubrid_get_charset() example 以上例程会输出： CUBRID current charset: iso8859-1 参见 • cubrid_client_encoding() - Return the current CUBRID connection charset
