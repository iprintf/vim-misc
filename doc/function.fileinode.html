fileinode (PHP 4, PHP 5, PHP 7) fileinode — 取得文件的 inode 说明 int fileinode ( string $filename ) 取得文件的 inode。 参数 filename 文件的路径。 返回值 返回文件的 inode 节点号，或者在失败时返回 FALSE。 范例 Example #1 将某个文件和当前文件的 inode 进行对比 错误／异常 失败时抛出E_WARNING警告。 注释 Note: 此函数的结果会被缓存。参见 clearstatcache() 以获得更多细节。 Tip 自 PHP 5.0.0 起, 此函数也用于某些 URL 包装器。请参见支持的协议和封装协议以获得 支持 stat() 系列函数功能的包装器列表。 参见 • getmyinode() - 获取当前脚本的索引节点（inode） • stat() - 给出文件的信息 User Contributed Notes 1 note =============================================================================== This function, in spite of its name, also works for directories. Most any valid path in the filesystem will generate an inode value.
