ibase_affected_rows (PHP 5, PHP 7) ibase_affected_rows — Return the number of rows that were affected by the previous query 说明 int ibase_affected_rows ([ resource $link_identifier ] ) This function returns the number of rows that were affected by the previous query (INSERT, UPDATE or DELETE) that was executed from within the specified transaction context. 参数 link_identifier A transaction context. If link_identifier is a connection resource, its default transaction is used. 返回值 Returns the number of rows as an integer. 参见 • ibase_query() - Execute a query on an InterBase database • ibase_execute() - Execute a previously prepared query
