The DateTimeInterface interface (PHP 5 >= 5.5.0, PHP 7) 简介 DateTimeInterface is meant so that both DateTime and DateTimeImmutable can be type hinted for. It is not possible to implement this interface with userland classes. 类摘要 DateTimeInterface { /* 方法 */ public DateInterval diff ( DateTimeInterface $datetime2 [, bool $absolute = false ] ) public string format ( string $format ) public int getOffset ( void ) public int getTimestamp ( void ) public DateTimeZone getTimezone ( void ) public __wakeup ( void ) } 更新日志 版本 说明 Trying to implement DateTimeInterface raises a fatal error now. Formerly 5.5.8 implementing the interface didn't raise an error, but the behavior was erroneous.
