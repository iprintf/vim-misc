ifx_close (PHP 4, PHP <=5.2.0) ifx_close — Close Informix connection 说明 bool ifx_close ([ resource $link_identifier ] ) ifx_close() closes the link to an Informix database that's associated with the specified link identifier. Note that this isn't usually necessary, as non-persistent open links are automatically closed at the end of the script's execution. ifx_close() will not close persistent links generated by ifx_pconnect(). 参数 link_identifier The link identifier. If not specified, the last opened link is assumed. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 Closing a Informix connection 参见 • ifx_connect() - Open Informix server connection • ifx_pconnect() - Open persistent Informix connection
