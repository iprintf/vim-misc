cubrid_error_msg (PECL CUBRID >= 8.3.0) cubrid_error_msg — Get last error message for the most recent function call 说明 string cubrid_error_msg ( void ) The cubrid_error_msg() function is used to get the error message that occurred during the use of CUBRID API. Usually, it gets error message when API returns false as its return value. 参数 此函数没有参数。 返回值 Error message that occurred. 范例 Example #1 cubrid_error_msg() example 以上例程会输出： Error facility: 2 Error code: -10015 Error msg: Invalid T_CCI_SCH_TYPE value 参见 • cubrid_error_code() - Get error code for the most recent function call • cubrid_error_code_facility() - Get the facility code of error
