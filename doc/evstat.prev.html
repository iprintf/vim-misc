EvStat::prev (PECL ev >= 0.2.0) EvStat::prev — Returns the previous set of values returned by EvStat::attr 说明 public void EvStat::prev ( void ) Just like EvStat::attr() , but returns the previous set of values. 参数 此函数没有参数。 返回值 Returns an array with the same structure as the array returned by EvStat::attr () . The array contains previously detected values. 参见 • EvStat::attr() - Returns the values most recently detected by Ev • EvStat::stat() - Initiates the stat call
