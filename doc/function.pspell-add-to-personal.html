pspell_add_to_personal (PHP 4 >= 4.0.2, PHP 5, PHP 7) pspell_add_to_personal — Add the word to a personal wordlist 说明 bool pspell_add_to_personal ( int $dictionary_link , string $word ) pspell_add_to_personal() adds a word to the personal wordlist. If you used pspell_new_config() with pspell_config_personal() to open the dictionary, you can save the wordlist later with pspell_save_wordlist(). 参数 dictionary_link word The added word. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 pspell_add_to_personal() 注释 Note: This function will not work unless you have pspell .11.2 and aspell .32.5 or later.
