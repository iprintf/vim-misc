oci_free_descriptor (PHP 5, PHP 7, PECL OCI8 >= 1.1.0) oci_free_descriptor — Frees a descriptor 说明 bool oci_free_descriptor ( resource $descriptor ) Frees a descriptor allocated by oci_new_descriptor(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 注释 Note: This function is commonly used as a method OCI-LOB::free. 参见 • OCI-LOB::free
