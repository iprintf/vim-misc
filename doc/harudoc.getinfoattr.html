HaruDoc::getInfoAttr (PECL haru >= 0.0.1) HaruDoc::getInfoAttr — Get current value of the specified document attribute 说明 string HaruDoc::getInfoAttr ( int $type ) Get the current value of the specified document attribute. 参数 type The type of the attribute. The following values are available: □ HaruDoc::INFO_AUTHOR □ HaruDoc::INFO_CREATOR □ HaruDoc::INFO_TITLE □ HaruDoc::INFO_SUBJECT □ HaruDoc::INFO_KEYWORDS □ HaruDoc::INFO_CREATION_DATE □ HaruDoc::INFO_MOD_DATE 返回值 Returns the string value of the specified document attribute. 错误／异常 发生错误时抛出 HaruException 异常。 参见 • HaruDoc::setInfoAttr() - Set the info attribute of the document • HaruDoc::setInfoDateAttr() - Set the datetime info attributes of the document
