realpath_cache_get (PHP 5 >= 5.3.2, PHP 7) realpath_cache_get — Get realpath cache entries 说明 array realpath_cache_get ( void ) Get the contents of the realpath cache. 返回值 Returns an array of realpath cache entries. The keys are original path entries, and the values are arrays of data items, containing the resolved path, expiration date, and other options kept in the cache. 范例 Example #1 realpath_cache_get() example 以上例程的输出类似于： array(2) { ["/test"]=> array(4) { ["key"]=> int(123456789) ["is_dir"]=> bool(true) ["realpath"]=> string(5) "/test" ["expires"]=> int(1260318939) } ["/test/test.php"]=> array(4) { ["key"]=> int(987654321) ["is_dir"]=> bool(false) ["realpath"]=> string(12) "/root/test.php" ["expires"]=> int(1260318939) } } 参见 • realpath_cache_size() - Get realpath cache size User Contributed Notes 1 note =============================================================================== Note that the realpath cache is not used if either safe_mode is on or an open_basedir restriction is in effect. This is having a huge performance effect, causing lots of calls to lstat. A bugreport has already been filed at http://bugs.php.net/bug.php?id=52312
