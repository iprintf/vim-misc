SplFileObject::fgetc (PHP 5 >= 5.1.0, PHP 7) SplFileObject::fgetc — Gets character from file 说明 public string SplFileObject::fgetc ( void ) Gets a character from the file. 参数 此函数没有参数。 返回值 Returns a string containing a single character read from the file or FALSE on EOF.
