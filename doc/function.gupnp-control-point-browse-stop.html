gupnp_control_point_browse_stop (PECL gupnp >= 0.1.0) gupnp_control_point_browse_stop — Stop browsing 说明 bool gupnp_control_point_browse_stop ( resource $cpoint ) Stop the search and calls user-defined callback. 参数 cpoint A control point identifier, returned by gupnp_control_point_new(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • gupnp_control_point_new() - Create a new control point • gupnp_control_point_browse_start() - Start browsing
