zip_entry_compressionmethod (PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PECL zip >= 1.0.0) zip_entry_compressionmethod — 检索目录实体的压缩方法 说明 string zip_entry_compressionmethod ( resource $zip_entry ) 返回由函数zip_entry确定的目录实体的压缩方法。 参数 zip_entry 由函数zip_read() 返回的目录实体。 返回值 压缩方法。 参见 • zip_open() - 打开ZIP存档文件 • zip_read() - 读取ZIP存档文件中下一项
