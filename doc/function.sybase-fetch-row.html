sybase_fetch_row (PHP 4, PHP 5, PHP 7) sybase_fetch_row — Get a result row as an enumerated array 说明 array sybase_fetch_row ( resource $result ) sybase_fetch_row() fetches one row of data from the result associated with the specified result identifier. Subsequent call to sybase_fetch_row() would return the next row in the result set, or FALSE if there are no more rows. 参数 result 返回值 Returns an array that corresponds to the fetched row, or FALSE if there are no more rows. Each result column is stored in an array offset, starting at offset 0.
