OAuth::setNonce (PECL OAuth >= 0.99.1) OAuth::setNonce — 为后续请求设置现时标志 说明 public mixed OAuth::setNonce ( string $nonce ) 为所有后续请求设置现时标志。 参数 nonce oauth_nonce 的值。 返回值 成功返回 TRUE ，如果 nonce 被认为无效，则返回 FALSE 。 更新日志 版本 说明 1.0.0 以前失败时返回 NULL，而不是 FALSE。 参见 • OAuth::setToken() - 设置令牌和 secret • OAuth::setAuthType() - 设置授权类型 • OAuth::setVersion() - 设置 OAuth 版本
