EventBase::getTimeOfDayCached (PECL event >= 1.2.6-beta) EventBase::getTimeOfDayCached — Returns the current event base time 说明 public double EventBase::getTimeOfDayCached ( void ) On success returns the current time(as returned by gettimeofday() ), looking at the cached value in base if possible, and calling gettimeofday() or clock_gettime() as appropriate if there is no cached time. 参数 此函数没有参数。 返回值 Returns the current event base time. On failure returns NULL.
