fbsql_field_seek (PHP 4 >= 4.0.6, PHP 5, PHP 7) fbsql_field_seek — Set result pointer to a specified field offset 说明 bool fbsql_field_seek ( resource $result [, int $field_offset ] ) Seeks to the specified field offset. If the next call to fbsql_fetch_field() doesn't include a field offset, the field offset specified in fbsql_field_seek () will be returned. 参数 result 由 fbsql_query() 或 fbsql_db_query() 返回的结果标识符。 field_offset The numerical offset of the field. The field index starts at 0. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • fbsql_fetch_field() - Get column information from a result and return as an object
