EventDnsBase::__construct (PECL event >= 1.2.6-beta) EventDnsBase::__construct — Constructs EventDnsBase object 说明 public EventDnsBase::__construct ( EventBase $base , bool $initialize ) Constructs EventDnsBase object. 参数 base Event base. initialize If the initialize argument is TRUE, it tries to configure the DNS base sensibly given your operating system’s default. Otherwise, it leaves the event DNS base empty, with no nameservers or options configured. In the latter case DNS base should be configured manually, e.g. with EventDnsBase::parseResolvConf() . 返回值 Returns EventDnsBase object.
