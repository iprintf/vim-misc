Counter::__construct Counter::__construct — 创建一个包含单一数值的计数器实例。 说明 Counter::__construct ( string $name [, integer $initial_value [, integer $flags ]] ) 创建一个包含单一数值的 Counter 实例。 参数 name 新计数器的名称。 initial_value 计数器的初始值。默认为零(0)。 flags 新计数器的标志，可在形如 COUNTER_FLAG_* 的常量中选择。 返回值 成功则返回 Counter 对象。 错误／异常 Counter::__construct() 在有错时抛出一个异常(Exception)。
