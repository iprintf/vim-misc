apcu_delete (PECL apcu >= 4.0.0) apcu_delete — Removes a stored variable from the cache 说明 mixed apcu_delete ( string $key ) Removes a stored variable from the cache. 参数 key The key used to store the value (with apcu_store()). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 范例 Example #1 A apcu_delete() example 参见 • apcu_store() - Cache a variable in the data store • apcu_fetch() - Fetch a stored variable from the cache
