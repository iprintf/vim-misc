hw_api::identify (PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN) hw_api::identify — Log into Hyperwave Server 说明 bool hw_api::identify ( array $parameter ) Logs into the Hyperwave Server. 参数 parameter The parameter array must contain the elements 'username' and 'password'. 返回值 Returns an object of typeHW_API_Error if identification failed or TRUE if it was successful.
