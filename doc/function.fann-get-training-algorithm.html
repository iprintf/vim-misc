fann_get_training_algorithm (PECL fann >= 1.0.0) fann_get_training_algorithm — Returns the training algorithm 说明 int fann_get_training_algorithm ( resource $ann ) Returns the training algorithm. This training algorithm is used by fann_train_on_data() and associated functions. Note that this algorithm is also used during fann_cascadetrain_on_data(), although only FANN_TRAIN_RPROP and FANN_TRAIN_QUICKPROP is allowed during cascade training. 参数 ann Neural network resource. 返回值 Training algorithm constant, or FALSE on error. 参见 • fann_set_training_algorithm() - Sets the training algorithm
