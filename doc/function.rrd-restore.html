rrd_restore (PECL rrd >= 0.9.0) rrd_restore — Restores the RRD file from XML dump. 说明 bool rrd_restore ( string $xml_file , string $rrd_file [, array $options ] ) Restores the RRD file from the XML dump. 参数 xml_file XML filename with the dump of the original RRD database file. rrd_file Restored RRD database file name. options Array of options for restoring. See man page for rrd restore. 返回值 Returns TRUE on success, FALSE otherwise.
