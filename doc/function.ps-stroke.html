ps_stroke (PECL ps >= 1.1.0) ps_stroke — Draws the current path 说明 bool ps_stroke ( resource $psdoc ) Draws the path constructed with previously called drawing functions like ps_lineto(). 参数 psdoc Resource identifier of the postscript file as returned by ps_new(). 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • ps_closepath_stroke() - Closes and strokes path • ps_fill() - Fills the current path • ps_fill_stroke() - Fills and strokes the current path
