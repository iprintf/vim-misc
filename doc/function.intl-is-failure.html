intl_is_failure (PHP 5 >= 5.3.0, PHP 7, PECL intl >= 1.0.0) intl_is_failure — Check whether the given error code indicates failure 说明 bool intl_is_failure ( int $error_code ) 参数 error_code is a value that returned by functions: intl_get_error_code(), collator_get_error_code() . 返回值 TRUE if it the code indicates some failure, and FALSE in case of success or a warning. 范例 Example #1 intl_is_failure() example 以上例程的输出类似于： false false true 参见 • intl_get_error_code() - Get the last error code • collator_get_error_code() - Get collator's last error code • Collator-getErrorCode() - Get collator's last error code
