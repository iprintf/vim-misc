fann_get_cascade_min_out_epochs (PECL fann >= 1.0.0) fann_get_cascade_min_out_epochs — Returns the minimum out epochs 说明 int fann_get_cascade_min_out_epochs ( resource $ann ) The minimum out epochs determines the minimum number of epochs the output connections must be trained after adding a new candidate neuron. The default min out epochs is 50. 参数 ann Neural network resource. 返回值 The minimum out epochs, or FALSE on error. 参见 • fann_set_cascade_min_out_epochs() - Sets the minimum out epochs
