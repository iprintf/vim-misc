gupnp_context_new (PECL gupnp >= 0.1.0) gupnp_context_new — Create a new context 说明 resource gupnp_context_new ([ string $host_ip [, int $port = 0 ]] ) Create a new context with the specified host_ip and port. 参数 host_ip The local host's IP address, or NULL to use the IP address of the first non-loopback network interface. port Port to run on, or 0 if you don't care what port is used. 返回值 A context identifier. 范例 Example #1 Create new UPnP context 错误／异常 Issues E_WARNING with unable to create context.
