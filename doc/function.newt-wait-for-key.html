newt_wait_for_key (PECL newt >= 0.1) newt_wait_for_key — Doesn't return until a key has been pressed 说明 void newt_wait_for_key ( void ) This function doesn't return until a key has been pressed. The keystroke is then ignored. If a key is already in the terminal's buffer, this function discards a keystroke and returns immediately. 返回值 没有返回值。 参见 • newt_clear_key_buffer() - Discards the contents of the terminal's input buffer without waiting for additional input
