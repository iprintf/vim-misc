EventBase::loop (PECL event >= 1.2.6-beta) EventBase::loop — Dispatch pending events 说明 public bool EventBase::loop ([ int $flags ] ) Wait for events to become active, and run their callbacks. Warning Do NOT destroy the EventBase object as long as resources of the associated Event objects are not released. Otherwise, it will lead to unpredictable results! 参数 flags Optional flags. One of EventBase::LOOP_* constants. See EventBase constants . 返回值 Returns TRUE on success. Otherwise FALSE. 参见 • EventBase::dispatch() - Dispatch pending events
