IntlChar::isMirrored (PHP 7) IntlChar::isMirrored — Check if code point has the Bidi_Mirrored property 说明 public static bool IntlChar::isMirrored ( mixed $codepoint ) Determines whether the code point has the Bidi_Mirrored property. This property is set for characters that are commonly used in Right-To-Left contexts and need to be displayed with a "mirrored" glyph. 参数 codepoint The integer codepoint value (e.g. 0x2603 for U+2603 SNOWMAN), or the character encoded as a UTF-8 string (e.g. "\u{2603}") 返回值 Returns TRUE if codepoint has the Bidi_Mirrored property, FALSE if not. 范例 Example #1 Testing different code points 以上例程会输出： bool(false) bool(true) bool(true) 参见 • IntlChar::charMirror() - Get the "mirror-image" character for a code point • IntlChar::PROPERTY_BIDI_MIRRORED
