trader_ppo (PECL trader >= 0.2.0) trader_ppo — Percentage Price Oscillator 说明 array trader_ppo ( array $real [, integer $fastPeriod [, integer $slowPeriod [, integer $mAType ]]] ) 参数 real 浮点数数组。 fastPeriod Number of period for the fast MA. Valid range from 2 to 100000. slowPeriod Number of period for the slow MA. Valid range from 2 to 100000. mAType Type of Moving Average. TRADER_MA_TYPE_* series of constants should be used. 返回值 Returns an array with calculated data or false on failure.
