ZMQSocket::getPersistentId (PECL zmq >= 0.5.0) ZMQSocket::getPersistentId — Get the persistent id 说明 public string ZMQSocket::getPersistentId ( void ) Returns the persistent id of the socket. 参数 此函数没有参数。 返回值 Returns the persistent id string assigned of the object and NULL if socket is not persistent.
