ifx_blobinfile_mode (PHP 4, PHP <=5.2.0) ifx_blobinfile_mode — Set the default blob mode for all select queries 说明 bool ifx_blobinfile_mode ( int $mode ) Set the default blob mode for all select queries. 参数 mode Mode "0" means save Byte-Blobs in memory, and mode "1" means save Byte-Blobs in a file. 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。
