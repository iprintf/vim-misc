ZMQSocket::send (PECL zmq >= 0.5.0) ZMQSocket::send — Sends a message 说明 public ZMQSocket ZMQSocket::send ( string $message [, integer $mode = 0 ] ) Send a message using the socket. The operation can block unless ZMQ::MODE_NOBLOCK is used. 参数 message The message to send. mode Pass mode flags to receive multipart messages or non-blocking operation. See ZMQ::MODE_* constants. 返回值 Returns the current object. Throws ZMQSocketException on error. If ZMQ::MODE_NOBLOCK is used and the operation would block boolean false shall be returned.
