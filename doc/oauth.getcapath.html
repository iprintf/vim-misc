OAuth::getCAPath (PECL OAuth >= 0.99.8) OAuth::getCAPath — 获取 CA 信息 说明 public array OAuth::getCAPath ( void ) 获取证书授权的信息，其中包括通过 OAuth::setCaPath() 设置的 ca_path 和 ca_info 。 Warning 本函数还未编写文档，仅有参数列表。 参数 此函数没有参数。 返回值 返回一个证书授权信息的数组，在返回的关联数组中明确地包含 ca_path 和 ca_info 键 。 参见 • OAuth::setCAPath() - 设置 CA 路径和信息 • OAuth::getLastResponseInfo() - 获取关于最后一次响应的 HTTP 信息
