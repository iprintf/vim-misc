GearmanClient::setDataCallback (PECL gearman >= 0.6.0) GearmanClient::setDataCallback — Callback function when there is a data packet for a task 说明 public bool GearmanClient::setDataCallback ( callable $callback ) Sets the callback function for accepting data packets for a task. The callback function should take a single argument, a GearmanTask object. 参数 callback A function or method to call 返回值 成功时返回 TRUE，或者在失败时返回 FALSE。 参见 • GearmanClient::setCompleteCallback() - Set a function to be called on task completion • GearmanClient::setCreatedCallback() - Set a callback for when a task is queued • GearmanClient::setExceptionCallback() - Set a callback for worker exceptions • GearmanClient::setFailCallback() - Set callback for job failure • GearmanClient::setStatusCallback() - Set a callback for collecting task status • GearmanClient::setWarningCallback() - Set a callback for worker warnings • GearmanClient::setWorkloadCallback() - Set a callback for accepting incremental data updates
