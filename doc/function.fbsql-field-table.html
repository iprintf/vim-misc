fbsql_field_table (PHP 4 >= 4.0.6, PHP 5, PHP 7) fbsql_field_table — Get name of the table the specified field is in 说明 string fbsql_field_table ( resource $result [, int $field_offset ] ) Returns the name of the table that the specified field is in. 参数 result 由 fbsql_query() 或 fbsql_db_query() 返回的结果标识符。 field_offset The numerical offset of the field. The field index starts at 0. 返回值 Returns the name of the table, as a string.
