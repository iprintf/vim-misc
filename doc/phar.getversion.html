Phar::getVersion (PHP >= 5.3.0, PECL phar >= 1.0.0) Phar::getVersion — Return version info of Phar archive 说明 public string Phar::getVersion ( void ) Returns the API version of an opened Phar archive. 参数 返回值 The opened archive's API version. This is not to be confused with the API version that the loaded phar extension will use to create new phars. Each Phar archive has the API version hard-coded into its manifest. See Phar file format documentation for more information. 参见 • Phar::apiVersion() - Returns the api version
