Imagick::commentImage (PECL imagick 2.0.0) Imagick::commentImage — Adds a comment to your image 说明 bool Imagick::commentImage ( string $comment ) Adds a comment to your image. 参数 comment The comment to add 返回值 成功时返回 TRUE。 错误／异常 错误时抛出 ImagickException。 范例 Example #1 Using Imagick::commentImage(): Commenting an image and retrieving the comment: newImage(100, 100, new ImagickPixel("red")); /* Add comment to the image */ $im->commentImage("Hello World!"); /* Display the comment */ echo $im->getImageProperty("comment"); ?>
