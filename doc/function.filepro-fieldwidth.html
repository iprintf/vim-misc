filepro_fieldwidth (PHP 4, PHP 5 <= 5.1.6, PHP 7) filepro_fieldwidth — Gets the width of a field 说明 int filepro_fieldwidth ( int $field_number ) Returns the width of the field corresponding to field_number. 参数 field_number The field number. 返回值 Returns the width of the field as a integer, or FALSE on errors.
